# 薄合规风险备忘

## 定位

本文件只在 `/compliance` 使用，用来给成稿做最后一层风险提示。它不是创作前置审核，不是 rewrite gate，也不是让 writer 把强刺激洗弱的清单。

默认原则：

- `/plan`、`/outline`、`/episode`、`/dialogue-polish`、`/review` 不因本文件主动降级剧情强度。
- 身体风险、真实伤害、亲密张力、羞辱、压迫、反派新伤害，本身不是问题；它们经常是短剧追看和评论驱动的商业刺激。
- 合规只处理“表达方式风险”，不删除源本赚钱功能。
- 如果用户明确表示自己会做最终制作合规，本文件只输出风险标签和建议，不强制返工。

## 创作优先级

短剧源头优先保住：

1. 可见伤害：过敏、摔倒、被关、被推、伤口裂开、站不稳、药物被拿走。
2. 关系压迫：家人不信、当众羞辱、身份剥夺、被逐出、被迫选择。
3. 身体/亲密/危险张力：靠近、扶住、擦伤、差点吻、同床尴尬、保护性触碰、诱惑试探、压迫逼近。
4. 反派升级：嫁祸、夺功、逼迫、反咬、夺权、公开毁掉体面。
5. 爽点兑现：当场失去席位、门禁、身份、权力、脸面、关系或选择权。

不要为了“看起来更安全”把这些改成会议、说明、证据递交、调查结论或泛泛对话。

## 真正需要提示的风险

只标这些风险：

| 风险 | 处理口径 |
| --- | --- |
| 血腥细节过重 | 保留伤害结果，减少血腥特写和生理细节。 |
| 可模仿危险行为 | 保留剧情后果，模糊具体操作步骤。 |
| 非自愿亲密被写成浪漫 | 改成双方张力、误会拉扯、即时停住，或把越界者明确写成反派压力。 |
| 露骨性行为/裸露直写 | 改成情绪、动作边界、镜头切走或事后关系变化。 |
| 未成年性化 | 删除或改年龄设定。 |
| 真实品牌/媒体/机构未授权 | 换成虚构品牌、虚构榜单、虚构媒体，但保留公开打脸功能。 |
| 医疗/药物误导 | 保留身体风险，避免给出错误操作或具体剂量。 |
| 可模仿犯罪/危险手段被详细展示 | 保留反杀爽感，模糊操作过程，把重点放到后果、反噬和关系变化。 |

## 身体伤害口径

允许并鼓励使用“观众一眼懂”的身体代价：

- 过敏发作、呼吸困难、手抖、站不稳、被迫吞下禁忌食物。
- 被推倒、撞伤、手腕被攥红、雨中发抖、旧伤裂开。
- 药物被藏、医生被拦、电话被抢、门被锁。

写法边界：

- 写受害者的处境、痛苦和周围人的冷漠。
- 写反派的故意、旁观者的偏心、主角被迫行动。
- 不写可复制的伤害步骤，不沉迷血腥特写，不把施害写成酷。

## 身体/亲密/危险张力口径

海外短剧的男频、女频和混合向都可以有更强的身体张力。区别只在承载方式：女频多落在保护、占有、拉扯、被偏爱；男频多落在诱惑、崇拜、权力确认、危险选择和反杀后的身份变化。允许：

- 贴近、扶住、压低声音、擦药、披外套、手腕被扣住后立刻停住。
- 差点吻、误会同床、被保护时撞进怀里、危险人物低声逼近、诱惑与反制并存。
- “嘴上拒绝、身体/眼神有犹豫”的拉扯可以写，但必须让观众看见被推进关系的一方仍有选择权。

边界：

- 不把真正的强迫、性暴力、限制自由写成浪漫奖励。
- 如果角色明确恐惧、逃避、说不，另一方必须停住；不停住就是反派压力，不是甜宠。
- 可以霸总、黑帮、狼人、逆袭、复仇、诱惑、危险、暧昧，但不要把“没有同意”包装成“其实喜欢”。

## 输出格式

`/compliance` 只输出薄风险报告：

```markdown
# 薄合规风险提示

## 不建议降级的强刺激
- {身体风险 / 亲密张力 / 羞辱压迫 / 反派新伤害：为什么应保留}

## 需要制作期注意的风险
- 第{N}集：{风险标签} -> {不削弱爽点的处理建议}

## 真实品牌/授权风险
- {真实品牌或机构} -> 建议替换为 {虚构等价物}

## 不需要修改
- {容易被误判但实际应保留的短剧刺激点}
```
