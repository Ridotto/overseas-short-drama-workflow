## 一、一句话：这是什么

**一个帮你把爆款短剧"洗"成新本子的工具。**

你手上有一批已经被市场验证过的爆款短剧。你想要新的本子，但不想从零编——风险高、又慢。这个工具帮你：**拿一个爆款当底子，保留它好看、能让观众追下去、能让观众付费的那个内核，把外面的壳（题材、人物、背景、世界观这些）换掉，变成一个看起来不一样、但同样好看的新本子。**

---

## 二、为什么要做这个（解决什么问题）

### 现实问题
- 写原创短剧爆款，风险极高——大部分原创扑街，你不知道哪个能成。
- 但爆款是有迹可循的：它为什么爆，是有内核的（钩子、爽点、卡点、情绪节奏这些）。
- 行业里"洗稿"本来就是常见做法：把验证过的爆款，换个壳再做一遍，成功率比纯原创高得多。

### 现在洗稿的痛点（没有工具时）
- 靠人脑洗：慢，而且洗稿的人水平参差。
- 直接丢给 ChatGPT 洗：它凭感觉改，没有短剧爆款的标准当尺子，洗出来经常是 cliche、AI 味重、把好的内核也洗丢了。
- 洗稿最容易翻车的地方：**换壳换得生硬**（新背景下人物动机站不住、假），或者**把爆款的内核洗没了**（壳换了，但好看的那个魂丢了），或者**把源本的烂点也一起照搬过来**。

### 这个工具要解决的
**让洗稿这件事，既快、又能稳定地保住爆款内核、还能让新壳自洽不假。** 不是靠运气和手感，是靠一套方法。

---

## 三、给谁用（用户是谁）

**写本子的人 / 编剧。** 就是你自己，或你团队里负责出本子的人。

不服务导演、监制、甲方、平台。这个工具只管"出一个洗好的本子"，下游怎么用（拍成片、投平台）不归它管。

---

## 四、用户拿它来干什么（用途）

用户的真实场景：
- 手上有一批爆款短剧（源本）。
- 想基于某个爆款，做一个新题材/新壳的本子。
- 心里一般已经有方向（"我想把这个狼人爆款洗成都市霸总"），也可能只有模糊想法、拿来先聊聊看。
- 可能还想顺便改集数（原本40集，我想做成80集 或 30集）。

工具帮他：**把这个想法落地成一个真正写出来的、好看的新本子**——而不是只给建议、只给分析报告。**最终产物是能用的剧本本身。**

---

## 五、它和用户怎么交互（交互逻辑）

这是关键，之前一直没讲清。核心交互逻辑是：

### 1. 不是"一键生成"，是"先对齐、再动手、边做边确认"
```
用户给一个源本 + 说想怎么洗
  → 工具先和用户把需求聊清楚（要换成什么、改多狠、要不要改集数、
    什么绝对不能丢、特别想要什么）
  → 对齐了，工具才动手
  → 不是闷头洗完整本，是先洗一小批（前几集）给用户看
  → 用户认可了，再按同样的路子往下洗
```
**为什么这样**：洗稿是有创作判断的事，工具不能自作主张洗完一整本（万一方向错了，整本返工）。要在关键节点让用户拍板。

### 2. 工具和用户是"分工"，不是"工具全包"
```
用户负责：定方向（换什么壳、改多狠）、做取舍判断（这个改法我喜不喜欢）、
         最终拍板
工具负责：把源本读透、找出它为什么好、动手洗、保证洗出来的本
         保住内核 + 新壳自洽 + 不带AI味
```
用户一般已经知道自己要什么（这是有经验的编剧），工具不是替他做决定，是**帮他把决定高质量地落地**，并在他没想清楚时给建议。

### 3. 工具要让用户"信得过"
```
工具洗完，要能告诉用户：
  · 我改了哪些地方、为什么这么改（让用户能对照源本看）
  · 源本好看的内核，我保住了没有
  · 哪些地方我拿不准，需要你来定
```
不是给个黑箱结果让用户猜，是改动透明、可追溯。

### 4. 输入输出
```
用户给：一个源剧本（文本）+ 用他自己的话说想怎么洗
工具给：洗好的新本子（纯文本，先给前几集，确认后续写）
        + 改了什么的对照说明
```
工具不碰视频、不输出代码或表格数据，就是给"能直接用的剧本文字"。

## 六、最根本的两条原则（贯穿一切）

### 原则一：本质是"写好剧本"，只是带一个参考（源本）
- 它不是机械的"查表替换"——换壳要让新本子真的自洽、不生搬硬套。
- 它也不是天马行空的创作机器——**源本永远是底子，不能脱缰乱编。**
- 准确说：它是一个**会动脑的洗稿器**——照着源本洗，但洗得动脑、洗得像个真本子。

### 原则二：一切为了"观众想付费、想追下去"，但不能靠堆砌
- 洗出来的本子，最终要让观众愿意付费、愿意一直追下去。
- 但**不能为了这个就拼命堆钩子、堆爽点、堆卡点**——那样堆出来是发疯式的烂本（情节不成立、人物不可信）。
- 正确的是：**先是一个扎实的好剧本，在这个基础上，那几个最重要的因素（钩子/爽点/卡点/情绪）写得好**——两者都有，观众自然就追、就付费。

---

## 七、这个工具不是什么（划清边界）

- ❌ 不是评分器（它不是用来给剧本打分排名的）。
- ❌ 不是分析报告工具（它最终要出本子，不是出一份分析）。
- ❌ 不是从零原创的写作工具（它一定基于一个源本）。
- ❌ 不是一键成片/视频工具（它只出剧本文字）。
- ❌ 不是查词典式的换皮机器（换壳要动脑、要自洽）。
