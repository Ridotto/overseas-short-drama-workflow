# 03 洗稿边界包

## 新壳

海外豪门商战：跨国港口救援科技集团、海上救援演示、慈善救援基金、家族信托、直播听证、医疗船、港口安全许可。

## 深度重构边界

保留体验功能和追看链，不保留源本事件外形。

允许大幅改：

- 事故空间；
- 男女主职业和权力来源；
- 反派作恶方式；
- 公开打脸场域；
- 旧物形态；
- 每集具体事件载体。

不允许改：

- 男主主动选错；
- 女主失子后死心；
- 女主高位归来；
- 旧爱继承人叙事分阶段塌房；
- 男主必须被孩子真相击穿；
- EP10 必须是“献上一切求原谅但女主不接”的卡点。

## 必须替换的高辨识外形

- 黑帮、军火、教父、枪战、黑道宴会。
- 卡车车祸 + 出租车爆炸。
- 地下黑帮审判。
- V5 胸针 / 黑卡 / 封条整套动作簇。
- V27 root key / CEO button / 债权拍卖整套外形。
- V28 后半段连续依赖音频、定位图、DNA 屏幕、终止按钮的安全实现。

## 真相模型

```text
Serena 当年确实怀孕，但孩子不是 Ethan 的。
她伪造的是“Calder 继承人”身份、受害者身份和事故责任链。
Nora 当年怀的是 Ethan 的双胎；双胎在港口事故后的离岸医疗船上死亡。
```

## 防洗飞红线

- Ethan 不能只是“不知道 Nora 在危险里”；他必须听见她求救。
- 港口事故不能变成流程播报；必须是人在现场选择人。
- Nora 归来不能只是“身份更高级”；必须让 Ethan 正在推进的项目当场停摆。
- Serena 不能只是撒谎小人；她要有资源、误判和继续作死的理由。
- Nora 在 EP10 不能软化。
