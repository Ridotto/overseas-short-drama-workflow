# 10 作者自检与返修路由

## 总判定

```text
workflow execution: S0-S9 complete, S10 skipped
creative outcome: conditional revise / promising but not pass
delivery outcome: not pass without reviewer and user review
```

V29 比 V28 更明确地让中后段从证据链转成人物行动链。  
但仍不是正式 pass：部分集数仍可能偏“机制清楚但情绪余韵不够”，EP10 的下一轮追问打开得有效但还不够爆。

## 与本轮小修目标对照

| 目标 | 执行情况 | 判断 |
|---|---|---|
| S3 追看强度地图 | 04 已明确波峰、呼吸、兑现、追问、反派燃料 | 通过 |
| S4 场面承载选择 | 05 每个高压节点给候选，并确认证据地位 | 通过 |
| S5 行动链 | 06 每集按动作 / 被迫反应 / 可见后果排戏 | 通过 |
| 不模板化 | 每集载体不同，未固定“开场钩子+打脸+反转” | 基本通过 |
| 不让证据当本体 | EP6/EP7/EP8/EP9 均由人物动作承载证据 | 明显改善 |

## 主要优点

1. **EP6 改善最大**：从“听音频 / 看记录”变成 Ethan 被迫站进旧水槽，亲手重扣当年选择，最后说 “I heard her.”  
   这是本轮最能证明 S4/S5 生效的地方。

2. **EP7 不再只是定位图**：Serena 被迫走她声称走过的危险路线，身体先垮，受害者花被摘。  
   这比单纯播证据更像短剧现场。

3. **EP8 把亲子结果变成人物动作**：Ethan 亲手摘掉孩子救生徽，孩子哭，小椅子撤走。  
   证据退到触发器，羞辱和权力变化落到人身上。

4. **EP10 比 V28 少按钮感**：主动作是公开认罪、救援队员撕下 Calder 标志、Nora 不碰腕带。  
   机构动作仍在，但不再是单纯“按键终止”。

## 仍然存在的问题

### P1：EP10 卡点还可以更强

当前卡点是 Serena/Owen 交易频道亮起。它有效，但情绪冲击不如 EP6-EP9。  
如果返修，优先回 S5，不回 S3：保留公开认罪和不接赎罪，只加强下一轮追问的画面冲击。

### P2：Nora 的内心痛感仍偏克制

本稿有身体动作和冷拒，但 Nora 在 EP4-EP8 的痛感主要靠动作和冷台词，缺少一两个不解释但能见痛的瞬间。  
如果返修，回 S6/S7：不加长对白，只加小动作或停顿。

### P3：Owen 线只作为后续燃料，首批内存在感仍薄

这符合 DNC，但如果用户希望 1-10 内反派压力更满，可以在 EP5/EP7 增加 Owen 的远程压迫动作。  
返修层：S3/S5。

## 不建议返修的方向

- 不要把每集继续加反转，会破坏当前压放节奏。
- 不要把 EP8 改回 DNA 屏幕。
- 不要让 Nora 解释五年前痛苦。
- 不要为了更爽让 Ethan 过早彻底站到 Nora 一边。

## 返修路由

| 问题 | 回滚层 |
|---|---|
| EP10 卡点不够爆 | S5/S6 |
| Nora 情绪余韵偏克制 | S6/S7 |
| Owen 线存在感不足 | S3/S5 |
| 若 reviewer 认为仍证据链过重 | 先查 S4，再查 S5 |
