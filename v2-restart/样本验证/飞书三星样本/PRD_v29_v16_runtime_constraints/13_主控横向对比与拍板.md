## 与 V5 对比

V5 的优点是场面发动粗暴直接，但也有强复刻风险和旧外形残留。

V29 相比 V5：

- **强于 V5**：新壳独立性、防复刻、DNC 分层、后续反派燃料更清楚。
- **接近 V5**：EP2 二选一、EP6 复演、EP8 摘徽、EP9 双腕带，这些动作链能承载短剧强度。
- **弱于 V5 的点**：V5 某些物件记忆点更粗暴，比如胸针 / 黑卡 / 封条；V29 更自然，但爆裂感略收。

判断：

```text
V29 不复刻 V5，但开始拿回 V5 的“动作链发动”能力。
```

## 小修是否有效

有效，但不是最终 pass。

有效证据：

1. S3 `追看强度地图` 让后半段不再只写“真相分层”，而是明确 EP6 承认、EP7 身体露馅、EP8 摘徽、EP9 旧物击穿、EP10 公开认罪。
2. S4 `场面承载选择` 让 EP6/EP7/EP8 没有默认回到音频 / 定位图 / DNA 屏幕。
3. S5 `行动链` 让 screenplay 有“手扣吊篮、走危险路线、摘救生徽、拉开保温箱、撕下 Calder 标志”这些可见动作。

仍不足：

- EP10 的下一轮追问有效但不够狠；
- Nora 的情绪外化主要靠冷动作，痛感还可以更细；
- 未跑 clean reviewer。

## 根因判断更新

V28 证明：深度重构能推外形距离，不能自动推强度。  
V29 证明：把 S3/S4/S5 的假执行点改成生产约束后，中后段强度确实能被推动。

所以当前根因不是“缺短剧知识”，也不是“只能靠 reviewer”。  
更准确是：

```text
短剧强度需要在 S3/S4/S5 变成生产动作。
只写在知识库、要求或 reviewer 里都不够。
```
