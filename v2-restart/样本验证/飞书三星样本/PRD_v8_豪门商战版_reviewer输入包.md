## 2. 用户需求

任务类型：洗稿。  
源本：《黑手党少主痛哭求原谅，前妻她杀疯了》。  
新壳：海外豪门商战 / 家族企业权力斗争。  
目标市场：海外短剧。  
集数：不变。  
本轮输出：首批 1-10 集。  
改写力度：中度偏重；不能只是换名，也不能丢源本有效性。

## 3. 源本有效功能摘要

首批 1-10 集要迁移的不是具体黑帮事件，而是功能链：

```text
女主被替代 / 被误判
-> 男主做出错误选择
-> 女主承受不可逆损失
-> 女主以隐藏能力和新权力回归
-> 女配骗局被拆
-> 男主知道自己不是被骗的无辜者，而是亲手造成损失的人
-> 男主权力坍塌并求赎罪
-> 女主听见但不原谅，形成后续追看
```

源本最强体验：

- 女主受尽委屈后反杀；
- 男主从高位傲慢到彻底崩溃；
- 女配伪善面具被拆；
- 孩子死亡形成不可轻易原谅的火葬场；
- 首批结尾不复合，只给追妻和赎罪悬置。

## 4. 洗稿边界

必须保留：

- 被替代、被误判、被迫离开；
- 男主错误选择；
- 不可逆损失；
- 五年后高位回归；
- 女配骗局拆穿；
- 男主同罪和火葬场；
- 求赎罪但不被原谅。

必须避开：

- 黑帮、军火、枪战；
- 车祸 / 火海二选一；
- 假孕白月光和亲子鉴定；
- 黑医证词式听证；
- 仓库保险箱、小毛衣、孕检单；
- 暴雨下跪 + 枪口测试。

V8 当前新载体：

- 女主隐藏能力：Atlas 风控系统真作者 / 跨境资本重组人 Aria North。
- 男主错误选择：Adrian 签 federal referral 和 no medical exception，把 Serena 推入拘押拒医。
- 不可逆损失：Serena 在拘押车和急救中失去双胎。
- 女主回归：以 North Capital 债权人身份让 Blackwell 停盘。
- 女配骗局：救市 token、继承信托、Atlas 作者签名不成立。
- 男主同罪：CEO key 打开五年前 referral log，旧语音和签名钉死他。
- 火葬场：联邦托管箱里的超声、拘押腕带、胎心录音和未寄周年卡。
- 求赎罪：公开认罪、CEO 辞任、投票权转让、联邦调查。

## 6. Reviewer 审稿问题

请按 `pass / revise / block` 输出结论，并明确失败层。

重点判断：

1. 源本有效性是否迁移，是否洗飞。
2. V8 是否仍像源本改名复刻。
3. 第 2 集的资本犯罪 / 拘押拒医载体是否足够短剧化，还是变成法务解释。
4. 第 7-8 集是否是动作链，还是包装过的证据听证会。
5. 男主罪责是否由他自己的行为成立，而不是只靠 Chloe 自白。
6. 第 9 集火葬场是否够强，是否提前消费。
7. 第 10 集是否完成首批小闭环，同时没有让女主过早原谅。
8. 台词是否有 AI 味、解释腔、同声化。
