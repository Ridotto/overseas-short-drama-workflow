## 3. 源本核心判断

这个源本不是整体都强。它的强点集中在前 1-10 集：

1. 女主被误解、被替代、被抛弃，且伤害不可逆。
2. 女主真实身份和能力提前埋下，后面强势归来。
3. 假白月光的谎言被一层层公开打爆。
4. 男主从“掌权者”跌成“求原谅的人”，关系权力彻底反转。

它的弱点从 11 集后开始变明显：

1. 男主赎罪方式反复变成下跪、挡枪、受伤、求留在身边。
2. 外部敌人不断接力，但很多功能是在延长追妻火葬场。
3. 20 集后从“核心情感复仇”转成“女王守盘 + 外敌战”，主期待被稀释。
4. 33 集后变成相对程序化的商业/黑帮信誉战，和开头的强情绪不是同一种驱动力。

所以这个样本适合验证产品：不是把源本全盘当范本，而是识别前段有效性、保留中段可用功能、修掉后段空转和重复。

## 5. 五层分析

| 层 | 源本有效点 | 迁移时要注意 |
|---|---|---|
| 留存骨架 | 前 10 集每集都有明确升级：逼离婚、车祸、流产、归来、断供、查孕、公开揭露、男主崩溃、下跪枪响 | 首批新本建议覆盖到同等的“男主/压迫方权力坍塌”节点，不能只写到女主归来就停 |
| 情绪爽点 | 羞辱、误解、抛弃、失子、归来、掌掴、断供、公开打脸、迟来的悔恨 | 爽点不是“多放钩子”，而是压抑和兑现要连续升级 |
| 主线推进 | 前段每集都有新信息或新局势：女主真孕、Lily 假孕、女主真身份、男主发现真相 | 新本每场至少要带来关系、局势、信息或权力的变化；不能只反复吵架 |
| 人物压力 | Lily 制造私情和假孕压力；Dante 提供权力压迫；Victor 提供外部支撑和后期威胁；Eve 是从受害者到掌控者 | 角色功能要迁移，不一定迁移身份。压迫源、背叛者、支撑者、见证者不能无故消失 |
| 干净度 | 开头视觉化很强，关键事件都能拍出来；但后段重复绑架、挡枪、下跪、暴雨、刑罚，容易疲劳 | 必须删掉重复赎罪动作，压缩同功能反派；把抽象心理写成可见动作 |

## 6. 变量迁移基准

| 变量 | 建议策略 | 说明 |
|---|---|---|
| 开头强羞辱 + 女主隐藏信息 | 强继承 | 这是第一集抓人的核心 |
| 女主被替代 / 被误判为恶人 | 强继承 | 观众站队由此建立 |
| 不可逆伤害 | 强继承或等效迁移 | 原本是流产和失子；如果新市场不适合，可换成同等级不可逆损失 |
| 女主隐藏能力 / 隐藏身份 | 强继承 | 后续权力反转必须有来源 |
| 五年后强势归来 | 等效迁移 | 时间跨度可改，但必须有“再见时权力倒置”的体验 |
| 公开打脸链 | 强继承 | Lily 假孕、出轨、孩子非亲生是连续打脸，不是单一证据 |
| 男主发现真相并崩溃 | 强继承 | 这是标题期待和追妻火葬场的启动点 |
| 男主求赎罪但不被立刻原谅 | 强继承 | 不能太快和解，否则前面不可逆伤害失效 |
| Mafia / 军火 / 地下世界 | 自由改写 | 只是外壳；可换成豪门、商战、医疗、律政、娱乐资本、科技集团等 |
| 枪、码头、黑帮宴会、军火线 | 等效迁移 | 保留“资源控制 / 证据公开 / 权力压制”功能即可 |
| Victor 支撑者转潜在反派 | 等效迁移 | 可保留“助推女主上位但另有所图”的人物功能 |
| Lily 反派功能 | 强继承 / 压缩 | 她是前 10 集发动机；后段反复作死可压缩 |
| 男主反复挡枪、重伤、下跪 | 必须压缩 | 功能重复，容易把戏写成单调赎罪展示 |
| 反复“一人赴约陷阱” | 必须压缩或换形 | 后段多次重复同一留存结构 |
| 过度刑罚、猎犬、鳄鱼池 | 条件替换 | 取决于目标市场和用户尺度；不应默认继承 |
| OS 解释人物心理 | 修复 | 能用动作、选择、现场后果表达的，不要靠 OS 解释 |

## 7. 首批范围判断

如果用这部源本做首批新本，首批不建议只做 3 集或 5 集。

建议首批覆盖到源本第 10 集的等效位置，因为第 1-10 集才形成完整小闭环：

1. 开头抓人：假孕白月光登堂入室，逼离婚。
2. 核心压力：男主误判女主，女主被抛弃并遭受不可逆伤害。
3. 阶段兑现：女主强势归来，公开打脸，断掉男主资源。
4. 真相引爆：男主知道自己害死亲生孩子。
5. 新追问：男主下跪求原谅，女主会不会杀他 / 原不原谅他。

如果首批只写到女主归来，会有“爽点刚开始就停”的问题。
如果首批写到第 8 集，真相公开够爽，但男主崩溃还没完成。
第 10 集的枪响卡点，是更完整的首批闭环。

## 8. 自检重点

| 检查项 | 不合格表现 |
|---|---|
| 是否洗飞源本有效性 | 新本没有强羞辱、没有不可逆伤害、没有权力反转，变成普通换皮 |
| 是否机械复刻 | 明明换了新壳，却还硬保留黑帮宴会、军火断供、码头枪战等不成立桥段 |
| 是否钩子堆叠 | 每分钟都塞危机，但人物动机和局势没有真正推进 |
| 是否只虐不爽 | 女主一直受害，迟迟没有权力反击或阶段兑现 |
| 是否太快原谅 | 男主几句忏悔或一次救人就被接纳，前面伤害失去重量 |
| 是否继承烂点 | 后段反复绑架、挡枪、下跪、暴雨受伤，没有新增信息和关系变化 |
| 是否不镜头化 | 写“她彻底黑化”“他无比崩溃”但没有动作、选择、证据、场面呈现 |
| 是否信息空转 | 一集看完，关系、局势、身份信息、危险程度都没变化 |
