# PRD V18 v8workflow 首批写作简报

## 0. 本步用途

把读本笔记、洗稿方案和关键场面迁移卡压成正文输入。正文只吃本简报，不展开厚分析。

## 全局写作约束

- 中文动作/场景描述 + 英文对白。
- 正文不出现 Purpose / Function / Source Ref / Gate 等说明词。
- 每集结尾停在动作、反应、选择或后果前一拍。
- 商战信息要落成可见动作：胸针摘下、门禁失效、直播红灯、投票牌熄灭、安保换边、文件被签。
- 女主不靠解释委屈取胜。
