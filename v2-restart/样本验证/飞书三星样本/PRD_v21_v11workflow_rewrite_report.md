# PRD_v21_v11workflow_rewrite_report

## Failure Layer

当前正文不需要回滚到 source_analysis / boundary / creative_pack / stage_skeleton。

需要重点自查的层：

- scene_option：Ep7-Ep8 是否彻底摆脱“报告/认证台”。
- screenplay：商战权力动作是否足够可见。
- dialogue：少量英文台词是否顺口、短、能推动关系。
- continuity：Ep6 怀孕线索、Ep8 反咬、Ep9 旧物之间的信息释放是否过早。

## Root Symptoms

1. Ep7 仍使用证人和文件入口，如果正文让证人“说完即解决”，会滑成证据戏。
2. 豪门商战容易冷成合同/债权说明，必须让每次商业动作都有身体和场面后果。
3. Ep10 的极端选择如果只靠台词，会显得舞台化；必须靠屋顶应急门、高空风、文件飞散和 Ethan 真实踏上边缘来成立。

## Prioritized Actions

1. 保留 Ep7 证人，但让高潮留给 Ep8 的孩子身体选择：Leo 扑向真实父亲并喊 Daddy。
2. Ep4-Ep5 的商业权力必须有可见动作：合同砸胸口、保安退后、酒店门锁落下、Serena 跪在碎玻璃边。
3. Ep9 不能只写“他后悔”，必须用 pregnancy scan、婴儿袜、手写卡和自扇/雨跪承载。
4. Ep10 必须让 Ethan 交出 Vale 权力物件后仍被 Ava 否定，防止“资产=赎罪”。

## Applied / Confirmed Fixes

- Ep2 事故外形已从源本车祸替换为 Vale 庄园 helipad 火门。
- Ep4 权力外形已从黑帮拍卖/军火线替换为债权拍卖、港口债权和供应链断供。
- Ep7-Ep8 没有使用当场验血或亲子鉴定报告作为高潮；高潮由孩子扑向真实父亲、Serena 当众崩溃和 Ethan 被反咬构成。
- Ep10 没有使用枪口或黑帮徽章；改为 CEO 印章、投票权、家族信托和屋顶边缘。
- 对少量台词做了顺口修订：
  - `She is leaving through the helipad path.` -> `She is leaving by the helipad path.`
  - `You were Nightingale?` -> `You're Nightingale?`
  - `maybe I forgive you` -> `maybe I'll forgive you`

## Rollback Point

- 若 reviewer 判断 Ep7-Ep8 仍是证据戏，应回 Step 8 高压场面候选，重选假继承人崩塌载体。

- 若 reviewer 判断正文台词过解释，应回 Step 13 Dialogue polish，不回滚结构。
