# PRD v17-v7workflow 洗稿方案

## 1. 本次改写一句话方案

把源本“黑手党少主误判孕妻、车祸抛弃、五年后追悔火葬场”洗成“家族企业继承人误信旧爱、发布会事故中抛弃怀孕前妻、五年后前妻以资本女王身份回归接管帝国”的海外豪门商战短剧。

## 2. 保留什么

- 核心期待：女主什么时候反杀，男主什么时候知道自己亲手毁掉亲骨肉。
- 钩子惯性：每集停在未完成的动作、反应或公开后果前。
- 爽点周期：先压低女主，再让女主公开反压；先让反派得意，再让反派崩。
- 情绪走势：E1 被替代，E2 被抛弃，E3 失子复仇，E4-E5 回归碾压，E6-E8 真相逼近，E9-E10 男主崩塌求赎。
- 关系拉扯：男主从“命令前妻”到“跪求罪人”；女主从“被赶走”到“决定他有没有资格赎罪”。
- 信息差：观众知道女主怀孕和被害；男主迟到知道；反派知道自己伪造。
- 权力变化：女主从被家族驱逐到掌握家族债权、舆论和发布会现场。

## 4. 删除什么

- 低信息反复解释；
- 只靠文件证明推进；
- 机械“放证据 -> 震惊 -> 再放证据”；
- 标志性桥段整串照搬。

## 6. 高压节点处理

本轮重点高压节点：

- E2 玻璃中庭事故：必须达到源本车祸二选一的视觉和情绪冲击。
- E3 失子：必须有可见动作，不只一句“孩子没了”。
- E7-E8 骗局崩塌：必须有人被逼失控，不只播证据。
- E9-E10 男主崩溃：必须有身体代价、公开代价和权力坍塌。

## 7. 主要风险

- 豪门商战容易写成会议和文件，本轮避免把文件当主戏。
- 反同构不能过头，不能把源本强度洗淡。
- 公开直播和股权转让容易程序化，必须让人物选择和当众后果成为核心。
