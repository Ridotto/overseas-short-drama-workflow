## Step 1-2. 源本拆文包

### 1. 一句话故事核

女主为爱隐忍并隐藏顶级能力，男主因白月光误判而在生死危机中抛弃怀孕女主，导致孩子死亡；女主五年后以更高权力归来，夺走男主世界、撕开女配骗局，逼男主在追悔和自毁中跪求原谅。

### 2. 核心期待

观众追的是三条线同时推进：

- Eve 如何从被抛弃的妻子变成掌控局面的女王。
- Dante 什么时候知道自己亲手害死了真正的妻子和孩子。
- Lily 的假孕、陷害、偷情和车祸阴谋如何被当众反噬。

### 3. 首批分集功能

| 源本集数 | 功能 | 关键体验 |
|---|---|---|
| 1 | 婚姻背叛 + 怀孕信息被吞回 | 女主被当众替换，观众知道她怀孕，男主不知道 |
| 2 | 生死选择 + 弃救 | 男主救女配不救女主，直接造成流产 |
| 3 | 流产觉醒 + 五年后归来预告 | 女主从爱转恨，身份从主妇转为夜枭 |

### 4. 高价值节点

#### HV1：孕检单被吞下 / 离婚签字

- 客观事件：男主带怀孕白月光回家，逼女主离婚；女主隐藏自己的孕检单并签字离开。
- 观众体验功能：极端委屈、信息差、压抑蓄力。
- 强度来源：观众知道女主也怀孕，男主不知道；女主把唯一求救证据吞回去。
- 关系 / 权力变化：妻子从家中正位跌成被驱逐者，白月光进入主位。
- 信息差状态：女主怀孕只有女主和观众知道。
- 卡点停法：女配露出真面目，观众知道下一场要害女主。
- 源本高辨识外形：黑帮庄园、孕检单、白月光装摔、离婚协议。
- 下游写作抓手：改成家族企业继承权场景：妻子带着孕检和救公司方案等丈夫，丈夫带“怀孕未婚妻”进继承发布会，并逼她签署离婚和股权放弃。

#### HV2：车祸火海弃救

- 客观事件：两车同时出事，男主选择救 Lily，把怀孕 Eve 留在爆炸车里。
- 观众体验功能：硬创伤、男主罪责不可洗、女主黑化动机成立。
- 强度来源：不是误会，而是现场选择；男主听到女主求救仍转身。
- 关系 / 权力变化：婚姻彻底死亡，女主对男主的爱转成恨。
- 信息差状态：男主仍相信女主恶毒，观众知道女主被陷害。
- 卡点停法：爆炸吞没女主。
- 源本高辨识外形：城郊车祸、卡车、火海。
- 下游写作抓手：改成 Bexley Tower 智能车库 / 货梯火灾事故。两处救援同时发生，Adrian 下令把应急电力和救援队切给 Chloe 的 VIP 玻璃电梯，Serena 被困在燃烧货梯里并流产。保留“当场选择”和“烈性身体后果”，避开车祸外形。

#### HV3：流产醒来 + Victor 扶起

- 客观事件：Eve 得知孩子没保住，被 Victor 认出夜枭身份并扶持复仇。
- 观众体验功能：女主黑化觉醒，复仇引擎点火。
- 强度来源：刚失去孩子，马上看到男主为白月光孩子庆祝。
- 关系 / 权力变化：女主从依附男主转向自建势力。
- 信息差状态：男主不知道自己害死孩子；女主知道要向所有人讨债。
- 卡点停法：五年后女王归来。
- 源本高辨识外形：私立医院、俄罗斯黑帮首领。
- 下游写作抓手：改成 Julian Cross / 对冲基金兼敌对资本方救下 Serena，五年后她以“Northstar Capital / Nova Frost”身份回归。

#### HV4：拍卖会回归 + 扇 Lily

- 客观事件：Eve 以夜枭身份出现在黑帮拍卖会，压 Dante，扇 Lily。
- 观众体验功能：身份反转、当众打脸、权力震撼。
- 强度来源：曾经被扫地出门的人，现在掌控男主命脉。
- 关系 / 权力变化：Dante 从可支配者变成被掐供应线的人。
- 信息差状态：众人刚知道 Eve 就是夜枭。

#### HV5：假孕 / 非亲生公开

- 客观事件：Eve 带黑医和录音揭穿 Lily 假孕、买凶、偷情，亲子鉴定确认孩子不是 Dante 的。
- 观众体验功能：女配被公开剥皮，男主高位塌陷。
- 强度来源：男主维护五年的“孩子”和体面瞬间变成嘲笑来源。
- 关系 / 权力变化：Lily 从受宠者变成被掐住脖子的骗子；Dante 从护她变成想杀她。
- 信息差状态：Dante 终于知道自己被绿，但还没完整承担弃救罪。
- 卡点停法：Lily 反咬“Eve 流产也有你一份”。
- 源本高辨识外形：黑医、床照、亲子鉴定、现场验血。
- 下游写作抓手：改成继承信托认证仪式。Chloe 的儿子要被刻上 Bexley heir wall，Serena 强制启动生物认证和信托法务认证，名字被红色封条覆盖，现场拒认继承人。证据触发“继承墙熄灯 / 信托冻结 / 记者直播”，不做文件宣读。

#### HV6：保险箱旧物 + 男主跪雨

- 客观事件：Dante 找到孕检单、婴儿毛衣、尾戒，得知双胞胎，崩溃跪雨自扇。
- 观众体验功能：火葬场核心兑现，男主罪责从理性知道变成身体崩溃。

- 信息差状态：男主完整知道孩子存在和自己伤害。
- 卡点停法：冲向女主，准备公开交出一切。
- 源本高辨识外形：废弃仓库保险箱、孕检单、婴儿毛衣、暴雨跪地。
- 下游写作抓手：改成 Bexley Tower 火灾旧封存层 / Serena 的旧保险柜，旧 ultrasound、未完成 baby blanket、Adrian 曾丢弃的尾戒和火灾手环。跪雨可改成城市高架雨中跪在 Bexley 大楼门前。

### 9. 写作手法 / 表达手法

- 高压节点要用现场行动承载：电力切换、货梯坠落、信用线冻结、继承墙熄灯、股权表决权转移、刑责 confession 提交。

- 台词短，尽量让人物用选择和身体反应表达。
- 结尾不把下一集结果说完，停在动作前一拍。

### 10. 水分 / 烂点 / 禁止复用外形

## Step 4. 洗稿边界包

### 必须保留的体验功能

- 女主被男主当众抛弃时，观众知道她怀孕 / 有关键隐情。
- 男主在危机现场做错选择，造成不可逆身体和情感伤害。
- 女主五年后以更高权力归来，先让男主和女配当众失去体面。
- 女配假孕 / 假继承人骗局要从“受宠优势”变成“公开羞辱”。
- 男主知道真相时，必须承担“我亲手害死孩子”的事实，不能只恨女配。
- 第 10 集必须有男主公开跪求和交出权力，但不让女主原谅。

### 必须避开的源本外形

- 黑手党 / 军火 / 黑帮会场 / 枪抵额头。
- 重型卡车车祸。
- 黑医口供 + 床照 + 亲子鉴定文件宣读的单一听证会形态。
- 废弃仓库保险箱照搬。
- 枪响黑屏作为第 10 集默认解。

### 可以自由换的外壳变量

- 家族名：Bexley。
- 女主：Serena Vale，前 Bexley 继承人之妻，隐藏身份是 Northstar Capital 创始人和 Bexley 风控架构的真正设计者。
- 男主：Adrian Bexley，家族企业继承人 / CEO。
- 女配：Chloe Rowe，伪孕未婚妻，实际孩子来自外部投资人情人。
- 男配：Julian Cross，敌对资本方 / Serena 的五年后盟友。
- 权力载体：继承墙、债权触发权、供应链合约、信用线、董事会投票、家族信托生物认证。

### 必须删除 / 修复的烂点

- 商业机制不能像开挂，要有可视化触发：屏幕变红、门禁换权、信用终端冻结、继承墙熄灯。
- 第 7-8 集不能只听证，必须让 Chloe 的权力当场被拿走。
- 第 10 集不走枪响，改成法律 / 股权 / 刑责不可逆提交前一拍。

### 新壳承载风险

- 豪门商战容易冷：必须把证据写成现场权力变化。
- 家族企业壳容易解释过多：每集只保留一个核心机制。
- 男主罪责容易被 Chloe 转移：第 9 集用旧物和当年应急调度命令把罪责压回 Adrian。

## Step 5. 新本 beat sheet / outline

### Story Engine

- 主角想要什么：Serena 要夺回被 Adrian 和 Bexley 家族抹掉的尊严，并让害死孩子的人付出不可逆代价。
- 阻碍是什么：Adrian 掌握 Bexley 家族名义权力，Chloe 占据“怀孕 / 继承人母亲”的道德和家族位置，董事会只认公开身份。
- 赌注是什么：Serena 的孩子之死是否会被永远掩埋；Bexley 家族控制权归谁；Adrian 是否能继续逃避弃救罪责。
