# PRD_v22_latest_v11_HighPressure_Scene_Options

## 节点 A：Ep1 妻位被夺

### 候选 1：家族晚宴签离婚 + 信托退出

- 场面一句话：Serena 被带到 Blackwood 家族晚宴，Ava 被迫在家族长桌边签下离婚和信托退出。
- 谁压谁：Adrian / Serena 压 Ava。
- 谁旁观：董事亲属、管家、家族律师。
- 谁被迫选择：Ava 在公开羞辱和保住双胎秘密之间选沉默。
- 身体 / 关系 / 社会性代价：Ava 被从主座移走，名字从家族屏上删除。
- 爽点前怎么压低：所有人祝贺 Serena 的“继承人”。
- 爆点是什么：Ava 吞下双胎报告，不给他们利用孩子。
- 可见后果 / 失去 / 反扑 / 卡点：Ava 的餐盘和名牌被撤；Serena 低声安排车祸。
- 卡点停在哪个画面 / 动作 / 声音：Serena 在楼梯阴影里拨通司机电话。
- 信息边界：Adrian 不知道双胎。
- 同构风险：离婚协议仍接近源本，但信托退出和家族长桌是新壳承载。
- 短剧降级风险：如果只是吵架会弱，必须有名牌撤下和信托文件。
- AI 成片可视化风险：低。

### 候选 2：董事会直播撤妻位

- 场面一句话：Adrian 在董事会视频会议里宣布 Serena 是继承人母亲，Ava 被踢出家族基金会。
- 淘汰原因：太会议化，Ava 的羞辱缺少身体和空间压迫。

### 选中候选

选中候选 1。它保留源本低位羞辱，同时有家族长桌、名牌、信托文件和楼梯阴影电话这些可见后果。

## 节点 B：Ep4 女王回归

- 场面一句话：Ava 用 Vesper 授权压过 Adrian，全场终端显示 Blackwood 三条量产线暂停。
- 谁压谁：Ava 压 Adrian。
- 谁旁观：全球供应商、投行、媒体、Serena。
- 谁被迫选择：Adrian 在开除 Ava 和保住供应链之间失去选择。
- 身体 / 关系 / 社会性代价：Adrian 的竞价牌落地，Blackwood 团队集体看向他。
- 爽点前怎么压低：Serena 当众羞辱 Ava 是弃妇。
- 爆点是什么：Ava 说出自己是 Vesper，Blackwood 命脉在她手里。
- 可见后果 / 失去 / 反扑 / 卡点：供应终端红屏；断供通知贴到 Adrian 胸前。
- 信息边界：不提孩子。
- 同构风险：竞价场有源本拍卖影子，但权力载体从军火变成芯片授权和量产线。
- 短剧降级风险：不能只靠屏幕，必须让 Blackwood 团队现场撤席/电话爆响。
- AI 成片可视化风险：低。

### 候选 2：董事会投票夺席

- 淘汰原因：更像会议，视觉记忆点弱。

### 选中候选

选中候选 1。它有记忆点的可见后果：全场终端红、生产线电话爆、断供通知贴胸。

## 节点 C：Ep7-Ep8 假继承人坍塌

### 候选 1：信托门血脉认证失败

- 场面一句话：在 Blackwood 继承人信托仪式上，Serena 儿子的徽章被家族信托门弹落，继承身份当众失效。
- 谁压谁：Ava 压 Serena；现场机制压 Adrian 体面。
- 谁旁观：董事会、媒体、亲属、家族律师。
- 谁被迫选择：Adrian 被迫在护 Serena 和保 Blackwood 信托规则之间选择。
- 身体 / 关系 / 社会性代价：孩子衣领上的徽章弹落，Serena 跪地去捡；董事后退。
- 爽点前怎么压低：董事宣布继承人即将入册，Serena 高位微笑。
- 爆点是什么：信托门红灯拒开，亲父被带进场。
- 可见后果 / 失去 / 反扑 / 卡点是什么：Serena 失去“继承人母亲”保护伞；她反咬 Adrian 害死双胎。
- 卡点停在哪个画面 / 动作 / 声音：徽章滚到 Adrian 鞋边，Serena 嘶喊“you killed her twins.”
- 信息边界：Ep7 只揭伪造孕周和事故链；Ep8 才揭血缘。
- 同构风险：有亲子验证功能，但现场后果不是报告，而是继承门拒开和徽章弹落。
- 短剧降级风险：若台词解释太多会弱，要用仪式动作推进。
- AI 成片可视化风险：中，信托门和徽章要写清楚。

### 候选 2：董事席投票无效

- 淘汰原因：可见后果不如徽章弹落，有会议化风险。

### 候选 3：儿童医院捐赠冠名被撤

- 淘汰原因：太偏公益，不够贴 Blackwood 继承权核心。

### 选中候选

选中候选 1。它直接把“孩子不是 Adrian 的”变成现场身份坍塌，不只是一份报告。

## 节点 D：Ep10 公开跪交一切

### 候选 1：顶层露台 + 创始人徽章 + 投票权转让

- 场面一句话：Adrian 当众跪在顶层餐厅，把 Blackwood 创始人徽章、投票权和辞任文件捧给 Ava；Ava 打开露台门，让他跨出安全线。
- 谁压谁：Ava 压 Adrian。
- 谁旁观：Julian、投行代表、供应商、媒体。
- 谁被迫选择：Adrian 在“保命保体面”和“证明愿意付出一切”之间选择。
- 身体 / 关系 / 社会性代价：Adrian 膝盖撞地，双手举着徽章；所有人拍下 Blackwood 继承人跪地。
- 爽点前怎么压低：他雨夜崩溃后不顾阻拦闯入。
- 爆点是什么：他交出 Blackwood 控制权和自己身份。
- 可见后果 / 失去 / 反扑 / 卡点：Ava 打开露台门，风灌入餐厅，转让文件飞散。
- 卡点停在哪个画面 / 动作 / 声音：Adrian 一只脚跨过露台安全线，黑屏。
- 信息边界：不让他真死；不让 Ava 原谅。
- 同构风险：功能继承源本枪口自毁，但外形完全换成豪门商战高空权力仪式。
- 短剧降级风险：如果只签文件会弱，必须有露台危险动作。
- AI 成片可视化风险：中，注意安全线和高空风声。

### 候选 2：董事会直播辞职

- 淘汰原因：公开但不够身体化，像商业新闻。

### 候选 3：砸碎 Blackwood 家族印章

- 淘汰原因：失去感强，但不如露台选择有卡点。

### 选中候选

选中候选 1。它有公开羞辱、权力丧失、身体危险和黑屏卡点，符合首批大卡需求。
