# 03 洗稿边界包

## 本轮新壳

海外豪门商战：跨国矿业与救援科技集团、家族信托、港口项目、私人医疗、慈善基金、董事会继承权、媒体直播。

## 深度重构边界

本轮不保留源本事件外形，只保留体验功能和追看链。

允许大幅改：

- 事故空间；
- 男女主职业和权力来源；
- 反派作恶方式；
- 证据出现方式；
- 公开打脸场域；
- 旧物形态；
- 每集具体事件载体。

不允许改：

- 男主主动选错；
- 女主失子后死心；
- 女主高位归来；
- 旧爱继承人叙事分阶段塌房；
- 男主必须被孩子真相击穿；
- 第 10 集必须是“献上一切求原谅但女主不接”的卡点。

## 必须替换的高辨识外形

- 黑帮、军火、教父、枪战、黑道宴会。
- 卡车车祸 + 出租车爆炸。
- 地下黑帮审判。
- V5 的胸针 / 黑卡 / 封条整套动作簇。
- V27 的顶层餐厅、电梯火警、债权拍卖、root key / CEO button 等整套外形。

## 真相模型

```text
Serena 当年确实怀孕，但孩子不是 Ethan 的。
她伪造的是“Calder 继承人”身份、孕周口径和救援事故里的受害者身份。
Nora 当年怀的是 Ethan 的双胎；双胎在港口事故后的离岸医疗船上死亡。
```

## 防洗飞红线

- Ethan 不能只是“不知道 Nora 在危险里”；他必须听见她求救。
- 港口事故不能变成流程播报；必须是人在现场选择人。
- Nora 归来不能只是“监管官更高级”；必须让 Ethan 正在推进的项目当场停摆。
- Serena 不能只是撒谎小人；她要有资源、误判和继续作死的理由。
- Nora 在 EP10 不能软化。
