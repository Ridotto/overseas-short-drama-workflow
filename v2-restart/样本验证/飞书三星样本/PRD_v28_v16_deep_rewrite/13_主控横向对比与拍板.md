# 13 主控横向对比与拍板

## 本轮问题

只测试一个变量：

```text
默认重构力度 -> 深度重构
```

不额外加入新规则，不按上一轮根因判断定向修补。

## 总结论

```text
workflow execution: S0-S9 + S11 complete, S10 clean reviewer skipped
creative outcome: revise
delivery outcome: not pass
```

深度重构确实解决了一部分“像改名复刻 / 外形太近”的问题，也让 EP1-EP3 的可视化和冲击力比 V27 更好。

但它没有自动解决“后半段短剧场面发动不足”的问题。V28 仍未达到“明显打过 V5”的标准。

## 与 V5 对比

V5 是残缺半成品，但它有一个优点：高压节点的动作链非常直接。

- V5：胸针被扯、切电、血从电梯缝渗出、旧物击穿，记忆点粗暴但直接。
- V28：吊篮二选一、坠海、停工牌、红腕带、医疗船腕带，这些比 V27 强，但 EP6-EP8 仍有证据链连续推进的问题。

判断：

```text
V28 在防复刻和新壳完整性上强于 V5；
V28 在 EP1-EP3 场面冲击上接近 V5；
V28 在 EP6-EP8 和 EP10 的场面发动上没有稳定打过 V5。
```

## 深度重构变量是否有效

有效，但不是充分条件。

有效处：

- 大幅降低了“只是换名词”的第一眼风险；
- 让新壳更有独立性；
- 给 EP1-EP3 带来更强的可拍画面；
- 让旧版高辨识外形基本退出正文。

无效处：

- 深度重构不会自动保证后半段每一集都有强场面发动；
- 不会自动把证据链变成人对人的强戏；
- 不会自动让反派反扑燃料进入正文；
- 不会自动让 EP10 的求赎卡点摆脱机构按钮。

## 关键发现

深度重构不是当前问题的总解。

它能解决“外形距离”，但不能自动解决“场面发动”。

这意味着：

```text
如果只把默认力度改成深度重构，
产物会更新、更不像复刻，
但不一定更能让观众持续付费。
```

## 失败层归因

- S1/S2：基本可用。源本有效性和深度边界没有明显洗飞。
- S3：新故事骨架可用，但后半段反派反扑燃料没有压进 1-10 集正文。
- S4：EP6-EP8 的场面承载仍偏“证据如何出现”，不是“人如何被逼到动作”。
- S5：施工稿已经比任务单更像戏，但后半段仍没有把每个证据点变成足够强的人际冲突。
- S6/S7：表达层完成度比 V27 好，问题主要不是格式和台词，而是上游场面发动不足。
- S8：DNC 基本稳定。
- S10：未跑 clean reviewer，本轮不能声称 reviewer 通过。

## 主控拍板

本轮不建议把“深度重构”设为默认万能解。

建议结论：

1. 保留“中度 / 深度重构”两种力度。
2. 深度重构适用于用户明确要更大外形距离、降低复刻感的场景。
3. 深度重构不能替代 S3/S4/S5 的场面发动能力。
4. 若下一步继续修 workflow，重点仍应是让后半段的证据、真相、反转变成短剧现场行动，而不是只扩大改写力度。
