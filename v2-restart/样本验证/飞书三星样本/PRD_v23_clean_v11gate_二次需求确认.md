# PRD_v23_clean_v11gate_二次需求确认

## 1. 源本读后判断

初始需求“海外豪门商战”可以承载源本有效性，但必须把事件载体从“黑道暴力 + 车祸 + 证据大屏”改成“家族企业控制权 + 智能总部事故 + 信托继承权 + 公开商业后果”。

如果只把：

- 黑手党家族改成财团；
- 军火线改成供应链；
- 黑帮宴会改成董事会；
- 亲子鉴定改成法务文件；

会直接落入改名复刻或短剧降级。

## 2. 本轮继承项

必须强继承：

- 女主低位隐忍、被低估、被替换。
- 男主在危机中选错人，造成不可逆失子。
- 五年后女主高位归来，第一轮就有当众权力释放。
- 女配谎言不是一次揭完，而是先裂外部权力，再裂男主尊严，再反咬男主罪责。
- 第 9-10 集保留男主从事实打击到公开下跪的火葬场节奏。

## 3. 必须替换的源本外形

- 车祸 / 火海 / 两车救人：替换为 King Tower 智能安全封锁事故。
- 黑帮拍卖 / 军火断供：替换为全球供应链峰会 + 债权控制权收购。
- 大屏证据 / 亲子报告 / 当场验血：替换为继承人信托现场解锁失败、投票权冻结、相关人物当场护子暴露。
- 仓库保险箱 / 婴儿毛衣：替换为被封存的“家庭育婴室 + 双胎信托文件 + 胎心吊坠录音”。
- 雨中街头跪：替换为顶层股东庆功宴 / 全球投资人面前公开交权跪求。

## 4. 可自由改的变量

- 人名、公司名、行业：全部可换。
- 女主隐藏能力：从武器设计 / 黑客，改为 Blackline Capital 创始人、危机收购和供应链风控操盘手。
- 男主权力：从黑手党少主，改为 King-Sterling 集团继承人 / CEO。
- 女配资源：从白月光假孕，改为利用假继承人和家族信托占位。
- 男配：从战斧帮首领，改为 Voss Capital 掌门人 / 女主救命投资人。

## 5. 必须修复的烂点

- 源本孩子 / 双胞胎信息冲突：本版拍板为双胞胎，Ep9 前不完整释放。
- Ep7-8 不能靠报告完成高潮：必须让现场有人失去席位、印章、信托投票权、保安控制权。
- Serena 不能只是拿材料审判：她必须亲自布置场、控制入口、调换安保、逼选择。
- Adrian 的罪责不能只靠 Chloe 喊：Ep8 只点破“你亲手选错”，Ep9 用私人物件完成真正击穿。
