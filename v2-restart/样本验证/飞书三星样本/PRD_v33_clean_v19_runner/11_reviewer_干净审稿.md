## 首轮必须返修项（已修）

### 1. EP6 把“双胎”明牌过早说出，挤坏了 `7-10` 的真相梯度

- 类型：`提前消费`
- 问题：
  - `08_台词视听polish稿_首批1-10集.md:289-300` 里，Serena 在车库对峙时已经直接说出 `So did the twins inside me` / `You left three lives there`。
  - 这会让 Adrian 在 `EP6` 就拿到“双胎数量”这个关键信息，而不是先被伤疤和旧债刺中、再一路查到后面。
  - 但控制包和蓝图给这段定的功能不是“双胎明牌”，而是“伤疤提示 / 起疑 / 进入真相通道”：
    - `04_故事控制包.md:38-47` 把 Adrian 的认知变化写成：`7-8` 先击穿“假孕/假继承人/自己不是被动误会者”，`9-10` 再进入“双胎 + 毁掉两个孩子”的新认知。
    - `04_故事控制包.md:70-74` 明确排的是：`Ep6 疤痕对视 -> Ep7 假怀孕崩盘 -> Ep8 假继承人崩盘 -> Ep9 双胎物证穿心 -> Ep10 狗跪交权`。
    - `09_story_memory_checkpoint.md:7` 也把过线顺序写成：`伤疤提示 -> 假孕崩盘 -> 假继承人崩盘 -> 双胎物证击穿 -> 公开跪求`。
  - 源本对应位置也没有在这一步明说“双胎”，而是先说“我肚子里的孩子”，之后再通过医疗记录和保险箱物证逐层坐实：`源本:205-220`、`源本:274-276`。
- 为什么影响产品目标：
  - 这个项目首批 `1-10` 的追更引擎，本来就靠“债不是一次说完，而是一层层翻上来”。
  - 现在 `EP6` 先把“双胎”掀开，`EP8` 的崩喊和 `EP9` 的保险箱物证就会被削成“补证”而不是“第一次不可抵赖的穿心”。这会直接削弱 `9-10` 应该承担的付费点强度。
- 建议回哪层修：
  - 回 `S5/S6` 修场景施工与正文，不必回 `S3` 洗稿总策略层。
- 方向性 suggestion：
  - `EP6` 只保留“伤疤 + 怀孕/孩子/当晚你把我留在火里”这一级，不要提前报出 `twins`。
  - 如果要保住 `EP8` 的公开剥皮，可以让 Chloe 在崩溃时先喊“她当时怀着你的孩子”，把“双胎数量”继续留给 `EP9` 保险箱物证一次击穿。
  - 如果坚持 `EP8` 保留双胎字样，那 `EP6` 也必须降回“伤疤提示”，不能再先把数字说穿。
