# 故事控制包

## 1. 真相账本

### 已存在真相

1. `Serena Vale` 在婚内一直隐藏身份，她不是花瓶太太，而是全球困境港口与特种物流操盘手 `Raven`
2. `Adrian Ashford` 误把 `Chloe Mercer` 视为自己唯一值得保护的女人和“继承人母亲”
3. `Serena` 当年已经怀上双胎，但还没来得及告诉 `Adrian`
4. `Chloe` 伪造怀孕和继承人叙事，并买通人制造事故
5. 事故现场 `Adrian` 做出了真实选择：先抱走 `Chloe`，把 `Serena` 留在燃烧车里
6. `Serena` 失去双胎，五年后以 `Raven` 身份回到纽约，先收资源，再收人
7. `Victor Sokolov` 既是 `Serena` 的扶持者，也是会在关键时刻衡量自身收益的交易型盟友

### 首批必须公开到什么程度

- 到 `10` 集末，必须让 `Adrian` 知道双胎、弃救、假继承人三件事同时成立
- 到 `10` 集末，必须让围观世界知道：
  - `Serena` 回来了，而且比 `Ashford` 家更高
  - `Chloe` 的白月光神话是假的
  - `Adrian` 已经从统治者跌成公开下跪的人

### 首批不能公开到什么程度

- 不能在 `1-6` 集就把所有物证打光
- 不能在 `10` 集前让 `Serena` 对 `Adrian` 出现可被误读成原谅的明确动作
- 不能提前打开 `Victor` 的后手和外部旧家族总攻，这些留给后两阶段

## 2. 角色错信账本

### Serena

- 错信已破：婚姻和隐忍能换来被珍惜
- 当前认知：`Adrian` 的悔只是迟来的自我赎罪，不等于她必须回头

### Adrian

- `1-6` 集错信：
  - Serena 只是普通豪门太太
  - Chloe 怀的是自己的孩子
  - 事故是 Serena 针对 Chloe 的恶意
- `7-8` 集被击穿：
  - Chloe 的孕与子都是假的
  - 自己不是被动误会，而是真正做错选择的人
- `9-10` 集新认知：
  - Serena 怀的是双胎
  - 他不是失去了一个婚姻机会，而是亲手毁掉两个孩子和一个未来

### Chloe

- 错信：
  - 只要抓住继承人叙事，就永远能压住 Serena
  - Adrian 再怎么发怒也会先护住家族体面和她

### Victor

- 错信：
  - 自己是 Serena 唯一可用的高位盟友
  - Adrian 只要跪了，也只是个可供玩弄的废人

- 允许把黑帮资产换成港口 / 冷链 / 家族 proxy / 资产托管
- 不允许把高压节点改成静态流程节点

## 4. 全剧阶段骨架

### Phase A `1-10`

主题：被烧死的婚姻回来了，而且回来就先拿你的命门

- `1-2`：婚姻当场被夺，双胎被埋，事故弃救成原罪
- `3-4`：Serena 在更高位归来，当众切开 Ashford 的资源喉咙
- `5-6`：Adrian 开始意识到自己从没真正认识过 Serena，也开始被旧伤击中
- `7-8`：Chloe 的假怀孕和假继承人神话在公众面前连续崩塌
- `9-10`：Adrian 被双胎物证击穿，带着整个家族和尊严跪回 Serena 脚边

### Phase B `11-20`

主题：跪下不等于还清，旧帝国、旧情债和新盟友开始同时反扑

- Serena 不杀 Adrian，但把他留在活着赎罪的位置
- Chloe 被关、旧董事会与旧家族元老起反弹
- Adrian 为护 Serena 与自家董事会翻脸
- `Salvatore Bellandi` 代表旧港口家族入局
- Victor 对 Serena 与 Adrian 的靠近开始失衡

### Phase C `21-32`

主题：外部强敌借孩子与软肋开战，Serena 看清 Victor 也不是纯盟友

- Salvatore 利用 `Anna` 和 Adrian 设局
- 旧船厂 / 港口救援把 Adrian 和 Serena 的关系推到“仍不原谅，但命债互认”
- Victor 晚来几分钟，暴露交易心态
- Victor、Marco、Chloe 与残部联手绑走 Adrian
- Serena 亲自收 Victor 在纽约的明牌敌人
- 北美基本盘归 Serena，Victor 退回俄罗斯

### Phase D `33-37`

主题：Victor 不再抢地盘，转而毁 Serena 的招牌与信誉

- 中立家族开始疑虑 Serena 的信誉
- Victor 通过仿造货源 / 假弹壳 / 假账线抹黑 `Raven`
- Serena 必须用证据链和行动把“我能掌盘”这件事重新钉死
- Victor 从远程下绊子转向下一阶段正面对撞

## 6. 首批追看债务

到 `10` 集末必须留下：

1. Serena 没有原谅 Adrian，只是没在这一枪里直接处决他
2. Adrian 愿意交出一切，但观众还不知道 Serena 会怎么用这条“疯狗”
3. Chloe 虽然已崩，但还没真正被 Serena 慢慢清算
4. Victor 已经看见 Adrian 会成为真正变量，不会坐视
5. Ashford 家族丢掉体面和权力后，一定会有更大反扑

## 7. DNC

- 不提前消费 `Victor` 的正式背叛
- 不提前消费 `Salvatore` 的总攻
- 不提前消费 `Anna` 的绑架线
- 不让 `Serena` 在首批就明确回到爱情线
- 不让 `Adrian` 在首批通过几次挨打就“洗白”
- 不把后续最强的港口 / 船厂 / 舆论战资源提前拿来替代首批的婚姻创伤线

## 8. 新壳承载硬边界

必须用这些承载体系来写：

- 家族庄园内部秩序
- 港口 / 冷链 / 海运 / 投票权 / 继承位
- 私人拍卖、基金会晚宴、顶层餐厅、地下车库、储物 loft
- 雨夜、伤疤、旧物、签字、proxy、seal、门口保安和当众旁观者

不能依赖：

- 冗长法务条文
- 税务或合规讲解
- 纯后台黑客操作画面
- 会议记录式真相推进
