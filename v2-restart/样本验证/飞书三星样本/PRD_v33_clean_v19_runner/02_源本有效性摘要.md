# 源本有效性摘要

## 一句话故事核

一个把全部真本事和真心都藏进婚姻里的女人，被丈夫为了一个假怀孕的白月光亲手丢进火里；她带着双胎之死和更高位的力量回来，让那个高高在上的男人在众目睽睽下被真相剥光，再像狗一样跪回她脚边。

## 源本为什么能追

源本真正抓人的，不是“黑帮”这层皮，而是下面这五股体验引擎同时在推：

1. 家中背刺够狠  
   女主不是在战场上输，而是在自己最想宣布怀孕的那一刻，被丈夫带着另一个怀孕女人闯回家，直接被赶出门。羞辱发生在厨房和婚姻内部，代入极强。

2. 弃救创伤够绝  
   核心不是车祸本身，而是男主在现场做了选择：他明知道女主在呼救，仍然先抱走女配。这一下把“渣”升级成“不可回头的罪”。

3. 女主回归够高、够冷、够当众  
   五年后不是偷偷报复，而是带着一个高位身份公开回场，先拿资源掐男主命门，再打脸女配。观众爽点来自“她回来了，而且现在轮到你抬头看她”。

4. 反转不是一句解释，而是连续剥皮  
   假怀孕、假继承人、双胎流产、男主亲手弃救，这些真相不是一次讲完，而是一层层剥开，让男主和围观者同步崩。

5. 追妻火葬场的力度够极端  
   男主不是普通后悔，而是从冷酷暴君一路塌到公开下跪、交出一切、愿意拿命换原谅。这个极端落差构成源本后半程的付费动力。

## 首批 1-10 集的留存骨架

### 前三集：先把创伤钉死

- 第 `1` 集必须让观众看到“婚姻内部最狠的一刀”
- 第 `2` 集必须完成“弃救选择”，不能软
- 第 `3` 集必须完成“她没死，而且她以后会回来收账”

### 第四到第六集：回归即碾压，但先不给男主完整真相

- 女主高位登场要当众
- 男主先被掐资源和脸面，再被女主伤疤和“你烧死了我们的孩子”击中
- 这里不能太快把所有牌打光，要留 `7-10` 集的真相崩塌和跪求

### 第七到第十集：真相集中剥皮，男主崩到跪

- `7-8` 集要完成女配假怀孕、假继承人的公开崩盘
- `9` 集要完成男主通过物证意识到双胎和未出生未来都被自己亲手毁掉
- `10` 集必须给出真正像样的“狗跪”名场面，不是口头后悔

## 源本的高辨识有效点

这些是体验资产，不是外形资产：

- 女主在最卑微的家务场景里藏着最强身份
- 她把双胎证据自己吞掉，等于亲手埋掉最后一丝期待
- 火场呼救时男主转身离开
- 五年后女主回场先打女配，再掐男主命门
- 男主是在公开场合被一层层剥掉尊严
- 双胎物证不是报告结论，而是沾血旧物、未完成的婴儿毛衣、戒指等具体物件
- 第一次真正下跪不是私下，是在能让所有人看见的地方

## 必须继承的付费债务

- 他什么时候才会知道她当年怀的是他的双胎
- 他什么时候才会意识到自己不是普通出轨，而是亲手害死了两个孩子
- 女主到底要把他折磨到什么程度才算够
- 女配什么时候会从“假白月光”崩成“全场笑话”
- 女主回归后，男主的家族 / 帝国会先丢哪一块

## 情绪密度与信息密度判断

- 情绪密度：高  
  首批不是平推，而是羞辱、绝望、归来碾压、公开崩盘、痛悔连打。

- 信息密度：高  
  关键身份、真相、误判、物证、假继承人、双胎、资源命门都压在首批里。

- 情节密度：高  
  基本每集都在推进关系和权力，没有真正的纯过渡集。

## 人物关系 / 权力 / 对立压力源

### 女主

- 明面地位：被丢弃的前妻 -> 高位归来的掌局者
- 暗线驱动：不是重新争宠，而是让负债者一个个付账

### 男主

- 明面地位：高位继承人 / 家主
- 内在线路：盲信 -> 误判 -> 怀疑 -> 崩塌 -> 公开跪求

### 女配

- 功能：假白月光、假继承人制造者、前半程最可恨的外部引爆器
- 必须让她前半程嚣张到位，后半程崩得够难看

### 男配 / 新盟友

- 功能：女主重生后的力量支点，也是男主嫉妒与危机感放大器

### 对立压力源

- 前半程：女配 + 男主自身误判
- 中段：男主家族 / 旧帝国利益
- 后段：外部强敌 + 新盟友的潜在交易性

## DNC

- 不能把“女主回来报复”洗成纯商业复盘
- 不能把“男主后悔”写成几句道歉
- 不能把“假怀孕 / 假继承人”只用报告通知完成
- 不能把“她为什么恨”讲成抽象心理创伤，必须由可见事故和物件支撑
- 不能让女主太早心软

## 源本自相矛盾或含混点

- 原文里“军火 / 黑帮 / 地下世界”是外形，内在真正有效的是婚姻背刺、火场弃救、资源反制、公开剥皮，不能错把外形当核心
- 源文后半的暴力尺度很高，直接照搬会让新壳退回黑帮片；新壳需要保强度，但必须换成更匹配豪门 / 基建 / 港口权斗的承载方式
- 原文 `7-8` 集已经有明显的“大屏证据 + 亲子鉴定”倾向；新稿必须保公开崩盘，但不能沦为证据听证会
