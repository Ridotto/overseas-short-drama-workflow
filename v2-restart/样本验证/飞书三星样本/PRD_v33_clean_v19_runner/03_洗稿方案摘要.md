# 洗稿方案摘要

## 改写总策略

这轮不是把黑帮词换成豪门词。

真正要做的是：

- 保住“婚姻内部背刺 + 火场弃救 + 高位归来 + 公开剥皮 + 迟来下跪”的体验骨架
- 把黑帮外形换成海外家族企业 / 港口物流 / 继承权权斗的承载系统
- 让每个高压节点仍然靠人、空间、物件、当众后果来成立，而不是靠屏幕说明

## 强继承体验

以下体验必须强继承：

1. 女主在宣布双胎前一秒，被丈夫带着另一个怀孕女人踩进泥里
2. 事故现场丈夫做出“先救她，不救你”的选择
3. 五年后女主不是回来哭，而是带着更高地位回来掌控命门
4. 女配必须先嚣张后公开崩塌
5. 男主的悔，不是温和忏悔，而是被真相一层层剥光后失态下跪
6. 女主对男主的态度不是“还爱不爱”，而是“我凭什么放过你”

## 等效迁移外形

| 源本外形 | 新壳外形 | 迁移理由 |
|---|---|---|
| 黑帮家族 | 老牌跨洋家族企业 / 港口与航运帝国 | 仍有家族权力、继承位、公开体面和资源命门 |
| 军火线 | 港口配额、冷链航线、矿运与特种物流控制权 | 仍然能形成“命门被掐”的现实后果 |
| 黑帮拍卖会 | 私人海事资产拍卖 / 家族资本晚宴 | 保留高位回归的公开性 |
| 教父 / 少主 | 家族控股继承人 / 董事会主席 | 保留支配地位与公开体面 |
| 地下世界女王 | 全球基础设施女王 / “Raven”资本操盘手 | 让女主回归更像权力升级，不是简单黑道升级 |
| 帮派大会 | 基金会晚宴 / 行业峰会 / 资本圈公开场 | 方便当众打脸、继承人崩塌和家族名声受损 |
| 仇家枪战 | 旧港口家族的雇佣安保与码头火力冲突 | 保住动作压力，同时贴近新壳 |

## 自由改写变量

这些可以大胆重写：

- 家族姓氏、城市路线、企业品类、晚宴外形
- 女主隐藏能力的具体业务：从地下武器设计改为全球港口 / 物流 / 困境资产操盘和暗盘供应链控制
- 男配的帮助方式：从黑帮扶持改为东欧港口联盟 / 资本盟友
- 后续敌人的对外外壳：旧港口家族、境外财阀、假弹壳抹黑可以改成假货源 / 仿造标识 / 舆论围猎

## 必须删除的烂点

- 纯黑帮名词替换式改写
- 女主只坐在电脑前查账
- 把强场面挪去会议室、档案室、合规流程
- 男主只靠听女配一句嘴炮才知道自己有罪
- 连续几集只靠 DNA 报告 / 视频投屏推进

## forbidden shape

- “她现在是审计总监 / 法务总监”式冷职能化降级
- “拿出一份报告，大家都懂了”式剧情偷懒
- “先开会，再念结论”式推进
- “车祸改成财务陷害”式降强度
- “跪求”只发生在私密室内、没有旁观者成本

## 新壳可承载范围

允许承载的高压场：

- 家族庄园厨房 / 早餐厅 / 主宅门口
- 雨夜高架 / 海边公路 / 地下车库
- 私人拍卖会 / 慈善晚宴 / 海运资产峰会
- 顶层套房 / 家族 war room / 港口仓区 / 旧船厂
- 高层旋转餐厅 / 董事会投票前厅 / 私人病房

这些场都要满足一个原则：

> 冲突先发生在人的身体和位置上，再落到资源和权力变化上。

## 例外风险

- 如果把女主回归写得过于像完美资本玩家，会洗掉她的疼和恨；必须保留她回场时的冷、狠和旧疤
- 如果把男主过早写得太可怜，会洗掉他前两集的原罪；他的坍塌必须建立在先知罪、再见罪、后见痛
- 如果把 `7-10` 集全写成真相公布，会丢掉“公开崩盘 + 物证击穿 + 狗跪”的层次递进
