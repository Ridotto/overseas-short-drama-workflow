# 二次需求确认结论

## 验证态说明

本轮是飞书三星样本的 workflow 候选验证。

由于没有真实用户逐步在线确认，以下内容视为“验证态冻结结论”：

- 以 `当前样本测试夹具.md` 作为需求冻结
- 以本轮 runner 指令作为范围冻结
- 允许进入蓝图与正文下游，但不宣称真实用户已正式量产拍板

## 确认后的核心判断

### 最需要保住的追看 / 付费效果

- 婚姻内部最狠背刺
- 事故弃救造成的不可逆原罪
- 女主高位回归后的连续公开剥皮
- 假继承人崩盘和双胎物证击穿
- 男主在众目睽睽下交出一切、跪到地上

### 改写力度与变量边界

- 改写力度：中度偏重
- 保体验，不保黑帮外形
- 外形必须进入豪门企业 / 港口物流 / 家族继承权系统
- 允许动作戏和安保冲突，但它们服务于家族权力斗争，不回到黑帮片壳

### 新本故事核

一个被写成装饰性太太的女人，在丈夫和假继承人母亲联手把她烧死后，以全球港口与供应链女王身份回场，让那个高高在上的家族继承人亲手看见自己毁掉的双胎、帝国和体面。

### 全剧阶段与首批范围

- 全剧先按 `37` 集理解
- 本轮只写 `1-10`
- `1-10` 的责任是：
  - 完成原罪建立
  - 完成女主回归与第一轮资源碾压
  - 完成假继承人崩塌
  - 完成双胎物证击穿
  - 完成公开跪求卡点

### 不能提前消费的信息

- Victor 的正式背叛
- Salvatore 的总攻
- Anna 的绑架与后续救援
- Serena 对 Adrian 的任何实质性原谅
- Lily / Chloe 的终局清算

### 不能复用的源本高辨识外形

- 黑帮家族、教父、帮派拍卖会、军火黑市这些显性词壳
- 纯黑帮式枪口威吓作为一切冲突默认动作
- 把“地下世界女王”原封不动移植

### 输出语言与正文用途

- 分析和控制材料：中文
- 正文：英文 screenplay
- 用途：样本验证 / 审稿样本
