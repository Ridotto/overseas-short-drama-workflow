# PRD v15-v4workflow reviewer round2 源本审计

## 结论

**source_audit_pass**

- **failure_layer**：无阻断性 failure layer。源本迁移链路没有断。
- **callback_target**：无强制返修；可在 polish 阶段回调 `正文 E5 / E8` 两个轻微风险点。
- **reason**：V15 正文完整继承了源本 1-10 的核心留存功能：女主被替代、男主亲手误判并二选一放弃、身体和双胎损失、五年后高位回归、白月光继承人叙事公开裂开、男主从震惊到主动追证再到公开跪求，结尾停在女主是否公开毁掉他的 live button 前。新壳没有洗成单纯公司权斗，也没有复刻黑手党/车祸/拍卖/亲子报告/雨中跪地枪口这一整套源本外形。

## 审计边界

本轮只审以下链路：

```text
Source Bible
-> Source Retention Anchor
-> Story Operating State
-> Episode Plan
-> Pressure-chain Writer Brief
-> 正文首批 1-10 集
```

未读取作者自检、rewrite report、manifest、版本对比、旧版正文。

## 链路判断

### 1. Source Bible -> Source Retention Anchor：通过

Source Bible 抽出的不是表层事件，而是源本有效体验：

- 被旧爱和白月光一起抹掉；
- 男主在女主求救时选择别人；
- 女主失去孩子和旧身份；
- 五年后以更高权力回归；
- 白月光当众崩盘；
- 男主发现自己亲手造成孩子死亡；
- 男主公开跪求，女主握住原谅/毁灭权。

Source Retention Anchor 没有把这些压缩成商业概念，仍保留了身体损失、孩子损失、公开羞辱、男主同罪、女主不赦免这些关键功能。

### 2. Source Retention Anchor -> Story Operating State：通过

Story Operating State 明确把 Serena 的目标定为“让 Blackwell 家族、Chloe 和 Adrian 看见自己当年被他们抹掉的代价”，不是拿钱或抢职位；也明确 Adrian 的恐惧是公开承认自己不是被骗而已，而是亲手放弃怀孕妻子。

这一层没有洗飞。它把源本的黑帮权力，正确迁移成 Blackwell 家族、基金会、媒体、医院和资本场，但情绪核心仍是“谁有资格审判谁”。

### 3. Story Operating State -> Episode Plan：通过

Episode Plan 的每集功能基本对齐留存锚点：

- E1：发布会后台替代，brooch / ultrasound 承接“女主人位置被抹掉”。
- E2：Adrian 签 folder，又在火场只有一把 cutter 时先救 Chloe，男主罪责成立。
- E3：Serena 摸空小腹，电视里 Adrian 护 Chloe，五年后黑衣女人入场。
- E4-E5：回归压场、打 Chloe、逼出 witness 线。
- E6：身体旧伤先刺 Adrian，再进入事故物袋。
- E7-E8：孩子喊 Bennett “Daddy”，Chloe 反咬 Adrian 也看见流血。
- E9：Adrian 主动回旧现场追回第二只腕带。
- E10：Adrian 在 press room 跪下说出自己的选择，Serena 卡在 live button。

Episode Plan 没有让证据、律师或董事会替代人物行动。

### 4. Episode Plan -> Pressure-chain Writer Brief：通过

Writer Brief 保住了每集压力链，没有把审计表塞给 writer。尤其 E6-E10 的指令是正确的：

- E6 要求身体旧伤优先于文件；
- E7 不能靠报告硬砸，要孩子本能喊错父亲；
- E8 要 Chloe 为保命反咬 Adrian；
- E9 要主动追查而非证据礼包；
- E10 要公开跪求，女主握住处刑权。

这层没有发现方向漂移。

### 5. Pressure-chain Writer Brief -> 正文：通过

正文首批 1-10 集实际执行了主要压力链。

关键证据：

- E1 中 Chloe 戴 Serena 的 Blackwell spouse brooch，Adrian 要她签字，Chloe 踩脏 ultrasound，Adrian 撕下 brooch 并驱逐 Serena；结尾 Chloe 把 red folder 交给 investigator。对应源本 E1 的替代、孕情被踩、离场仍被害。
- E2 中 Serena 当众说出双胎，Adrian 先签 folder，再在 glass corridor / Chloe 之间选择 Chloe，火门落下，Serena 留下血手印并倒在玻璃后。对应源本 E2 的生死二选一和不可逆身体/孩子损失。
- E3 中 Serena 醒来摸空小腹，得知双胎没了，同时电视里 Adrian 继续保护 Chloe；五年后黑衣女人入场不摘墨镜。对应源本 E3 的失子断裂和回归钩子。
- E4 中 Serena 用 North Capital 买下命名权、当众打 Chloe、沉掉 donor plaque，让 Blackwell 房间权力转向她。对应源本 E4 的回归压场和第一轮打脸。
- E6 中 Adrian 先误触 Serena 小腹、看见手铐旧伤，再追查 hospital / Lydia / evidence bag。不是先读报告后震惊，保住了“旧伤刺穿男主”的功能。
- E7 中 Noah 当众冲向 Bennett 叫 Daddy，Chloe 捂嘴并抢 guardian form，记者和董事看见 Adrian 的脸。对应源本 E7-E8 的继承人骗局公开裂开和男主颜面被挂起来。
- E8 中 Chloe 为保命说出“你签了 folder / 你看见她在玻璃后流血 / 我没有让你转身”，把 Adrian 从受骗者拖成共犯。对应源本 E8 的白月光反咬男主同罪。
- E9 中 Adrian 独自回废弃发布会后台，亲手撕开封板、看见旧血痕和 handrail，再到南门拿到第二只腕带并跪地自扇。对应源本 E9 的主动追旧物和追悔火葬场。
- E10 中 Adrian 在 press room 交出职位无效，被 Serena 要求对镜头说罪；他跪下说自己签 folder、选 Chloe、把妻子和孩子留在玻璃后，Serena 手停在 live button 倒计时。对应源本 E10 的公开跪求和女主不赦免。

## 风险项逐条判断

### 洗飞：未发现

V15 没有把故事洗成“女主夺回公司”或“资本复仇爽文”。商业场只是新壳，核心仍是丈夫误判、二选一放弃、孩子死亡、男主同罪和女主不原谅。

### 改名复刻：未发现

V15 没有复用源本的高辨识外形整串：黑手党庄园厨房、吞孕检单、出租车爆炸、夜枭军火女王、黑帮拍卖、亲子鉴定报告当众甩、仓库小毛衣、雨中跪地枪口。新本替换成发布会后台、glass corridor、hospital wing、North Capital、press room 和 live button，外壳成立。

### 短剧降级：未发现阻断

正文多数高潮由人物选择和身体动作推动，不是靠屏幕、文件、律师或董事会宣读：

- E2 是 Adrian 手握 cutter 选择 Chloe；
- E6 是身体触碰和旧伤先刺穿；
- E7 是孩子本能喊 Daddy；
- E8 是 Chloe 抱腿反咬；
- E9 是 Adrian 亲手回现场；
- E10 是 Adrian 跪着亲口说罪。

### 提前消费：未发现

信息释放节奏基本正确：

- E1-E2 让 Serena 说真话，但 Adrian 不信；
- E6 只给 Baby A 和第二只腕带线索；
- E7-E8 先爆 Chloe / Bennett / Noah，再把 Adrian 拖成共犯；
- E9 才补完整第二只腕带；
- E10 只停在 live button 半按，没有给原谅，也没有播出结果。

### 继承烂点：基本规避

源本 E6-E8 容易变成“证据连续播放 / 报告宣判”。V15 大体规避了这个问题。E7 没有用亲子报告硬砸，而是孩子喊错父亲；E8 没靠报告，而是 Chloe 情绪崩溃反咬；E9 的腕带也不是直接送到 Adrian 手上，而是经过旧现场、缺页 ledger、Lydia 要他说出“wife / children”的行动成本。

核心留存都在：

- 被替代；
- 被男主误判；
- 求救时被放弃；
- 失去双胎；
- 五年后高位回归；
- 白月光继承人叙事崩；
- 男主承认自己亲手害死孩子；

- 女主不赦免，停在处刑权前一拍。

## 轻微 callback，不影响 pass

### callback 1：E5 的“动利益”可更显性

Source Retention Anchor 要求 E5 “女主证明自己能动利益和动人”。正文 E5 的“动人”很强：Chloe 闯入、被 Serena 压到窗前、说漏 Lydia。但“动 Blackwell 利益”的可见度略弱，更多停留在 E4 的买房间 / E5 的 witness 线。

建议只做 polish：E5 开头可加一个很短的可视化后果，例如 Blackwell donor / lender 临时撤场、foundation transfer 被 North Capital 卡住，或 Chloe 闯入的动机明确为“你让三个捐赠方冻结了 Blackwell wing”。不要扩成商业解释。

### callback 2：E8 的公开围观感可再抬一档

E7 的公开羞辱足够强：记者、董事、孩子喊 Daddy、emergency contact 露出。E8 的反咬发生在 private hallway，虽然有“camera shutters still crack from the far end”，但公开压迫感比源本宴会厅稍弱。

建议只做 polish：让 E8 的记者/董事/医院人员隔着安保线能听见或拍到 Chloe “I didn't make you turn away” 这一句，强化“男主颜面被公众挂起”。不要改成证据播放，也不要让 Serena 解释。

## 最终判断

V15 可以作为本轮源本迁移通过稿进入下一轮质量审。两个 callback 都属于局部 polish，不需要回滚到 Source Bible / Episode Plan / Writer Brief 重跑。
