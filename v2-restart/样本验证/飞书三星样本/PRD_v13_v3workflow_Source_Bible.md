## 1. 首批核心体验链

源本 1-10 的有效性不是“黑手党 / 车祸 / 亲子鉴定 / 仓库旧物”这些外形本身，而是下面这条体验链：

```text
女主被旧爱 / 白月光替代
-> 男主当众误判女主，并做出明确抛弃选择
-> 女主承受不可逆身体与情感损失
-> 女主被外部势力救起，身份和能力重建
-> 五年后女主以更高权力姿态回到男主权力场
-> 反派继续作死，女主开始当众打脸
-> 男主先被女主吸引 / 震住，再被真相击穿
-> 白月光骗局撕开，男主尊严坍塌
-> 男主意识到自己亲手害死了自己的孩子和爱人
-> 男主跪下求原谅，女主不给轻易赦免
```

## 3. 强场面不是功能标签

源本强戏成立时，通常同时具备四件事：

1. **人对人压力**：Dante 对 Eve 说“先救 Lily”，不是系统自动锁门。
2. **身体和空间后果**：车门打不开、火逼近、Eve 拍窗、爆炸、流血。
3. **当众羞辱或权力倒置**：Eve 被打、Lily 被扇飞、Dante 绿帽当众曝光。
4. **钩子停在将爆未爆**：大门打开、高跟鞋入场；枪响黑屏；观众知道下一秒要更狠。

## 4. 源本烂点 / 不能机械继承

- 标志性桥段不能整串照搬：车祸、孕检单、亲子鉴定、小毛衣、雨中跪地如果一比一替换，只会像改名版。
- “文件 / 报告 / 大屏证据”可以存在，但不能成为高压戏的唯一发动机。
- 后半段如果全靠证据链，会变成听证会，不像短剧强戏。
- 女主回归后不能只做审计员，她必须亲手制造局面、压迫对手、逼对手出错。

## 5. 首批必须保住的情绪

| 情绪 | 源本形态 | 新本迁移要求 |
|---|---|---|
| 被替代 | 白月光怀孕登堂 | 新壳中必须是“女主被旧爱 / 家族 / 公众一起抹掉” |
| 冤屈 | 女主说怀孕没人信 | 女主说真话，被男主当成拖延 / 争宠 / 破坏集团 |
| 不可逆 | 车祸流产 | 新壳需要同等级身体与关系损失，不可只变成职位 / 权限损失 |
| 回归爽 | 夜枭压场 | 女主必须用新身份压住男主熟悉的权力场 |
| 打脸 | Lily 被公开撕开 | 反派不能被文件直接打败，要被当众逼到失态 |
| 追悔 | Dante 发现孩子 | 男主必须看到“自己当时的选择”造成的后果，不只是读到事实 |
| 拉扯 | 男主跪求，女主不原谅 | 男主越卑微，女主越冷，观众想看他还能付出什么 |
