# PRD_v22_latest_v11_Source_Bible

## 1. 一句话故事核

一个被丈夫和白月光联手逼到流产、被抛在火里的隐形强者，五年后以更高权力身份归来，公开撕碎白月光谎言，让迟来的丈夫从高位暴君变成跪地赎罪的人。

## 2. 核心期待

观众追的是三层期待：

- 女主什么时候从低位变高位，并让所有人知道她不是废物。
- 男主什么时候知道自己为了假货害死了真正的妻子和孩子。
- 男主跪到什么程度，女主会不会、以及凭什么不原谅。

## 4. 高价值节点

### S1. Ep1 离婚羞辱与吞下孕检单

- 客观事件：丈夫带怀孕白月光回家，要求女主签离婚；女主被诬陷推人，被打后吞下孕检单。
- 观众体验功能：把女主压到最低；让观众同时知道她怀孕和她隐忍的尊严。
- 强度来源：男主当面替外人夺妻位；女主把最该公开的真相吞下去。
- 关系 / 权力变化：妻子从名义正位变成被驱逐者；白月光夺位。
- 信息差状态：观众知道女主怀孕；男主不知道。
- 卡点停法：女配低声宣判“想走没那么容易”。
- 源本高辨识外形：黑帮庄园、孕检单塞进嘴里、离婚协议。
- 下游写作抓手：豪门商战可换成董事会晚宴 / 家族信托改签 / 妻位与继承人资格被夺；孕检单可变成胚胎报告、董事会监控盲区里的私密动作。

### S2. Ep2 事故抛弃与流产

- 客观事件：女配设计车祸，男女主同场；男主救女配，抛下被困女主。
- 观众体验功能：把男主的罪钉死，让女主黑化有血债。
- 强度来源：女主在火里求救并说怀孕，男主仍相信女配。
- 关系 / 权力变化：婚姻彻底断裂；男主从冷漠变成直接害人者。
- 信息差状态：男主不知道女主真的怀孕；女配知道自己作恶。
- 卡点停法：爆炸吞车。
- 源本高辨识外形：双车车祸、男主抱走白月光、爆炸。
- 下游写作抓手：商战壳可以保留事故作为身体代价，但事故诱因必须转为商业继承/信托/股权骗局，不是黑帮追杀。

### S3. Ep3 失子确认 + 新势力扶起

- 客观事件：女主醒来得知孩子没了；男配揭她隐藏身份，提出合作；五年后她回归现场。
- 观众体验功能：从虐转复仇，给女主起势支点。
- 强度来源：电视上男主正在庆祝假孩子，与女主病床失子并置。
- 关系 / 权力变化：女主从受害者转向复仇者；男配成为外部支点。
- 信息差状态：观众知道女主活着并开始反击；男主不知道。
- 卡点停法：五年后高跟鞋踏入高端场。
- 源本高辨识外形：黑医、武器设计师、黑帮拍卖会。
- 下游写作抓手：换成私立医院、隐名量化/芯片/风控系统创始人、全球投资酒会或股权拍卖。

### S4. Ep4 女王回归

- 客观事件：女主高价压过男主，揭示自己掌控市场线，是男主前妻。
- 观众体验功能：第一波爽点，昔日弃妇在所有人面前高位回归。
- 强度来源：男主的武力/权力威胁被女主供应链反制。
- 关系 / 权力变化：男主第一次低位；女配第一次恐慌。
- 信息差状态：男主知道她是前妻和夜枭，但还不知道孩子真相。
- 卡点停法：女主逼他“现在杀我”的对峙。
- 源本高辨识外形：黑帮拍卖、枪、军火线。
- 下游写作抓手：换成董事会竞价、债权/专利/芯片供给断供、安保与投票权，不靠枪。

### S5. Ep5-Ep6 女主切资源 + 男主发现孩子线索

- 客观事件：女主断男主军火线、惩罚女配；男主在车库听到孩子线索，查到怀孕病历。
- 观众体验功能：让男主从“她变狠”转向“我可能欠她血债”。
- 强度来源：伤疤与医疗记录不是抽象证据，而是身体后果。
- 关系 / 权力变化：男主开始动摇；女主仍冷。
- 信息差状态：男主开始知道女主当年怀孕，但还不知道女配孩子是假。
- 卡点停法：怀孕记录落到男主手里。
- 源本高辨识外形：车库对峙、伤疤、医疗报告。
- 下游写作抓手：保留伤疤/病历，但让资源切割变成供应链、债权、董事席、信托权限。

### S6. Ep7-Ep8 公开揭谎与反咬同罪

- 客观事件：女主公开带出黑医和证据，揭女配假孕、出轨和孩子非男主；女配反咬男主也害死女主孩子。
- 观众体验功能：强打脸 + 男主当众丢权威 + guilt bomb。
- 强度来源：男主不只是被戴绿帽，更被迫听见自己同罪。
- 关系 / 权力变化：男主与女配同盟崩塌；女主掌控公开叙事。
- 信息差状态：男主知道女配骗他，但还没完整消化女主孩子物证。
- 卡点停法：女主说“现在才知道，晚了”。
- 源本高辨识外形：黑医证言、大屏录音、亲子报告、当场验血。
- 下游写作抓手：必须避免只靠屏幕/报告。豪门商战版应选择有记忆点的公开后果：继承人信托仪式失败、董事席当场冻结、家族徽章被取下、男主亲手摘除女配身份。

### S7. Ep9-Ep10 迟来真相与当众下跪

- 客观事件：男主找到孕检单、婴儿衣物、戒指，雨中崩溃；当众跪求女主，交出全部。
- 观众体验功能：追妻火葬场启动，男主从高位暴君变成跪地赎罪者。
- 强度来源：私人遗物比报告更刺痛；公开下跪比私下道歉更降位。
- 关系 / 权力变化：女主彻底高位；男主开始失去身份和尊严。
- 信息差状态：男主终于知道全部早期血债，但女主不释放原谅。
- 卡点停法：女主提出“一命换两命”的残酷条件。
- 源本高辨识外形：保险箱、双胞胎孕检单、婴儿毛衣、雨中跪、枪响黑屏。
- 下游写作抓手：商战壳保留私人遗物 + 公开跪交权力；终局卡点可改成高空露台 / 创始人徽章 / 终身退出协议 / 危险选择，不照搬枪。

## 5. 钩子与卡点

- 入口钩子：低位妻子其实怀孕且有隐藏强身份。
- 早期卡点：女配要杀她；车爆炸；五年后女王回归；身份揭示。
- 中段卡点：男主查到怀孕记录；公开证据未完；女配反咬男主同罪。
- 首批大卡：男主当众跪求，女主给出不可能的原谅条件。

## 6. 爽点 / 压抑-释放

- 压抑：妻位被夺、耳光、事故抛弃、流产。
- 第一释放：高位回归、压价夺标、断男主资源。
- 第二释放：公开揭女配假孕和出轨。
- 第三释放：男主跪地、交出一切。
- 保留原则：释放不能提前变成女主原谅；女主每次赢都必须带可见后果。

## 7. 关系 / 权力变化

- 女主：隐忍妻子 -> 失子复仇者 -> 全球资本操盘者 -> 男主赎罪对象。
- 男主：家族/集团高位继承人 -> 被女配误导的施害者 -> 迟来知情者 -> 公开跪地赎罪者。
- 女配：伪弱者 -> 准继承人妻位 -> 谎言被拆 -> 反咬男主同罪。
- 男配：救命恩人与资本支点，早期不能变成真爱兑现，主要服务女主起势和男主嫉妒。

## 9. 写作手法 / 表达手法

- 用身体伤痕、公共仪式、身份标识和物件承载情绪。
- 每次证据都要连到现场权力变化，不能只播屏幕。
- 女主台词冷，动作先行；男主前期傲慢，后期崩塌；女配前期装弱，后期恐慌失控。
- 高压场面要有旁观者，旁观者的态度变化是爽点一部分。

## 10. 水分 / 烂点 / 禁止复用外形

- 禁止复用黑手党、军火、枪口抵头、黑帮拍卖、黑医、暗网武器设计等外形。
- 禁止把 Ep7-Ep8 写成屏幕报告一条条展示，人物只等结果。
- 禁止过早让男主深情解释或替自己洗白。
- 禁止把女主变成全知上帝，每个反击需要现场可见代价和准备痕迹。
- 禁止让男配早期替女主完成核心复仇；他只能扶起，不能夺主。

## 11. 源本冲突点

### 冲突 A：女配是假孕，后文又出现“男主养了五年的儿子”

- 源本表现：前文写女配伪造怀孕；Ep8 又有男主与“养了五年的儿子”的亲子鉴定。
- 可能解释：女配当年不是完全无孕，而是怀了别人的孩子，并伪造成男主孩子；“假孕”实际是假父亲/假报告。
- 不允许原样下传的风险：如果下游同时写“根本没怀孕”和“有五岁孩子”，新本核心真相会自相矛盾。
- 需要新本创作开发包拍板的问题：女配到底有没有孩子？孩子是谁的？男主被欺骗的是“怀孕事实”还是“父亲身份”。

### 冲突 B：女主孩子数量

- 源本表现：梗概多处写“孩子”单数；Ep9 孕检单写“双胞胎”；后文多次写“两个孩子”。
- 可能解释：源本后段为了加强火葬场把孩子升级为双胞胎。
- 不允许原样下传的风险：若前后不锁，会影响男主罪责强度和台词一致性。
- 需要新本创作开发包拍板的问题：新本首批采用单胎还是双胞胎；对应物证如何一致。

### 冲突 C：男主五年内对女配的关系

- 源本表现：女配享受“功臣”地位但迟迟未婚；同时男主公开对她和孩子保护极高。
- 可能解释：男主把她当救命/旧爱和继承人母亲，但仍因权力或潜意识不信任未正式给名分。
- 不允许原样下传的风险：如果新本里女配既完全掌权又没有任何正式身份，观众会问为什么。
- 需要新本创作开发包拍板的问题：女配在新壳中的身份是未婚妻、继承人监护人，还是董事会准夫人；她凭什么能继续作死。
