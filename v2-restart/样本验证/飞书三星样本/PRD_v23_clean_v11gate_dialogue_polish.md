# PRD_v23_clean_v11gate_dialogue_polish

## 1. Polish Goal

目标：英文台词短、准、有动作功能；不解释设定，不替动作说话。

本轮 screenplay 文件已按该口径写成，dialogue polish 主要记录关键压缩策略和保留线。

## 3. 压缩原则

- Adrian 前期台词偏命令式，不长篇辩解。
- Serena 台词少，动作多：签字、刷卡、关门、按键。
- Chloe 台词前期柔弱，危急时变尖锐和自保。
- Marcus 不说大段情话，只用旁观和一句提醒保留存在感。
- 所有解释性商业信息尽量放在动作前后：胸牌红灯、席位灯灭、徽章被摘。

## 4. 风险登记

- Ep7-Ep8 如果后续扩写，不能让董事或律师长篇解释金库规则。
- Ep10 如果后续扩写，不能让 Adrian 连续忏悔超过三句，否则会稀释白气卡点。

## 5. 结论

dialogue polish：pass。
