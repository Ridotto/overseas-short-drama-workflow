# 05 场面承载原料包

## 高压节点原料

| 节点 | 源本体验功能 | 新壳承载原料 | 禁止外形 |
|---|---|---|---|
| E1 夺位 | 正妻当众被替代 | 周年股东晚宴、联合创始人席位、家族戒指/董事权限被摘 | 厨房签离婚、吞孕检单整串 |
| E2 弃救 | 男主现场选择别人 | 地下装卸区火灾，两处被困；Adrian 亲手命令先开 Celeste 车门 | 原样公路车祸 |
| E4 回归 | 女主高位压回男主 | 债权拍卖厅，Ava 用 Nocturne 债权砸停 Adrian 竞拍 | 黑帮拍卖和枪口威胁 |
| E5 切资源 | 女主开始夺权 | 现场停掉港口信用、剪断 Celeste 慈善晚宴黑卡和捐赠墙 | CFO 口播 |
| E7 假继承人上台 | 谎言被迫进入公共验证 | 继承人基金发布会，Ava 要求当场采样，不给 Celeste 下台 | 会议听证纯证据播报 |
| E8 亲子叙事崩 | 反派保护伞被摘 | 大屏不只是报告，立刻触发名字从继承墙移除、保镖撤开、记者围堵 | 只念 DNA |
| E9 失子实物 | 男主被旧物击穿 | 烧焦的婴儿腕带、双胎超声、装卸区录音；他必须亲手读 | 文件播报 |
| E10 火葬场 | 男主交出一切求赎 | 股东直播、不可撤销控制权转让、当众认罪、Ava 卡在直播键 | 枪抵眉心、枪响黑屏 |

## 写给 writer 的可用原料

- 每场必须有一个“现场后果”：戒指落地、火门落下、名牌被拔、黑卡被剪、继承墙撤名、直播倒计时。
- 每场必须有人被迫行动：Ava 被迫沉默/逃生/亮权；Adrian 被迫选择/看见/认罪；Celeste 被迫公开验证/失去保护伞。
- 高压场不能只靠信息出现。信息出现后必须造成空间、身体、权力或物件变化。
