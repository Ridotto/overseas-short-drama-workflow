# 03 洗稿边界包

## 强继承

- 开场：正妻位置被公开剥夺，怀孕秘密被压下。
- 第 2 集：男主在高压现场选择反派，女主和孩子被抛弃。
- 第 3 集：失子后重生，五年后回归前卡点。
- 第 4-5 集：女主以新权力打脸，并开始切男主资源。
- 第 6-8 集：真相不是一次爆完，而是层层击穿男主错信。
- 第 9-10 集：男主真正面对失子，进入火葬场。

## 等效迁移

- 黑帮权力 -> 豪门商战控制权、董事会、信托、家族继承叙事、媒体和资产。
- 武器断供 -> 信贷、港口、酒店、基金、董事席位、家族名誉被切。
- 白月光假孕 -> 假继承人叙事 / 伪造亲子和产检文件。
- 黑帮复仇女王 -> 控股债权人 / 风险引擎 Nocturne 的实际拥有者。

## 自由改写

- 人名、公司名、地点、资产类别、职业身份、商业工具。
- 第 1-10 集具体场景外形，只要追看功能和情绪功能不丢。

## 必须删除或避开

- 源本整串高辨识动作。
- 靠文件、报告、会议完成高压场。
- 让女主一路碾压没有反派反扑燃料。
- 男主过早知道完整真相。
- 用枪口自杀替代豪门商战自然长出的毁灭性选择。
