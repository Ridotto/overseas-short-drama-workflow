# 02 源本拆文包

## 源本核心

源本不是靠黑帮设定本身好看，而是靠这条情绪链：

```text
妻子隐忍付出
-> 白月光怀孕夺位
-> 男主公开选择白月光
-> 女主失子死心
-> 五年后高位回归
-> 女主连续打脸夺权
-> 男主一点点发现自己杀死了真正的孩子
-> 男主从暴君跌成求原谅的人
```

## 首批 1-10 集追看功能

| 集 | 源本功能 | 观众追问 |
|---|---|---|
| 1 | 正妻被夺位、怀孕秘密被吞下 | 她会不会说出自己也怀孕？男主会不会后悔？ |
| 2 | 生死二选一，男主选择白月光 | 女主会不会死？孩子保不保得住？ |
| 3 | 失子、死心、复仇转向、五年后回归前一拍 | 她回来会有多狠？ |
| 4 | 女主高位回归，当众压过男主和反派 | 男主什么时候认出她？她会怎么打脸？ |
| 5 | 女主切男主资源，反派上门作死 | 反派会不会被反杀？ |
| 6 | 男主开始看见当年真相碎片 | 他会不会知道她怀过自己的孩子？ |
| 7 | 公开揭谎升级，反派的继承人叙事开始塌 | 假孕/假继承人会不会当众爆？ |
| 8 | 反派亲子谎言崩，男主错信崩第一层 | 男主到底害死了谁？ |
| 9 | 男主第一次摸到失子的实物证据 | 他会怎么崩？女主会不会原谅？ |
| 10 | 火葬场极限卡点 | 男主拿什么换原谅？女主会不会按下毁灭按钮？ |

## 必须迁移的体验

- 开场强羞辱：不是离婚信息，而是位置被当众夺走。
- 二选一弃救：不是事故本身，而是男主在现场主动选择别人。
- 失子死心：女主不是伤心一下，而是从爱转成复仇。
- 五年后回归：女主必须以更高权力进入，不解释，先压人。
- 爽点压放：打脸前必须有人不信、看衰、压她。
- 关系拉扯：男主后期不是简单道歉，而是每次靠近都被自己的旧选择反噬。
- DNC：男主不能太早完全知道“自己害死了自己的孩子”，否则第 9-10 集没有新刀口。

## 源本高辨识外形禁用

- 厨房端饭签离婚；
- 吞孕检单的整串动作；
- 公路车祸、男主抱走白月光、女主车里爆炸的原样组合；
- 黑帮拍卖会举牌、枪口抵胸/眉心；
- 枪响黑屏作为第 10 集卡点。
