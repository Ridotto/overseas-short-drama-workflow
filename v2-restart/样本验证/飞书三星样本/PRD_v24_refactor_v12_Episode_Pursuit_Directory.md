# PRD_v24_refactor_v12_Episode Pursuit Directory

## Episode 1

- Title：The Wife No One Saw
- Summary：Serena 在 Blackwell Tower 后台修复能救公司的风控模型，准备告诉 Adrian 自己怀了双胞胎；Adrian 却带着假孕的 Chloe 入场，逼她签离婚和放弃配偶信托席位。
- Opening Hook：CEO 妻子一边被当成装饰，一边在后台阻止一场十亿美元风控事故。
- Hook Type：身份信息钩 + 情绪钩。
- Core Friction：Serena 的真实价值无人知道，Chloe 用“继承人”抢她位置。
- Pressure Setup：董事、律师、媒体都在场，Serena 没法失控。
- Max Spike：Chloe 假摔，Adrian 当众认定 Serena 伤害孕妇。
- Satisfaction Type：压抑蓄力。
- Visible Loss / Gain：Serena 失去妻子席位和 Blackwell 配偶信托权限。
- Change：Serena 从想分享孕情，变成沉默签字离开。
- End Button：Chloe 给安全主管发消息：“Drop her elevator before she leaves.”
- Next-Episode Payoff：电梯事故必须发生。
- Do Not Consume：不说出 Serena 的 Sable 身份。
- Source Function Ref：HV1。
- Forbidden Shape：庄园厨房、吞孕检单、黑帮少主。
- Opposition Fuel：Chloe 掌握楼宇安全主管和假孕报告。

## Episode 2

- Title：The Elevator He Didn't Save
- Summary：Chloe 制造双电梯火灾，Serena 和 Chloe 分别被困。Serena 隔着玻璃告诉 Adrian 自己怀孕，Adrian 仍先救 Chloe，并让安保切断 Serena 电梯电源。
- Opening Hook：Blackwell Tower 火警响起，两台玻璃电梯同时卡在半空。
- Hook Type：危机钩。
- Core Friction：Adrian 必须二选一，他选择自己相信的“继承人”。
- Pressure Setup：媒体、董事、消防警报和倒计时让选择不可拖。
- Max Spike：Serena 的电梯坠入烟火中，她的 ultrasound envelope 被火吞掉。
- Satisfaction Type：极端压抑。
- Visible Loss / Gain：Serena 失去孩子，Adrian 失去未来赎清的可能。
- Change：Serena 对 Adrian 的爱死亡。
- End Button：电梯坠落黑屏。
- Next-Episode Payoff：Serena 是否活下来。
- Do Not Consume：不让 Adrian 知道双胞胎。
- Source Function Ref：HV2。
- Forbidden Shape：出租车车祸、两车相撞。
- Opposition Fuel：事故日志被 Blackwell 安保系统改写。

## Episode 3

- Title：Sable Is Born
- Summary：Serena 在私立医院醒来，得知双胞胎没保住。Victor Crane 递给她 Blackwell 的庆祝新闻和她曾写下的风控模型备份，邀请她用自己的能力重建资本武器。五年后，Sable Ridge 的女王走进 Blackwell 的年度资本晚宴。
- Opening Hook：Serena 第一动作是摸小腹。
- Hook Type：情绪钩 + 身份悬念。
- Core Friction：失子后的 Serena 要选择活下去还是复仇。
- Pressure Setup：电视里 Adrian 正宣布 Chloe 和“继承人”将进入 Blackwell trust。
- Max Spike：Victor 叫出她当年在模型里使用的代号 Sable。
- Satisfaction Type：复仇启动。
- Visible Loss / Gain：失去孩子；获得重建身份的入口。
- Change：Serena 从幸存者转为清算者。
- End Button：五年后，她以 Sable Ridge 掌权人身份下车，全场转头。
- Next-Episode Payoff：她如何压场。
- Do Not Consume：不展示完整债权布局。
- Source Function Ref：HV2 / HV3。
- Forbidden Shape：俄罗斯黑帮、武器设计师。
- Opposition Fuel：Victor 的救命恩和资本野心。

## Episode 4

- Title：She Owns the Debt
- Summary：Blackwell 争夺救命的 CrownLink 芯片供应合约。Serena 以 Sable 身份出价并启动债权条款，冻结 Blackwell 的信用额度，迫使 Adrian 认出她。
- Opening Hook：Adrian 以为自己稳拿救命合约，屏幕突然显示 Sable Ridge 已买下 Blackwell 债权。
- Hook Type：身份碾压。
- Core Friction：Adrian 习惯命令别人，Serena 现在是他公司命脉上游。
- Pressure Setup：全场投资人和媒体看着 Blackwell 被拒付。
- Max Spike：Chloe 嘲讽 Serena 后，被安保按规章移出主桌，Blackwell 黑卡当众 declined。
- Satisfaction Type：身份碾压 + 打脸。
- Visible Loss / Gain：Blackwell 失去融资主动权；Serena 获得全场主导。
- Change：Adrian 第一次失去控制。
- End Button：Serena 贴近他说：“Your ex-wife just bought your leash.”
- Next-Episode Payoff：Adrian 调查她五年经历。
- Do Not Consume：不提双胞胎真相。
- Source Function Ref：HV3。
- Forbidden Shape：黑帮拍卖会、粉钻、枪口。
- Opposition Fuel：Adrian 可用董事会和媒体反击。

## Episode 5

- Title：The First Cut
- Summary：Adrian 查到 Sable 就是 Serena，却不明白她如何崛起。与此同时，Serena 冻结 Blackwell 三条供应链。Chloe 闯入 Serena 顶层套房企图制造丑闻，反被 Serena 用禁令、安保和事故旧账压跪。
- Opening Hook：Blackwell 大屏连续跳红，三条供应链被冻结。
- Hook Type：危机钩。
- Core Friction：Adrian 想把 Serena 拉回谈判桌，Serena 只想让他感受失控。
- Pressure Setup：董事会要求 Adrian 立刻处理 Sable。
- Max Spike：Chloe 在 Serena 套房开直播抹黑，门一开却是法院禁令和酒店安保。
- Satisfaction Type：小型打脸复仇。
- Visible Loss / Gain：Chloe 失去“柔弱受害者”直播叙事。
- Change：Serena 第一次对 Chloe 明确提到五年前的 elevator fire。
- End Button：Chloe 听见“service elevator log”后脸色变白。
- Next-Episode Payoff：事故旧账被重新查。
- Do Not Consume：不揭完整事故证据。
- Source Function Ref：HV4 前置。
- Forbidden Shape：酒店保镖打断腿扔窗。
- Opposition Fuel：Chloe 开始销毁旧安全主管。

## Episode 6

- Title：The Scar in the Glass
- Summary：Adrian 在地下车库截住 Serena，质问她为何报复。Serena 展示手腕和锁骨的电梯烧伤疤，说出“你留下的不止我一个”。Adrian 命令重查医疗记录，发现她当年怀孕。
- Opening Hook：两队车在地下车库对峙，车灯把 Serena 的疤照出来。
- Hook Type：信息钩。
- Core Friction：Adrian 想要解释，Serena 不给他赦免。
- Pressure Setup：双方保安都在，任何靠近都是权力挑衅。
- Max Spike：Adrian 看到医院档案：Serena Vale, pregnant, eight weeks.
- Satisfaction Type：真相逼近。
- Visible Loss / Gain：Adrian 失去“Serena 自导自演”的防御。
- Change：他第一次意识到自己可能害死亲骨肉。
- End Button：档案第二页被烧掉，只剩“Twin fetal...”的半行。
- Next-Episode Payoff：他必须继续查，Serena 会公开进攻。
- Do Not Consume：不让他见到完整双胞胎物件。
- Source Function Ref：HV5 前置。
- Forbidden Shape：黑帮车库枪口互指照搬。
- Opposition Fuel：Blackwell 医疗基金会篡改过档案。

## Episode 7

- Hook Type：信息钩 + 公开审判。

- Max Spike：安全主管承认 Chloe 付钱让他锁 Serena 的电梯。

- End Button：Serena 要求启动 Ethan 的 heir verification protocol。
- Next-Episode Payoff：Ethan 是否真是 Blackwell heir。
- Do Not Consume：不让 Adrian 完全崩溃。
- Source Function Ref：HV4。
- Forbidden Shape：黑医被押进黑帮宴会。
- Opposition Fuel：Adrian 可用 CEO 权限暂停流程。

- Opening Hook：投资人举杯庆祝，新闻屏滚动：Sable Ridge takes control of Blackwell debt.
- Hook Type：选择钩 + 付费卡点。
- Core Friction：Adrian 想用一切换原谅，Serena 要他先承认罪。
- Pressure Setup：投资人、媒体、Victor、Blackwell 律师都在场。
- Max Spike：Adrian 单膝跪地，CEO 徽章掉在 Serena 鞋边。
- Satisfaction Type：权力逆转 + 情感爆发中断。

- Do Not Consume：不播完整 confession，不给原谅。

- Forbidden Shape：枪响黑屏。
