# 源本有效性摘要

## 一句话故事核

被丈夫和白月光联手误判、抛弃并失去孩子的隐忍妻子，五年后以更高权力身份回归，逐层撕开白月光骗局，让丈夫亲眼承认自己亲手害死亲生骨肉，最终跪在她脚下求原谅。

## 核心期待

- 看女主从“被轻贱的妻子”变成“全场不可冒犯的女王”。
- 看男主从“不可一世的误判者”被真相一层层压垮。
- 看白月光从受保护者变成被公开剥夺体面和身份的骗子。
- 看“迟来的深情”被女主冷处理，男主越跪越惨。

## 首批留存骨架

| 段落 | 源本功能 | 留存原因 |
|---|---|---|
| EP1 | 白月光怀孕登堂入室，女主吞下孕检单签离婚 | 立刻建立侮辱、误判、隐忍和失子伏笔 |
| EP2 | 车祸中男主救白月光，弃女主于火中 | 全剧最大原罪，观众追看复仇和追悔的根 |
| EP3 | 女主失子后觉醒，五年后以高位身份回归 | 完成死亡/重生转场和身份期待 |
| EP4 | 女主公开压制男主势力，掌掴白月光 | 第一波回归爽点 |
| EP5-6 | 女主切断男主资源，男主第一次知道女主当年怀孕 | 让男主误判开始松动，但还没彻底塌 |
| EP7-8 | 白月光骗局公开坍塌，男主体面被剥光 | 首批中段强刺激区，必须避免证据播报化 |
| EP9 | 男主独自找到双胎物证，情绪彻底崩塌 | 追妻火葬场的关键情绪定锤 |
| EP10 | 男主在众人前跪地交出一切，女主枪口相对 | 首批强卡点，观众必须看下一集枪响结果 |

## 付费和追看机制

- 免费段沉没成本：女主被辱、被打、被弃、失子，观众必须等她回来清算。
- 首个强卡点：EP2 爆炸吞没女主。
- 回归爽点：EP4 女主新身份碾压旧权力。
- 中段追看债：Lily 的假孕、孩子身世、车祸主谋、男主亲子罪责逐层拆开。
- 首批尾钩：EP10 男主交出帝国和命，女主要求“一命换两命”后枪响黑屏。

## 爽点 / 压抑释放

- 压抑：三年婚姻被轻贱、白月光登堂、男主误判、车祸抛弃、失子。
- 释放：女主五年后回归、当众打脸、夺走资源、逼白月光露馅、让男主公开跪地。
- 情绪密度：高，靠“孩子债”和“误判追悔”持续供能。
- 信息密度：中高，核心信息必须分层释放，不能一次性说完。

## DNC 和含混点

- DNC：男主不能太早完全相信女主；女主不能太早心软；白月光不能太早被彻底处理；双胎真相不能早于 EP9 定锤。
- 含混点：源本 EP7-8 高度依赖证据、屏幕、报告、亲子鉴定，容易滑成 proof-flow。
- 本轮必须修正：证据只能触发人物行动，不能成为高压戏本体。
