# PRD V18 v8workflow 最小需求卡

## 0. 本步用途

给后续读本、洗稿方案、关键场面迁移卡和首批正文提供开工边界。只记录会被下一步直接使用的信息。

## 1. 输入

- 源本：`[修改版] 黑手党少主痛哭求原谅_前妻她杀疯了.txt`
- 本轮新壳：海外豪门商战 / 家族企业权力斗争
- 目标市场：海外短剧
- 集数：不变，本轮只输出首批 1-10 集
- 改写力度：中度偏重
- 输出语言：中文动作/场景描述 + 英文对白
- 审稿语言：中文
- 正文用途：AI 出海短剧可视化剧本文字，先供人审稿判断方向

## 2. 本轮必须保留的源本体验

- 被深爱的人当众放弃，女主从隐忍到死心。
- 女配假弱者真加害者，利用男主偏爱制造误判。
- 女主失去孩子后黑化，五年后以更高权力身份回归。
- 男主先误判、后调查、再被事实击穿。
- 女主回归后不是解释委屈，而是用新权力当场改写局势。
- 公开场合揭谎和打脸必须有当众后果。
- 男主火葬场要建立在“亲手造成不可逆伤害”的真相上。
- 首批结尾要停在男主放弃权力求原谅、女主给出残酷条件的一拍。

## 3. 本轮必须换掉的源本外形

- 黑手党家族、军火线、暗网武器设计师、黑帮拍卖会。
- 车祸以外的黑帮式私刑、地下刑房、枪口威胁作为核心解决方式。
- 直接复用“夜枭”“柯里昂”“战斧帮”等识别性外形。
- 公开证据不能只做“放录音/放照片/众人震惊”的单一展示。

## 4. 新壳承载方式

- 家族企业：Whitmore Group，跨国奢侈酒店、物流与医疗基金会集团。
- 男主：Adrian Whitmore，Whitmore 家族继承人兼集团 CEO。
- 女主：Ava Hart，Adrian 前妻。隐忍三年，真实身份是危机投资人兼供应链风控系统设计者，代号 Kestrel。
- 女配：Celeste Vane，Adrian 旧爱，伪怀孕，借基金会和继承人叙事登堂入室。
- 助攻/竞争者：Julian Frost，Frost Capital 掌门人，曾救下 Ava，扶她重建资本与风控网络。
- 核心权力工具：董事会投票权、供应链总控、基金会审计、医疗记录、事故调度记录、信托文件、公开股东会。

## 5. 已知禁区

- 不能按集翻译源本剧情。
- 不能把所有冲突写成会议发言和文件宣读。
- 不能让女主靠解释委屈获得优势；她必须让别人被迫行动。
- 不能让男主的悔意只靠哭和说台词成立。
- 不能让新壳弱化源本的身体代价、当众羞辱和不可逆损失。

## 6. 本轮输出

- 首批 1-10 集正文。
- 作者问题清单。
- 干净 reviewer 两轮审稿。
- V3 / V5 / V17 / V18 整体对比。
- 主控判定。
