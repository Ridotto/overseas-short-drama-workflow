# PRD v3 豪门商战版独立 reviewer 审稿

审稿范围：只读 PRD v3、源本、写作前工作稿和本轮 1-10 集正文。第一轮未读作者自检，未读旧中度版、旧深度版和旧 reviewer 审稿。

## 1. 结论

`revise`

## 2. 一句话判断

这版已经把源本 1-10 的主要留存功能迁到了豪门商战壳里，可以作为 PRD v3 流程版进入演进式对比，但给导演或用户前还需要修掉几个源本外形残留和“商业权限开挂”的硬伤。

## 3. 硬失败

无明确硬失败。

源本有效性没有洗飞：婚姻羞辱、致命误判、失子、五年后高位回归、反派继承人骗局拆穿、男主确认自己亲手造成不可逆伤害、公开下跪求赎，这条链都在。

新壳也基本承载得住：锁仓事故、债务契约、家族信托、股东大会直播、刑责自证，已经不再只是把黑手党词汇换成董事会词汇。

## 4. 重警告

1. 第一集同构风险仍然明显。Serena 怀孕进门、说“孩子不能没有父亲”、假装被 Claire 推倒、Adrian 当场指责 Claire，这一组太接近源本“孕白月光进门 + 装摔 + 男主误判”。建议改成豪门商战动作：例如 Serena 提前利用基金会授权、伪造 Atlas 访问日志、让家族董事误以为 Claire 偷密钥，而不是继续用装摔。

2. 第七集真相公开的形式还偏“押证人 + 大屏幕证据”。股东大会壳是对的，但 Mara 当场作证、付款记录和监控连续砸出来，仍有源本黑帮宴会审判的影子。建议更像企业问询：独立审计报告、外包商预录 deposition、董事会邮件链、合规委员会临时动议逐层推进。

3. Claire 的商业权力偶尔像魔法。她能冻结 Serena 的基金会权限、配偶席位、信托支取和私人黑卡，但正文没有每次都给出足够清楚的触发依据。需要补一句具体机制，例如债务 covenant、信托章程、基金会违规支出、法院临时禁令或董事会合规冻结，否则爽点会显得硬堆。

4. 男主崩溃证据有轻微提前透支。第六集已经拿到双胎病历和一段求救录音，第九集再用完整录音和超声碎片，情绪仍有效，但第九集的爆破力会被削弱。建议第六集只让 Adrian 查到“怀孕/双胎/缺氧流产”的冰冷事实，第九集再第一次听见 Claire 给孩子的语音和完整求救。

5. 少量台词还带源本口号感或翻译腔。“孩子不能没有父亲”“我的名字是你的，我的命也是你的”“一条命，换两条命”等句子功能对，但外形太熟。建议改成更贴企业语境和人物身份的表达，例如自由、签字权、信托控制权、刑责、Hawthorne 姓名这些更具体的代价。

## 5. 可选优化

- Marcus 在 1-10 集里主要是救人、陪站和讽刺，利益盟友的锋利度还不够。可以给他一个明确商业动作：他救 Claire 的同时拿到 Hawthorne 的某个债务入口，或在第十集暗示他并不希望 Adrian 彻底回到 Claire 身边。
- 第四集的回归爽点成立，但 Serena “配偶席位”这个称谓略怪。她不是配偶，更像未来继承人母亲、基金会负责人或信托预登记代理人。
- “Rook Capital 控制人”“Hawthorne 先生”等称谓可以保留海外感，但中文对白要再压一遍，减少解释型台词。
- 第十集直播认罪很强，但法律风险太直。若想更商战，可以改成 SEC/FCA 调查协助声明、董事会特别委员会公开证词，仍然保留他亲口毁掉自己的效果。

## 6. 对演进式对比的价值

这版最值得观察的变量是：PRD v3 新增的“写作前工作稿”能不能把源本功能迁移成新事件载体。

从结果看，它有效降低了整体同构。最明显的进步是：

- 车祸火海变成低氧档案库事故；
- 黑帮军火断供变成债务、信贷和酒店管理权；
- 亲子鉴定变成家族信托和胚胎文件；
- 枪口逼命变成刑责自证和公开直播。

但这版也暴露了 PRD v3 下一步要验证的红线：即使大事件换了，局部桥段如果沿用源本外形，导演仍可能第一时间感到“还是那个戏”。所以对比旧中度版、旧深度版时，最该看的是“事件载体新鲜度”和“观众是否仍能识别出源本外形”。

## 7. 需要主控/用户拍板的问题

1. 进入演进式对比前，要不要先做一轮小修？我建议先修第一集装摔、第七集证据公开、Claire 权限依据这三处，再拿去和旧版对比。

2. 用户能不能接受“怀孕白月光 / 失子 / 下跪求赎”这些强功能继续保留？如果目标是明显避开源本辨识度，第一集和第十集还要再去源本化。

3. 豪门商战壳要偏“短剧爽感”还是偏“商业可信”？当前稿偏爽感，商业机制够用但不严。如果导演看重可拍可信度，需要再补契约和权力触发逻辑。

4. 第十集公开直播认罪是否过重？如果后续还要写 11-37 集，主控需要决定男主坐牢风险是后续主线，还是只作为首批强留存卡点。
