# PRD v17-v7workflow 二次需求确认

本轮没有新增用户实时确认，按当前项目已确认设定继续：

```text
新壳：海外豪门商战 / 家族企业权力斗争
集数：不变
输出范围：首批 1-10 集
改写力度：中度偏重
输出：英文对白 + 中文动作/场景描述
用途：用户/导演审稿，兼顾 AI 出海短剧可视化
```

## 主创判断

豪门商战壳可以承载源本，但不能承载为“文件、档案、授权、会议”这种冷载体。

本轮选择的新壳核心不是商业术语，而是：

```text
发布会 / 董事会 / 家族徽章 / 继承人叙事 / 玻璃中庭事故 /
基金会儿童翼 / 资本收购 / 直播发布会 / 股权和身份公开坍塌
```

其中强戏必须由人对人选择、公开羞辱、身体危险和权力坍塌承载，文件只作为辅助证据。

## 变量边界

强继承：

- 被替代和误判；
- 男主错误选择；
- 女主不可逆失子；
- 五年后权力反转；
- 白月光骗局崩塌；
- 男主迟到真相后的公开赎罪；
- 每集留存钩子和首批追看惯性。

等效迁移：

- 黑帮权力 -> 家族企业 / 资本 / 慈善基金会 / 继承人秩序；
- 车祸二选一 -> 发布会玻璃中庭坍塌的二选一；
- 拍卖会回归 -> 家族债权拍卖 / 继承人发布会回归；
- 亲子鉴定 -> 继承人血样 / CFO 私生子公开崩塌；
- 旧仓库旧物 -> 旧发布会中庭封存物和安保影像。

自由改写：

- 具体身份、职业、地点、商业项目名称；
- 公开场面形式；
- Marcus 的动机和站位。

必须删除 / 修复：

- 整串事件链复刻；
- 低信息多人对白；
- 文件/屏幕替代强戏；
- 角色站桩冷静解释。
