# PRD_v23_clean_v11gate_Source Bible

## 1. 一句话故事核

一个被丈夫当成无用妻子的隐藏强者，在丈夫选择假孕白月光并害她失去孩子后，五年后以更高权力归来，先摧毁情敌谎言，再让男主亲手面对自己当年的罪。

## 2. 核心期待

观众追的是：

- Eve 为什么能从低位妻子变成让所有人仰望的 Night Owl。
- Dante 什么时候知道自己选错人、害死亲生孩子。
- Lily 的假孕、买凶、野种谎言如何被公开撕开。
- Eve 会不会原谅 Dante，还是让他一直跪着还债。

首批 1-10 集必须完成：

- 低位压迫和生死抛弃成立；
- 女主高位归来成立；
- 女配第一层谎言崩塌；
- 男主从“怀疑”进入“崩溃求赎”；
- 第 10 集以男主公开下跪 + 女主枪口卡点结束。

## 4. 高价值节点

### HV1. 第一集：妻子被替换

- 客观事件：Dante 带怀孕 Lily 回家，要求 Eve 签离婚。
- 观众体验功能：开局强压低女主，建立“她凭什么忍”和“她藏着什么”。
- 强度来源：婚姻、孕事、耳光、当场吞下孕检单。
- 关系 / 权力变化：Eve 从妻子变成被扫地出门的人；Lily 从外来者进入高位。
- 信息差状态：观众知道 Eve 真怀孕且有强身份痕迹，Dante 不知道。
- 卡点停法：Lily 在阳台露出恶意，说明离开不是结束。
- 源本高辨识外形：黑帮庄园厨房、孕检单塞嘴、白月光假摔。
- 下游写作抓手：在新壳里用“家族董事晚宴 / 继承人条款 / 婚前股份协议 / 公开签字”替代厨房离婚，不复用假摔。

### HV2. 第二集：生死选择

- 客观事件：车祸中 Eve 和 Lily 同时被困，Dante 选择 Lily，Eve 爆炸流产。
- 观众体验功能：把男主罪责钉死，不是误会小吵，是不可逆伤害。
- 强度来源：呼救被无视、孕事被否定、火光吞没。
- 关系 / 权力变化：Eve 对 Dante 的爱死亡；Dante 的后悔资格被未来锁定。
- 信息差状态：Dante 认为 Eve 害 Lily；观众知道 Lily 有阴谋。
- 卡点停法：爆炸吞没车身。
- 源本高辨识外形：两车相撞、火海救一个弃一个。
- 下游写作抓手：改为“智能总部安全封锁 / 服务器层氧气抽排 / 玻璃门内外可视救援选择”，保留二选一体验，不复刻车祸。

### HV3. 第 3-4 集：女主高位归来

- 客观事件：Eve 失子后与 Victor 联手，五年后作为 Night Owl 出现在拍卖会，夺走生意。
- 观众体验功能：压抑后的第一轮释放。
- 强度来源：过去被说没用的人，现在掌握对方命门。
- 关系 / 权力变化：Eve 从被抛弃者变成规则制定者。
- 信息差状态：Dante 不知 Night Owl 是 Eve；Lily 知道 Eve 应该已死。
- 卡点停法：Eve 把 Dante 的枪按到自己胸口，让他开枪。
- 源本高辨识外形：地下拍卖、军火断供、枪口抵心口。
- 下游写作抓手：改为全球供应链峰会 / 债权拍卖 / 股份救援协议，枪口换成“当众签署控制权 / 保安撤换 / 股价墙下封杀”。

### HV4. 第 7-8 集：女配公开裂开

- 客观事件：Eve 在宴会公开黑医、录音、床照、亲子鉴定；Lily 反咬 Dante。
- 观众体验功能：把“她骗了你五年”和“你也有罪”两刀连打。
- 强度来源：公开场合、男主尊严崩塌、Lily 从受宠者变成污点。
- 关系 / 权力变化：Dante 对 Lily 的保护崩；Eve 掌控全场。
- 信息差状态：揭 Lily 假孕和野种，但不能让 Dante 完成完整悔悟。
- 卡点停法：Lily 喊出“你也害死了自己的孩子”。
- 源本高辨识外形：大屏播放证据、床照、亲子鉴定、当场验血。
- 下游写作抓手：必须换成商战现场的不可逆后果载体，例如继承人信托解锁失败、董事投票权当场冻结、供应商撤章、情夫为救孩子/权益冲出来暴露，而不是屏幕证据会。

### HV5. 第 9-10 集：男主火葬场启动

- 客观事件：Dante 在旧仓库找到孕检单、婴儿毛衣、尾戒，雨中自毁，公开下跪献出家族。
- 观众体验功能：从外部羞辱进入内心坍塌，男主开始低位追妻。
- 强度来源：私人物件比证据更扎心；公开跪比电话道歉更有社会性死亡。
- 关系 / 权力变化：Dante 从高位暴君变成求赎者；Eve 成为审判者。
- 信息差状态：第 9 集才允许完整确认孩子的存在和失去细节。
- 卡点停法：Eve 枪响黑屏，观众追她是否真杀他。
- 源本高辨识外形：仓库保险箱、婴儿毛衣、雨中下跪、旋转餐厅枪口。
- 下游写作抓手：改成“被封存的公司育婴室 / 未送出的股份信托文件 / 旧工牌和胎心录音 / 顶层股东晚宴公开交权”，让火葬场通过商战物件完成。

## 5. 钩子与卡点

- Ep1：白月光带“继承人”登堂，女主签走。
- Ep2：男主在安全封锁中选择救白月光，女主被隔在玻璃后。
- Ep3：女主失子、死亡重生；五年后黑线资本代表入场。
- Ep4：女主身份揭开并当众夺控制权。
- Ep5：男主刚想控制局面，公司命门被切；女配私下报复失败。
- Ep6：女主露出封锁事故伤痕，男主查到她曾怀孕。
- Ep7：女配当年雇人制造封锁和假孕利益链开始公开崩。
- Ep8：继承人信托现场崩盘，女配反咬男主。
- Ep9：男主打开被封存的育婴室 / 信托箱，发现双胎真相。
- Ep10：男主公开交权跪求，女主把选择权变成枪响式卡点。

## 6. 爽点 / 压抑-释放

- 压抑：女主被说无用、被打/辱、被弃、失子。
- 第一释放：女主归来，旧家族经济命门被她按住。
- 第二释放：女配私闯失败，被女主现场踩回低位。
- 第三释放：公开场合，女配多年谎言反噬她和男主。
- 延迟释放：男主第 9 集才被私人物件击穿，第 10 集公开跪下。

## 7. 关系 / 权力变化

- Eve / Dante：妻子低位 -> 被弃者 -> 对手 -> 审判者 / 求赎者。
- Eve / Lily：被踩者 -> 复仇者；Lily 的每一次得势都要被新的现场后果反噬。
- Eve / Victor：救命盟友 / 复仇合作方；首批只承担外部助力和刺激男主嫉妒，不提前变成主矛盾。
- Dante / Lily：保护者 / 被保护者 -> 怀疑 -> 公开厌弃，但完整处置留到 11 集后。

## 8. 信息差与真相路径

真实事实：

- Eve 当年是真怀孕，且被安全封锁事故伤到流产。
- Lily 的怀孕和继承人身份是谎言，背后有商战利益。
- Dante 在关键时刻主动选择救 Lily、抛下 Eve，这不是单纯误会。
- Eve 五年后的新身份来自自身能力 + 外部资本合作，不是只靠男配。

分阶段信息：

- Ep1 观众知道 Eve 有隐藏能力、真孕、被压低。
- Ep2 观众确认 Dante 的抛弃造成不可逆后果。
- Ep3 观众知道孩子没了，Eve 重生。
- Ep4-5 众人知道 Eve 是 Blackline，但不知道全部失子细节。
- Ep6 Dante 只知道她当年怀孕和伤疤，不知道双胎、不知道她保存过什么物件。
- Ep7-8 公开打碎 Lily 的假孕、买凶和继承人谎言；Dante 被迫面对“你也有罪”。
- Ep9 才释放双胎、旧物件、他当年冷落她的私人记忆。
- Ep10 才让 Dante 公开交出权力和身体尊严。

## 9. 写作手法 / 表达手法

- 强开局：直接把替换妻子和签字放到可视化权力场。
- 高压场面必须有人失去东西：头衔、投票权、保安控制权、身体安全、公开脸面。
- 情绪靠物件：旧工牌、未签的胎儿信托、封存的婴儿房、烧黑的董事通行卡。
- 男主转折不能靠“我明白了”台词，要靠他亲手打开当年自己无视的物件。
- 女主台词短，更多用动作、签字、撤章、踩碎、调换保安控制权来压人。

## 10. 水分 / 烂点 / 禁止复用外形

水分和风险：

- 源本后段有“孩子”与“双胞胎”表述不一致；新本必须拍板：本版真实事实为 Serena 当年怀的是双胞胎，但 Ep9 前不公开。
- 源本第 7-8 集高度依赖大屏、报告、亲子鉴定；新本不能照搬为公司大屏展示证据。
- 源本男配 Victor 后期复杂化，首批只需要作为救命资本盟友和暧昧压力，不提前引出外部战争。
- 源本第 6 集已经说“孩子在火里烧着”，容易提前吃掉第 9 集火葬场；新本第 6 集只能让男主知道“她当年怀孕 / 事故受伤”，不说双胎和完整物件。

禁止复用外形：

- 黑帮、军火、地下拍卖、枪林弹雨式黑道秩序。
- 车祸、两车、火海救 Lily 弃 Eve。
- 大屏连播录音、床照、亲子鉴定、当场验血。
- 仓库保险箱、婴儿毛衣、雨中跪街。
