# 源本有效性摘要

## 一句话故事核

一个把天赋、婚姻和孩子都押给丈夫的女人，被丈夫在最致命的一次选择里亲手丢进火里；她五年后带着更高的权力回来，让他在知道真相以后活着受罚，而不是痛快去死。

## 源本为什么能追

### 1. 第一波抓力不是“身份高”，而是“抛弃够狠”

源本前两集真正把人钉住的，不是黑帮设定本身，而是三连击：

1. 丈夫把怀孕白月光直接带回家；
2. 女主连自己的孕检单都没机会拿出来；
3. 火场里男主公开做出“先抱走她，不救你”的选择。

这不是普通误会，是主动站队和主动抛弃。
所以后面男主再怎么跪，观众都不会轻易心软，追感才够强。

### 2. 源本前十的主爽点是“延迟醒悟后的极端卑微”

前十不是女主简单复仇，而是：

- 女主先回来碾压；
- 男主还盲；
- 观众比男主先知道他会迎来火葬场；
- 物证一层层砸下来以后，男主才崩。

因此首批最值钱的不是“复仇已经完成”，而是“他终于知道自己到底毁了什么，而且还没资格被原谅”。

### 3. 首个强付费窗口靠三种债务叠压

源本前十最能逼人往后看的，是三笔债同时悬着：

- 女主的孩子债：到底怎么死的、死到什么程度；
- 女配的假孕债：她到底骗到哪一步；
- 男主的罪责债：他知道以后会疯成什么样。

如果只剩一笔，追感会塌。

## 留存骨架

### 免费段必须建立的沉没成本

- 女主婚内隐忍不是“笨”，而是把全部情感和身份都押进去了。
- 男主的抛弃是亲眼可见、无法洗白的。
- 女配不是简单作妖，而是带着夺位、谋杀、篡改继承权的系统性恶意。

### 首个强付费卡点应当保留的体验

前十的最强付费不是普通 cliffhanger，而是：

- 男主终于拿到那张会让他彻底疯掉的物证；
- 然后他在公众场合把尊严、资产、身份一起交出去；
- 女主仍然不给他“死得痛快”的资格。

## 首批 `1-10` 强节点

### 必保节点

1. 进门带孕白月光 + 离婚协议
2. 女主来不及说出的怀孕事实
3. 车祸/火场里男主的选择
4. 病床醒来，确认孩子没了
5. 五年后归来，第一场公开碾压
6. 女主把男主的生意命门掐住
7. 女配的假孕/假继承权开始公开崩
8. 男主拿到实锤自己害死孩子的物证
9. 男主暴雨崩溃
10. 公众场合下跪求原谅

### 当前批高窗口排序

- `EP2`：火场抛弃
- `EP4`：回归首压
- `EP7-8`：公开拆穿 + 男主当场塌
- `EP9`：物证兑现
- `EP10`：跪地献家业

## 情绪 / 信息 / 情节密度

### 情绪密度

- 女主的情绪线不是哭哭啼啼，而是“压住 -> 熄灭 -> 冰化”。
- 男主前期冷，后期疯。落差越大越有爽感。
- 女配必须持续招恨，不能只靠一次恶。

### 信息密度

- 每一集都要推进“她怎么回来”“他什么时候知道”“她到底准备把他折磨到哪一步”。
- 不能连续两集只做同一种羞辱。

### 情节密度

- 重要转折都靠动作：带人进门、撞车、抱人离开、扇耳光、压枪口、踩手指、冲进餐厅下跪。
- 一旦变成坐着讲、站着解释，源本最值钱的冲击就会蒸发。

## 关系 / 拉扯 / 权力变化

### 核心关系

- 女主 vs 男主：从“她爱他他不值”切到“他想要她但她要他活着受刑”。
- 女主 vs 女配：不是纯雌竞，是“夺位 + 谋杀 + 篡继承”。
- 男主 vs 女配：从盲信到厌恶到暴怒，但转折必须付出代价。
- 男主 vs 男配：不是单纯争风吃醋，而是“谁能站在女主权力侧”。

### 权力变化

- 前两集：男主完全压制。
- `EP4` 起：女主开始掌握交易命门。
- `EP7-8`：女主把男主从“上位者”打成“被围观羞辱的人”。
- `EP9-10`：男主主动交权，权力关系彻底倒挂。

## 对立压力源

- 女配的假孕和篡位欲，是第一层压力。
- 男主的盲目站队，是第二层压力。
- 家族/商业权力结构，是第三层压力。
- 男配/ rival 的介入，是第四层压力：既是助力，也是对男主羞辱。

## DNC

- 不能把男主洗成“只是误会”。
- 不能让女主一回归就心软。
- 不能让女主的强大只停在头衔和台词。
- 不能把白月光拆穿做成纯文件宣读。
- 不能用一次 confession 就结清男主全部罪责。

## callback / payoff 债务

- 未说出口的孕检单 / 妊娠证明：后续必须回收。
- 火里“你回来救我”这一下：后续必须反向回收为他来不及求她回头。
- 旧戒指 / 给孩子准备过的物件：适合做后续物证重击。
- 女主“你给不起我要的”：后续要兑现为家业、命、尊严一起上桌也不够。

## 源本高辨识外形

这些不是必须原样保留，但必须找同强度承载：

- 庄园厨房里的离婚现场
- 火场 / 爆炸 / 车窗拍打
- 五年后高调入场
- 公开掌掴
- 手踩手指
- 当众曝光假孕和养错孩子
- 旧保险箱里的孕检单 / 婴儿物件
- 暴雨跪街
- 公共场合双膝下跪献出一切

## 源本含混点与本轮处理

### 1. 孩子数量口径漂移

前段像单胎，后段物证写成双胞胎。
本轮不直接删掉这个强刺激，而是收束为：

- 前半所有角色只知道“她怀孕了”；
- 直到 `EP9` 物证层，才揭晓是“双胎 / 两个心跳”；
- 这样不但补掉矛盾，还能把物证打击做得更重。

### 2. 前十中段容易变成“放证据给大家看”

源本 `EP7-8` 的风险不是信息不够，而是太容易滑成解释。
本轮必须改成公开活动中的动作性拆穿，保证权力和关系当场变化。

### 3. 男主醒悟触发太单线

如果只靠女配崩一句“你也有罪”，男主崩塌会显得廉价。
因此本轮必须保留两段式：

- `EP8`：意识到自己被戴帽、被骗五年；
- `EP9`：在物证面前意识到自己亲手害死的是自己的两个孩子。
