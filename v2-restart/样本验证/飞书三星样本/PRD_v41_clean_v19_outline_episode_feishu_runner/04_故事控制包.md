# 故事控制包

## 本层 gate

- 通过了什么：
  - 已把源本核心债务重组为“港口与冷链帝国”可承载的公开权力战。
  - 已保住“抛弃 -> 归来碾压 -> 假 heir 崩盘 -> 物证重击 -> 公开下跪”主链。
  - 已为 `EP1-10` 明确高窗口和追看债务。
- 因为什么通过：
  - 当前批每集都有“谁在公众面前失势 / 谁掌握了路线命门 / 下一步更痛的债是什么”。
  - 没让控制层滑成审计说明或法律流程。
- 本层仍需注意：
  - `EP7-8` 不能因为证据密度而削弱动作密度。

## 1. 新本故事核

被港口豪门亲手丢进火里的前夫人，五年后以北美冷链女王身份归来，先夺命门，再夺脸面，最后逼那个昔日继承人在全城注视下跪着把帝国交到她脚边。

## 2. 真相账本

### 已经发生的客观真相

1. `Serena Quinn` 曾是 `Adrian Vale` 的妻子。
2. Serena 实际是 Vale 帝国早期冷链安全与灰区航运架构的隐形设计者，化名 `Night Harbor`。
3. `Chloe Hart` 伪造怀孕和 heir 叙事，借机登堂入室。
4. Chloe 买通私人生育 concierge、司机和港桥冷链油罐车，策划撞车与火场灭口。
5. Adrian 在火场里主动选择先救 Chloe，把 Serena 留在燃烧车内。
6. Serena 因此流产，且后续物证显示失去的是双胎。
7. `Viktor Sokolov` 在事故后救下 Serena，并资助她重建北美冷链与保税网络。
8. 五年后 Serena 以 `Night Harbor` 身份切断 Vale 帝国关键航线和供应。
9. Adrian 前十里只会先拿到零散真相，直到 `EP9` 才触碰到“双胎物证”。

### 当前批只应释放到什么程度

- `EP1-2`：观众知道 Chloe 恶、Adrian 抛弃、Serena 怀孕；角色层面仍未全面知道。
- `EP3-4`：观众知道 Serena 活着回来且已成更高阶操盘手。
- `EP5-6`：Adrian 才开始摸到 Serena 掌权真相与她当年怀孕线索。
- `EP7-8`：公众知道 Chloe 的假 heir/假孕/谋害链开始崩，但 Adrian 还没拿到最后物证。
- `EP9`：Adrian 亲手打开旧港仓保险箱，拿到双胎物证。
- `EP10`：公众见证他交权、下跪、求原谅。

## 3. 角色错信

### Serena

- 错信：Adrian 现在的痛苦更多是他无法接受自己当年选错，而不是他终于懂她。
- 可用价值：她会把他留在刑架上，不给轻易赎罪。

### Adrian

- 前段错信：Serena 只是无趣、依附、不会离开的妻子；Chloe 才是值得他护着的人。
- 中段错信：交还产业也许能换回一丝原谅。
- 前十结束时仍未真正懂：她想要的不是财富，而是他活着吞下自己害死两颗心跳的事实。

### Chloe

- 错信：只要手里有“Vale heir”和 Adrian 的旧情，她永远有退路。

### Viktor

- 错信：救起 Serena、扶她上位，就能在未来把她变成自己的北美合伙人或配偶。

### Vale 家族老臣 / 信托董事

- 错信：Serena只是 Adrian 的情感失控源，不是结构性威胁。

## 5. 全剧阶段骨架

### 第一阶段：婚内背刺与火场抛弃（`EP1-3`）

- 建立 Serena 的隐形价值、怀孕希望、婚内卑位。
- 用火场主动抛弃完成情感底板。
- 让 Serena 在“孩子没了 + 他给别人办 heir 庆宴”后彻底冰化。

### 第二阶段：女王归来与首轮掐命门（`EP4-10`）

- Serena 公共场合夺价、断线、打脸。
- Chloe 的 heir 神话开始崩。
- Adrian 先摸线索，再碰实锤，最后公开跪。

### 第三阶段：假死刑与忠犬化开端（`EP11-20`）

- Serena 不给 Adrian 便宜死法。
- Chloe 失去一切，Vale 老臣逼宫。
- Anna 绑架、旧船厂救援、Mediterranean 港口家族入场。

### 第四阶段：外敌升格、Victor 暗心显形、北美暂平（`EP21-32`）

- Salvatore Bianco 以地中海港口联盟身份杀入。
- Adrian 以残躯替 Serena 探线、救人、挡枪。
- Serena 识破 Victor 迟来、收走北美控制权。

### 第五阶段：名声战反扑（`EP33-37`）

- Victor 不抢地盘，改毁 `Night Harbor` 信誉。
- 假冷链封签和仿造货流引发街头血案，脏水泼给 Serena。
- Serena 与 Adrian 反查纽约钉子，准备把战场推到莫斯科。
