# P9 作者自检摘要

> 输入：P1-P8 + P7 正文修订稿  
> 性质：主创自检，不展示给用户，不替 reviewer 下结论。

## 1. 是否源本级洗飞

初判：没有明显洗飞。

保住的源本有效机制：

- EP1 保留正妻被白月光替代、怀孕信息差、吞下证据的强羞辱。
- EP2 保留男主现场选择白月光、女主遭受不可逆伤害。
- EP3 保留失子后由外部强者救下、隐藏身份被激活。
- EP4-5 保留女主归来后的身份碾压和资源压制。
- EP7-8 保留白月光假孕 / 出轨 / 孩子非亲生的公开拆穿链。
- EP9 保留孕检单 / 婴儿物件等实物定锤。
- EP10 保留男主权力坍塌、跪求原谅、枪响硬切。

风险：

- 黑帮暴力外形已替换成豪门商战，但 EP10 老枪仍有强 melodrama 外形，需要 reviewer 判断是否破坏新壳。

## 2. 是否源本级改名复刻

初判：不是简单改名复刻，但仍保留了源本前 10 集骨架。

有效替换：

- 军火线 -> 港口 / 冷链 / 信用保险；
- 黑帮拍卖会 -> 港口续约慈善晚宴；
- 黑医 / 亲子鉴定 -> 私人诊所会计 / 信托采样；
- 车祸爆炸 -> Blackwell 港口冷链舱事故；
- 夜枭 -> Night Harbor / Grey Harbor；
- 教父徽章 -> CEO 印章 / 控股权 / 港口地契。

仍接近源本的功能：

- EP1 逼离婚 + 假孕白月光；
- EP2 事故中救白月光不救妻；
- EP9 旧物定锤；
- EP10 跪求原谅 + 枪响。

判断：这是本轮“中度偏重”改写策略允许的功能继承，不属于外形机械复刻。

## 3. 是否提前消费

初判：关键真相分配基本正确。

- EP6 只让 Ethan 知道 Serena 当年怀孕，不给双胎和授权码。
- EP7 公开 Chloe 买凶 / 假产检第一层。
- EP8 公开孩子非亲生，并让 Chloe 反咬 Ethan 罪责。
- EP9 才给双胎超声、小毯子、防火门授权码。

风险：EP7 的司机证言已经指向事故买凶，可能稍强；但 EP9 的真正痛点是 Ethan 的授权码和双胎实物，未被 EP7 吃掉。

## 4. 是否 proof-flow 替代高压戏

初判：大部分高压点已落到人物行动。

- EP1：吞报告、签字、离开。
- EP2：Ethan 下令关门。
- EP4：Serena 当众打 Chloe，冻结函按到 Ethan 胸口。
- EP5：Serena 踩住 Chloe 签信托的手。
- EP7：Chloe 被迫签继承人信托后再被证人指认。
- EP8：Ethan 亲手把孩子放到采样台，结果公开。
- EP9：Ethan 亲手砸柜、抱小毯子、自扇耳光。
- EP10：Ethan 当众跪下交权，Serena 把枪塞给他。

风险：EP7-8 仍有直播、证人、采样、屏幕结果，需 reviewer 重点看是否足够有现场压迫，不是纯证据展示。

## 5. 是否状态漂移

初判：无明显状态漂移。

- Serena 从隐忍到冷硬复仇，状态连贯。
- Ethan 从傲慢误判到动摇，再到 EP9 彻底崩溃，层级清楚。
- Chloe 从胜利者、私下施暴者到公开崩塌，递进清楚。
- Mason 只在 P7 保持同盟功能，未越权抢主线。

## 6. 是否正文长剧化

初判：没有明显长剧化，但 EP3、EP6 有轻微信息解释风险。

优点：

- 每集都有结尾追问。
- 每集都有可拍的动作承载。
- EP1-3 压力密度较高。
- EP7-10 形成连续强窗口。

风险：

- EP3 的 Mason 对 Serena 价值点仍偏说明，可以在后续修订里加一拍更可见的“他展示 Blackwell 曾靠她系统续命”的动作。
- EP6 档案室查病历有信息场风险，但它只给第一层信息，仍可接受。

## 7. 根因层判断

如果本轮不通过，最可能回到：

1. P4：EP7-8 的用户可见集纲是否仍偏证据链。
2. P5：EP7-8 是否需要更强的现场承载，例如董事会当场切割 Ethan / Chloe 失去信托资格。
3. P7：台词还可进一步压缩，减少“你害死孩子”类直白句。

不建议回到：

- P1/P2：源本有效性与洗稿方案基本清楚。
- P6：P6 已按 P5 执行，若弱，多半是 P4/P5 强度问题。
