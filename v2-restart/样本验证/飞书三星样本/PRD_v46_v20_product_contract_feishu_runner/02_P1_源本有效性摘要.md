# P1 源本有效性摘要

> 输入：P0 需求确认摘要 + 源本前 1-10 集  
> 方法资产：`hook-design` / `satisfaction-matrix` / `paywall-design` / `villain-design`

## 1. 源本主追问

这部源本前 10 集的主追问不是“Dante 会不会后悔”，而是：

```text
被他亲手抛弃并失去孩子的 Eve，归来后会怎样让 Dante 亲眼看见自己错杀了什么？
```

前 10 集的观看驱动力来自三层递进：

1. Eve 被替代、被羞辱、被抛弃。
2. Eve 活下来并以更高权力归来。
3. Dante 从掌权者跌到跪地求原谅的人。

## 2. 源本沉没成本

源本用这些动作建立观众投入：

- EP1：Eve 吞下孕检单，签离婚协议离开。观众知道她有孕，Dante 不知道。
- EP2：Dante 明确选择救 Lily，放弃 Eve。伤害不是误会，而是现场选择。
- EP3：孩子没了，Victor 揭示 Eve 的真实能力，复仇发动。
- EP4：Eve 归来，第一次让 Dante 在公开场合失控。
- EP7-8：Lily 的假孕、出轨、孩子非亲生被公开拆穿，观众获得连续打脸。
- EP9-10：Dante 找到孕检单和婴儿毛衣，从“被骗的人”变成“亲手害死孩子的人”。

沉没成本的核心是：观众比男主更早知道真相，因此每一次男主维护 Lily 都在累积恨意。

## 3. 有效钩子

| 源本节点 | 钩子类型 | 有效原因 |
|---|---|---|
| EP1 Eve 吞孕检单离开 | 情绪钩 + 信息钩 | 观众知道她怀孕，男主不知道；她为什么吞下去形成追问 |
| EP2 火中弃救 | 危机钩 | 男主现场选择白月光，女主生死与孩子命运悬住 |
| EP3 五年后红底高跟鞋入场 | 身份悬念 | 死去的弃妇归来，身份与权力反差强 |
| EP4 Eve 压下 Dante 枪口 | 身份反转 | 男主第一次发现前妻不是自己能处置的人 |
| EP7 Lily 罪证公开 | 反转钩 | 观众期待白月光假面被撕 |
| EP8 Lily 说出 Dante 也有罪 | 信息钩 | Dante 的罪责被推到他面前，但物证未完全落地 |
| EP10 Eve 枪口抵 Dante | 情绪钩 + 危机钩 | 男主求原谅，女主给出生死选择 |

## 4. 有效爽点

| 爽点类型 | 源本表现 | 迁移要求 |
|---|---|---|
| 打脸复仇 | Eve 当众掌掴 Lily，公开揭露 Lily 谎言 | 新本必须让反派在公开权力场失去体面 |
| 身份碾压 | Eve 从弃妇变 Night Owl | 新本必须让 Serena 的真实身份能压住 Blackwell |
| 逆袭翻盘 | Eve 断掉 Dante 的军火线 | 新本必须是资源 / 供应链 / 控股权的实质压制 |
| 情感爆发 | Dante 发现孕检单和婴儿毛衣后崩溃 | 新本必须有实物定锤，不只靠报告或口供 |
| 关系权力反转 | Dante 从教父变跪求原谅的人 | 新本必须让 Ethan 当众失去“继承人 / CEO /丈夫”的位置 |

## 5. 付费冲动机制

本轮不默认具体付费集。只拆源本前 10 集的追付费机制：

- 付费前必须已经建立：女主被害、男主误判、反派谎言、女主归来能力。
- 付费窗口最容易落在：真相开始公开但还没完成定锤的位置。
- 源本 EP7-10 的强机制是：先揭 Lily 假，再揭 Ethan 真罪，最后让 Ethan 跪到 Serena 面前。
- 付费后不能只继续欠债，必须兑现一次打脸或一次权力坍塌，再欠下一层更痛的答案。

因此新本首批的强窗口候选应该围绕：

```text
Chloe 公开崩塌
-> Ethan 以为自己只是被骗
-> 物证证明他才是杀死亲生孩子的人
-> Ethan 当众交出权力求原谅
```

## 6. 对立压力源

源本前 10 集不是单反派，而是三层压力：

- Lily：直接陷害者，负责挑衅、伪孕、车祸和出轨谎言。
- Dante：权力压迫者，负责误判、抛弃、保护 Lily。
- Victor：外部支撑者，负责救下 Eve 并给她复仇资源，同时制造 Dante 嫉妒。

新本必须保留这三种功能：

- Chloe：假孕白月光，制造家庭和继承人压力。
- Ethan：Blackwell 继承人 / CEO，做出抛弃 Serena 的现场选择。
- Mason：竞争资本 / 港口联盟掌控者，救下 Serena 并扶她回到牌桌。

## 7. 不能洗丢的体验

- 观众必须站 Serena，不是中立看商业斗争。
- Ethan 必须先错得够重，后面跪才有价值。
- Chloe 必须先赢得够恶，后面被公开拆才爽。
- Serena 归来必须是“人走进现场，局势变了”，不是报告发出来。
- EP9 的实物定锤必须留到 EP9，不能被 EP7/8 的证据提前吃掉。

## 8. 不能继承的烂点

- 后段重复求原谅、挡伤、下跪不能在首批里提前泛滥。
- 证据不能只是屏幕播放，要逼人物现场选择、失去关系或权力。
- 暴力惩罚不能机械继承黑帮外形。
- OS 和解释性台词要压低，用动作和现场后果表达。

## 9. 高辨识外形与同构风险

高辨识外形：

- 黑手党庄园；
- 军火线；
- 黑帮拍卖会；
- 地下车库持枪；
- 黑医与亲子鉴定现场；
- 暴雨下跪与枪口。

新本不能直接复刻这些外形。必须迁移成：

- Blackwell 家族庄园 / 董事会 / 港口冷链仓；
- 全球供应链和信用保险封锁；
- 慈善晚宴 / 股东直播 / 继承人信托；
- 私人医院 / 冷链事故记录 / 婴儿保温箱物证；
- 百年家族晚宴上的 CEO 印章、股权和公开忏悔。
