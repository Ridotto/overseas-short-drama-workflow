# P2 洗稿方案摘要

> 输入：P0 + P1  
> 目标：把源本有效机制迁移到豪门商战新壳，守住洗稿边界。

## 1. 总体方案

把源本从“黑手党少主 / 军火女王 / 地下世界复仇”洗成：

```text
Blackwell 家族继承人 Ethan 为了怀孕白月光 Chloe 当众逼妻 Serena 离婚。
Serena 在 Blackwell 港口冷链事故中失去孩子。
五年后，她以灰港资本实际控制人和 Night Harbor 调度系统设计者身份归来，
切断 Blackwell 的港口、冷链和信用保险命脉，
逼 Ethan 亲眼看见自己为了一个假继承人，害死了真正的孩子。
```

核心不变：

- 被替代；
- 被抛弃；
- 失子；
- 女主隐藏身份；
- 女主归来夺权；
- 白月光公开崩塌；
- 男主当众跪求原谅。

外形替换：

- 黑帮权力 -> 家族企业 / 董事会 / 供应链命脉；
- 军火断供 -> 港口调度、冷链合约、信用保险和融资封锁；
- 黑帮宴会 -> Blackwell 百年慈善晚宴 / 股东直播；
- 黑医证据 -> 私人诊所、冷链事故调度记录、继承人信托文件、旧保温箱物证；
- 枪口 -> Blackwell 家族老枪 / 企业继承权与公开忏悔叠加的羞辱场。

## 2. 角色迁移

| 源本 | 新本 | 功能 |
|---|---|---|
| Eve | Serena Vale | 被误判的妻子，隐藏能力者，失子后归来复仇 |
| Dante | Ethan Blackwell | 家族继承人，误判妻子并做出抛弃选择 |
| Lily | Chloe Hart | 假孕白月光，直接陷害者，伪继承人来源 |
| Victor | Mason Voss | 竞争资本掌控者，救下 Serena 并扶她回牌桌 |
| Corleone 家族 | Blackwell Group | 男主权力来源，被女主夺走命脉 |
| Night Owl | Night Harbor | 女主隐藏身份 / 全球港口调度系统 |

## 3. 新壳承载逻辑

Serena 不是“忽然有钱回来打脸”。

她婚内被 Ethan 当成只会做家务的太太，但实际上她曾匿名修复 Blackwell 的供应链模型，设计过 `Night Harbor` 原型。Ethan 不知道这套系统救过家族，只把功劳给 Chloe 的家族资源。

失子后，Mason 救下 Serena，并把她的系统推成灰港资本的核心引擎。五年后，Serena 掌握 Blackwell 最依赖的三条命脉：

- 港口调度权；
- 医药冷链认证；
- 海外信用保险。

因此她归来时能让 Ethan 当场明白：她不是回来争宠，是回来切断他站着说话的地面。

## 4. 首批强节点迁移

| 源本节点 | 新本等效承载 | 洗稿判断 |
|---|---|---|
| Eve 吞孕检单签离婚 | Serena 把超声报告塞进红酒杯吞下碎片，签股权净身协议 | 保留“她知道、他不知道”的信息差 |
| Dante 救 Lily 留 Eve | Ethan 指挥安保先救 Chloe 和假继承人，关闭 Serena 所在冷链舱消防门 | 保留现场选择和不可逆伤害 |
| Eve 流产 + Victor 救 | Serena 失去孩子，Mason 揭示她是 Night Harbor 设计者 | 保留失子与复仇发动 |
| 拍卖会归来 | Blackwell 慈善晚宴 / 港口续约会 Serena 归来 | 替换黑帮拍卖，保留公开权力场 |
| Eve 断军火线 | Serena 冻结 Blackwell 港口、冷链和保险 | 保留资源压制 |
| Lily 假孕公开 | Chloe 的伪孕、外遇、继承人信托骗局在董事会直播中崩塌 | 保留连续打脸 |
| Dante 找到孕检单和毛衣 | Ethan 找到旧超声、未完成婴儿毯、冷链事故锁门记录 | 保留实物定锤 |
| Dante 跪求原谅，枪响 | Ethan 交出 CEO 印章和股权，Serena 给出“公开自毁 / 死”的选择，枪响硬切 | 保留权力坍塌与强钩子 |

## 5. 首批 1-10 集功能

首批必须完整覆盖一个闭环：

1. Serena 被替代。
2. Ethan 做出抛弃选择。
3. Serena 失去孩子。
4. Serena 归来掌权。
5. Chloe 公开崩塌。
6. Ethan 找到实物定锤。
7. Ethan 当众跪下。
8. Serena 把“是否原谅”转成更残酷的新追问。

## 6. 不能做的写法

- 不把 Serena 写成只会查账的审计者。
- 不把高压节点写成“屏幕播放证据”。
- 不让 Ethan 在 EP8 已经完全知道全部真相，EP9 必须保留更痛的实物定锤。
- 不把 Chloe 一次证据就打完，必须让她先用 Ethan 的体面反扑。
- 不把 Mason 写成纯工具人，他必须既帮 Serena 又让 Ethan感到自己被替代。

## 7. 本轮交给 P3 的核心问题

- EP1-3 如何把羞辱、抛弃、失子压到足够强？
- EP4-6 如何让 Serena 的归来不是报告，而是现场夺权？
- EP7-8 如何把 Chloe 的公开崩塌写成人物被迫行动，而不是证据展示？
- EP9 如何保留实物定锤，避免被 EP8 偷吃？
- EP10 如何让 Ethan 的权力坍塌既像豪门商战，又保留源本“跪求原谅”的短剧冲击？
