# PRD v15-v4workflow High-pressure Scene Viability

## 结论摘要

当前新壳“海外豪门商战 / 家族企业”可以承载源本首批 1-10 的高压节点，但必须把高压戏放在人和人的选择、羞辱、背叛、身体后果上，而不是放在文件、授权、屏幕或董事会流程上。

## 不可进入正文的风险

以下写法本轮禁止作为高压场发动机：

- 让系统自动宣判 Adrian 有罪；
- 让文件连续播放替 Chloe 崩溃；
- 让律师/董事会读出全部真相；
- 让 Adrian 只是“看到资料然后震惊”；
- 让 Serena 像审计员一样拿材料说服全场。

## 通过判断

高压节点可进入 Episode Plan。

原因：每个节点都能找到“谁压谁 / 谁被迫选择 / 谁失去什么”的人际发动机。文件、屏幕、道具只作为让人物失控或选择的辅助，不作为高潮本体。
