# PRD V18 v8workflow 二次需求确认

## 0. 本步用途

读完源本后，确认豪门商战 / 家族企业权力斗争是否能承载源本追看机制，并锁定变量边界。由于本轮任务已给定新壳、市场、输出范围和力度，本文件记录主创建议后的确认口径。

## 1. 新壳承载判断

可以承载。

源本核心不是“黑帮有枪”，而是：

- 一个封闭权力系统里，男主拥有绝对解释权；
- 女配借男主偏爱制造假弱者叙事；
- 女主被排出权力中心并承受不可逆身体损失；
- 五年后女主以更高层级的权力资源回归；
- 男主被公开证据、商业损失和亲子真相逐步击穿。

豪门商战可以用家族董事会、慈善基金会、继承信托、医疗体系、供应链控制权和公开股东会承载这些体验。

## 2. 改写力度

采用中度偏重。

- 保留追看发动机、情绪走势、爽点周期、误判结构、公开揭谎和火葬场节奏。
- 重建事件外形、场面、证据形式、权力工具和对白。
- 不按集翻译源本，不保留黑帮拍卖、军火断供、枪口求饶等识别性场面。

## 3. 变量边界

### 贴近源本

- 女主曾隐忍三年并怀孕。
- 女配伪怀孕并设计事故。
- 男主事故现场选择女配，导致女主流产。
- 五年后女主以强者身份回归。
- 女主公开揭穿女配，男主逐步得知真相。
- 首批结尾男主以极低姿态求原谅。

### 等效迁移

- 黑手党家族 -> 上市家族企业 Whitmore Group。
- 军火市场控制权 -> 全球供应链风控系统、债务/订单/董事投票控制权。
- 黑帮宴会 -> 家族慈善晚宴、股东大会、董事会特别听证。
- 黑医和亲子证据 -> 私人诊所主任、基金会医疗档案、信托继承亲子证明。
- 保险箱遗物 -> 家族档案库/婚前公寓储物柜里的孕检单、婴儿毯、旧婚戒。
- 枪响卡点 -> 直播股东听证里，男主是否签署自毁证词并公开承认。

### 自由改写

- 女主新身份：Kestrel，不是武器设计师，而是危机投资人、供应链风控系统设计者。
- 助攻角色 Julian Frost：资本盟友与救命人，参与搭建女主回归平台。
- 女主复仇方式：通过债务、审计、供应链冻结、董事投票和舆论场造成当众后果。

### 删除或弱化

- 黑帮私刑、狗笼、鳄鱼池等本轮首批不需要。
- 过度血腥的惩罚不进入 1-10。
- 长篇解释型董事会发言压缩成动作、投票、屏幕数字、现场资格失效等可视化后果。

## 4. 源本外形禁区

- 不写“纽约黑道女王”“军火线断供”“顶级黑帮拍卖会”。
- 不写女主直接拔枪压男主额头作为 E10 主动作。
- 不把 E7-E8 写成简单播放录音和众人震惊，必须让证据触发投票权冻结、继承资格暂停、基金会审计和 CEO 当场失控。

## 5. 风险提示

最大风险是短剧降级：商战容易变成文件、会议和英文金句。解决方式：

- 高压节点必须有现场动作链。
- 文件/屏幕信息出现后，必须触发人物被迫行动。
- 女主每次出手都要改变权力、关系、信息或情绪状态。

## 6. 确认结论

本轮按以下口径继续：

- 新壳：海外豪门商战 / 家族企业权力斗争。
- 集数：不变。
- 输出：1-10 集。
- 改写：中度偏重。
- 正文：中文动作/场景描述 + 英文对白。
- 验收重点：是否比 V17 有更清楚的场面抓手，是否接近或超过 V5 的场面迁移有效性。
