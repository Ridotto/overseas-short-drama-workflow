# PRD_v22_latest_v11_Adaptation_Boundary

## 必须保留的体验功能

- 低位妻子在私密 / 半公开场合被准继承人母亲压制。
- 女主真实怀孕的信息被男主错过，观众先知道，男主后知道。
- 男主在事故中选择女配，造成女主流产，这是不可洗白的原罪。
- 女主五年后高位回归，第一次公开见面就让男主的权力工具失效。
- 女配的“继承人母亲”身份必须被公开拆穿，并造成现场社会性死亡。
- 男主迟来得知自己害死双胞胎，必须有私人遗物击穿，不只靠报告。
- Ep10 需要公开下跪和交出核心权力，女主不给原谅。

## 必须避开的源本外形

- 黑手党 / 教父 / 军火市场 / 枪口威胁。
- 黑帮拍卖会、黑医证词、暗网武器设计师。
- 单纯大屏播放证据、亲子报告、当场验血作为高潮唯一动作。
- 女主派人打断腿、把人扔窗外等黑帮暴力外形。
- “白月光”中文网感称谓直译到英文台词。

## 可以自由换的外壳变量

- 家族名、公司名、行业、资本工具。
- 男配从黑帮敌人改为资本盟友 / 私募基金掌舵人。
- 女主隐藏能力从武器设计改为供应链风控、芯片授权、金融建模和商业证据链。
- 公开羞辱场从黑帮宴会改为家族信托仪式 / 股东晚宴 / 慈善拍卖。
- Ep10 残酷条件从枪改为高空露台 + 永久退出 Blackwood 的不可逆选择。

## 必须删除 / 修复的烂点

- 修复女配“假孕”和“有孩子”冲突：新本锁定为“她真有孩子，但孩子不是男主；她伪造父亲身份、孕周和产检记录”。
- 修复孩子数量冲突：新本锁定女主当年怀的是双胞胎。
- 修复高压节点证据化：Ep7-Ep8 的证据必须引发现实动作，例如信托门禁拒认、董事席冻结、家族徽章取下。
- 修复男主动机：他前期不是纯蠢，而是被继承人焦虑、旧情、家族董事会压力和 Serena 的伪证推着选择错人，但不能因此免罪。

## 新壳承载风险

- 豪门商战容易变会议和文件对白：每集必须有可见动作，如改签、撤席、徽章、门禁、竞价锤、车祸火、雨中跪、露台边缘。
- 商战权力线容易抽象：供应链断供、信托冻结、董事投票、专利许可撤回要落到人物当场失去什么。
- 男主跪求容易变口头忏悔：必须交出 Blackwood 控制权、创始人徽章和不可逆文件。
