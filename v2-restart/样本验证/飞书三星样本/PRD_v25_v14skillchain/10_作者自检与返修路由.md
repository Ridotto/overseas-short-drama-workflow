## 重点场面自检

### EP2 塔楼电梯事故

有效：

- Adrian 听见 Ava 说 `I'm pregnant` 后仍选择 Celeste。
- 事故不是远处消息，而是电梯、烟、火、钢缆、对讲断线的可见场面。

风险：

- 如果后续 reviewer 认为“塔楼电梯事故”仍接近源本“车祸爆炸”的功能外形，需要回 S4 换成更商业空间的等效危险，例如董事会大楼安保封锁 + 电力火灾 + 货梯坠落保留，但进一步弱化“爆炸”。

### EP4 慈善拍卖身份碾压

有效：

- Ava 的权力不是“有钱”，而是拿住债权和融资命门。
- Celeste 受辱有红酒白裙的可见画面。

风险：

- 红酒羞辱偏 melodrama，若目标市场要求更现实商务，可回 S7 改成“取消 Celeste 基金会主席资格 + 白裙被投影证据打亮”，但会损失短剧即时爽感。

### EP7-EP8 公开审判

有效：

- 假孕 / 事故 / 出轨 / DNA 分两集释放。
- Adrian 先护家族颜面，再被更大证据反打。

风险：

- 证据密度高。若 reviewer 认为信息过载，应回 S5 重新拆拍，把 EP7 的工单和语音留一个，EP8 再用 DNA 和 Celeste 反咬承接。

### EP9 旧物击穿

有效：

- 超声单、婴儿毯、尾戒都是触觉物件。
- Adrian 的悔不是口头总结，而是砸保险柜、跪雨、自扇。

风险：

- 自扇耳光较强，若海外口径嫌中式，可回 S7 改成他把拳头砸到玻璃门出血，保持自毁视觉。

### EP10 顶层餐厅枪响

有效：

- Adrian 交出权力和身份，不只是道歉。
- Ava 不软化，卡点仍强。

风险：

- 枪作为豪门商战外形略重，虽然只是少量极端安保物件。若主控希望新壳更纯商业，可回 S4 / S7 把枪替换为“公开认罪直播 + 高层露台边缘 + Ava 按下直播按钮 / 砸碎酒杯黑屏”，但当前版本保留源本“生死偿还”的强度。

## 返修路由建议

- 如果 reviewer 判“短剧降级 / 商业太冷”：回 S4 场面承载，优先重做 EP4、EP7、EP8 的可见后果。
- 如果 reviewer 判“改名复刻”：回 S2 洗稿边界和 S4 场面载体，重点审 EP2、EP10。
- 如果 reviewer 判“正文长剧化”：回 S7 台词视听，只压对白，不改主线。
- 如果 reviewer 判“信息太满”：回 S5 施工稿，拆分 EP7-EP8 证据释放。
- 如果用户不接受枪：回 S4 / S7，仅改 EP10 卡点载体。
