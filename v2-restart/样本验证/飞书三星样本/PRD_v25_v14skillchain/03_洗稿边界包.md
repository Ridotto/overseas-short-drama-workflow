# 03 洗稿边界包

## Skill 工位切换

当前工位：S2 洗稿边界 / 变量迁移 skill。

本工位决定继承体验、替换外形、自由改写和删除修复。不把防同构压过防洗飞。

本阶段不固定加载 0xsline 文档，以源本迁移、防洗飞、防复刻为主。

## 新壳总边界

新壳：海外豪门商战。

故事场域从黑帮帝国换成跨国家族集团与资本控制权争夺：

- 家族集团：Lancaster Global
- 家族继承 / 信托 / 董事会
- 慈善晚宴、艺术基金会、酒店资产拍卖、债务重组
- 企业塔楼、私人诊所、董事会宴会、顶层餐厅
- 控制权、融资线、供应链、媒体公关、合规调查

不能把新壳写成“董事会开会念报告”。每个商业动作必须变成能拍的场面：门被关上、电梯冒烟、屏幕亮起、合同砸脸、股权证书被撕开、全场静默、安保退后、记者镜头逼近。

3. 女主怀孕信息被男主错过或不信。
4. 事故现场男主主动选择救假怀孕旧爱，抛弃女主。
5. 事故造成孩子死亡，女主从爱转恨。
6. 五年后女主以高位身份公开归来。
7. 女主第一次回归必须当众碾压男主和女配。
8. 女配假孕、买凶 / 造事故、出轨 / 野种必须分阶段公开塌房。
9. 男主必须被“孩子真相 + 旧物”击穿，不只是听人说。
10. 第 10 集必须形成男主献上一切求原谅、女主冷拒的高压卡点。

## 自由改写

以下可以自由改：

- 角色英文名、集团名、资产类型。
- 回归身份的具体产业：资本、酒店、供应链、风控算法、家族信托。
- Julian 和 Ava 的关系是否被误会成订婚或资本联盟。
- 事故技术细节，只要保持“男主能救但没救”的罪感。
- 公开审判的证据排列，只要遵守分阶段释放。

## 必须删除 / 修复

- 删除黑手党、军火、黑帮教父作为默认外形。
- 删除大量枪战式威胁，除非作为极少数豪门安保极端场面。
- 修复源本怀孕周数口径：统一为 six weeks, twins。
- 避免“她只是变有钱回来”的浅层爽点，必须显示她掌握 Lancaster 的真实命门。
- 避免商业战变成 PPT：每个商业动作必须有当场后果。

## 同构红线

以下属于改名复刻风险：

- 仍写成“黑帮拍卖、军火断供、枪口对峙、地下势力宴会”。
- 事故仍是两辆车 + 卡车 + 爆炸。
- 女主仍靠黑道手下打人解决每场冲突。
- 男主仍是“教父 / 少主”式称谓，只换成 CEO 但行为不变。

## 防洗飞红线

以下属于洗飞：

- 男主不是主动选择抛弃女主，而只是“不知道她在危险里”。
- 女主失子不够具体，只靠口述带过。
- 五年后回归没有公开身份碾压。
- Celeste 的假孕和野种不当众塌房。
- Adrian 在 EP8 之前就彻底悔悟。
- EP10 女主因为他交资产就软化。

## 新壳可承载点

海外豪门商战最能承载的强场面：

- 家族信托继承资格。
- 集团控制权和投票权转让。
- 银团贷款撤回造成股价和董事会即刻崩盘。
- 慈善晚宴 / gala 上的大屏证据。
- 私人医疗记录、DNA swab、法律公证和审计报告。
- 企业塔楼电梯事故作为“豪门空间里的生死选择”。
- 媒体镜头和股东围观放大羞辱。

## S2 给下游的硬交接

S3-S7 必须遵守：

- 体验继承优先：背叛、失子、归来、公开碾压、真相击穿、冷拒。
- 外形必须换：黑帮 -> 豪门集团 / 资本 / 信托 /塔楼事故。
- 场面必须可看：不要让文件替代人戏，文件只能成为人戏里的武器。
- Adrian 的罪不能被淡化；Celeste 的恶不能只变成职场小人。
