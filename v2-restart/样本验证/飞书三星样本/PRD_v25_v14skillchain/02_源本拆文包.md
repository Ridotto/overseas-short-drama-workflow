# 02 源本拆文包

## Skill 工位切换

当前工位：S1 源本拆文 skill。

本工位只拆源本为什么有效，不设计新壳、不提前改剧情、不把源本外形当成必须照抄。

阶段加载 0xsline 参考：

- `rhythm-curve.md`：用于识别起势段的紧张蓄力、释放和卡点。
- `hook-design.md`：用于拆每集结尾的追问类型。
- `satisfaction-matrix.md`：用于拆压抑释放、身份碾压、打脸复仇、情绪爆发。

## 一句话故事核

一个被丈夫误判、被白月光陷害、失去孩子的隐忍妻子，五年后以隐藏巨头身份归来，公开夺权、揭穿骗局，让前夫从不可一世的掌权者坠成追悔疯狗。

## 核心期待链

观众连续追的不是“黑手党设定”，而是这条期待链：

1. 她什么时候不忍了？
2. 他什么时候知道自己错得多狠？
3. 假白月光什么时候当众塌房？
4. 女主归来后每次打脸能不能更狠？
5. 男主知道孩子真相后会崩到什么程度？
6. 女主会不会原谅？如果不原谅，她会怎么惩罚？

首批 1-10 集覆盖第一个小闭环：

- 1-3：背叛、事故、失子，完成压低。
- 4-6：女主强势归来，身份和伤疤开始反杀。
- 7-8：白月光假怀孕、野种、车祸阴谋公开引爆。
- 9-10：男主确认孩子真相，彻底跪地求原谅，女主拒绝。

## 高价值节点

### HV1：孕检单被吞

- 功能：把“她怀孕”变成观众知道、男主不知道的强信息差。
- 有效点：女主不是弱哭，而是把证据吞掉，形成决绝动作。
- 下游必须保留：隐藏怀孕证据不能轻易被男主知道；女主必须有一个可见的“把希望吞回去”的动作。
- 禁止照抄外形：厨房、意面、黑帮庄园不是必须保留。

### HV2：事故现场错误选择

- 功能：男主罪的根。必须是一场无法解释成“小误会”的可见选择。
- 有效点：Eve 说出怀孕仍被抛弃；旁人见证；危险即时逼近。
- 下游必须保留：男主必须主动选择救假怀孕者、放弃女主；这会直接导致孩子死亡。
- 禁止照抄外形：卡车车祸、出租车爆炸不是必须保留。

### HV3：失子醒来 + 新闻补刀

- 功能：女主从爱转恨的发动机。
- 有效点：身体创伤、孩子没了、电视里男主却在给假孩子庆祝。
- 下游必须保留：女主的恨不是口号，要有同时发生的视觉补刀。

### HV4：五年后公开身份碾压

- 功能：从受害者切到掌权者，让观众第一次爽。
- 有效点：全场见证、资金/资源碾压、男主当众失控、女主冷。
- 下游必须保留：不是私下复合戏，必须是公开权力场。

### HV5：Lily 私闯被反制

- 功能：让反派先尝痛，证明女主能惩罚。
- 有效点：Lily 带人闹事却被瞬间反包围；身体性惩罚放大爽感。
- 下游必须保留：反派主动来犯、女主设局等她、惩罚有可见后果。

### HV6：伤疤 + 医疗记录

- 功能：把男主从“愤怒质问”推向“疑心自己错了”。
- 有效点：伤疤是身体证据，医疗记录是事实锤。
- 下游必须保留：不能只靠女主口述，必须有身体 / 物件证据。

### HV7：公开审判白月光

- 功能：假怀孕、买凶、出轨、野种集中爆雷。
- 有效点：当众、证据连环、男主护短被反打脸。
- 下游必须保留：第一锤不让男主立刻相信，必须先让他因面子 / 旧情护她，再被第二锤打穿。

### HV8：男主发现旧物

- 功能：把外部真相变成内疚体验。
- 有效点：孕检单、婴儿毛衣、尾戒是触觉物件，能逼出崩溃。
- 下游必须保留：他不能只是“知道了”，要被具体遗物压垮。

### HV9：当众献上一切求原谅

- 功能：第一个小闭环的最高火葬场。
- 有效点：曾经掌权的人在公共场域失去尊严，女主仍不软。
- 下游必须保留：献出的必须是他最看重的权力 / 资产 / 身份；女主的回应必须冷硬。

## 源本首批的压放结构

- EP1-EP2：连续压低。女主被轻视、被误会、被抛弃。
- EP3：情绪爆发，同时埋复仇引擎。
- EP4-EP5：第一轮释放，女主回归打脸并证明资源真实。
- EP6：信息回流到男主，压出悔意前奏。
- EP7-EP8：第二轮强释放，Lily 公开塌房，男主同时被羞辱和定罪。
- EP9-EP10：情绪爆点转为追妻火葬场，男主跪下，女主拒绝，形成下一阶段追问。

## 人物关系 / 拉扯

### Eve

- 初始状态：爱、隐忍、希望被看见。
- 断裂点：危险中被抛弃，孩子死亡。
- 归来状态：冷、掌控、拒绝解释，主动让对方自证错误。
- 首批拉扯：她不是单纯赢，她要让 Dante 每一步都亲眼看见自己当年的盲和恶。

### Dante

- 初始状态：权力高位、错信 Lily、轻视 Eve。
- 变化路径：轻蔑 -> 愤怒 -> 不安 -> 被证据击穿 -> 自毁式悔恨。
- 首批不能提前发生：不能过早清醒、不能过早保护 Eve、不能在 EP7 前彻底相信 Lily 有罪。

### Lily

- 功能：假柔弱、真施害；她不断逼女主，给女主反杀机会。
- 变化路径：登堂入室 -> 阴谋杀人 -> 五年后仗势挑衅 -> 被公开剥皮 -> 反咬男主。
- 首批不能提前发生：不能太早认罪；每次证据出来前都要继续演。

### Victor

- 功能：救命人、资源扶持、男主嫉妒催化剂。
- 首批功能不重在感情，而是证明 Eve 五年后有新资源和新秩序。

## 信息边界 / Do Not Consume

- EP1：观众知道 Eve 怀孕，Dante 不知道；Lily 必须不知道或至少不能公开知道孕检单细节。
- EP2：Eve 说出怀孕，但 Dante 不信；他不能因此回头。
- EP3：女主失去孩子，男主仍沉浸假孩子叙事。
- EP4：夜枭身份可公开；事故完整真相不能公开。
- EP5：Lily 设计车祸可由女主点出，但不能让所有证据一次性完成。
- EP6：男主只确认 Eve 当年怀孕；双胞胎和旧物不能提前给完。
- EP7：假怀孕、车祸买凶先公开；野种证据作为下一拍更大打脸。
- EP8：野种和男主罪同时引爆；孕检旧物仍留给 EP9。
- EP9：双胞胎、婴儿衣物、尾戒打穿男主。
- EP10：女主不能原谅；只能形成“他愿意拿一切换原谅”的下一阶段火葬场。

## 源本含混点

- EP6 医疗记录说“两个月身孕”，EP9 孕检单写“双胞胎，6周”。下游可统一为“six weeks, twins”，避免前后口径跳动。
- 源本 1-10 的主要有效性来自“人和人当众互相压迫”，不是黑帮枪械本身。新壳如果改成纯法律文件或商业会议，会短剧降级。
- 男主跪求原谅是强情绪节点，但海外豪门商战新壳里不应只靠跪地礼俗成立，必须搭配交出控制权、公众见证、身体失控或身份坍塌。

## S1 给下游的硬交接

下游必须保活：

- 观众追问：她何时反击、他何时知道、她会不会原谅。
- 压放：前 2 集必须足够痛，4-5 集必须给第一波身份 / 资源碾压，7-8 集必须公开塌房，9-10 集必须男主崩溃求赎。
- 新壳替换不能洗飞：事故现场错误选择、失子、公开审判、旧物击穿、献上一切求原谅，都必须有等效强场面。
- Do Not Consume：怀孕真相、假孕真相、野种真相、双胞胎旧物、女主是否原谅，必须按阶段释放。

结论：pass。
