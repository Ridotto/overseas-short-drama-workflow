# 13 主控版本对比与拍板

## 结论

V25 判定：`workflow execution complete through S11 / creative outcome revise / delivery outcome not pass yet`。

不是 block。V25 比 V24 更像短剧，也证明 v14 的 `writer 施工稿` 这一步有效：正文不再主要像报告、会议或流程说明，EP2、EP4、EP7-EP9 都能读出现场动作、压迫、可见后果和短剧卡点。

但 V25 不能直接 pass。它还没超过 V5 的可送审强度，主要因为：

1. EP10 枪口黑屏又滑回源本高辨识外形；
2. EP6-EP10 反复使用“他听见她怀孕却离开”这一把刀，导致终局新鲜刀口变弱；
3. Celeste “假孕 / 孩子非亲生”口径容易误读；
4. EP7 Adrian 的阻止动机不够准确，像还在护 Celeste，而不是护 Lancaster 家族颜面和控制权；
5. EP5 商业崩盘仍有轻微口播感。

所以本轮最合理动作不是推翻 v14，也不是继续加 workflow 规则，而是按 reviewer 指出的层级做一次窄返修，再做窄验证。

## 打回原因：不是套路问题，是局部高辨识外形过近

不能把“火葬场跪求、交出全部、女主不原谅、极限卡点”当成要删除的东西。

这些都是短剧常见套路，可以用，而且应该用。套路之所以是套路，是因为它能快速调动观众期待。

V25 被打回的不是“用了套路”，而是 EP10 的一串动作组合太贴源本外形：

```text
男主湿透冲入公共高压场
  -> 当众跪求
  -> 交出全部
  -> 女主冷拒
  -> 枪抵眉心
  -> 枪响黑屏
```

这已经不是单个 trope，而是源本高辨识火葬场外形的近距离复用。尤其“枪抵眉心 + 一命换两命 + 枪响黑屏”不是豪门商战天然长出来的动作，更像把源本黑帮枪口审判搬到新壳里。

所以修法不是降低短剧感，也不是删火葬场，而是保留火葬场功能，换掉源本专属外形：

- 可以保留当众跪求；
- 可以保留交出全部；
- 可以保留女主冷拒；
- 可以保留极限卡点；
- 但不要再用源本同款枪口审判作为卡点。

这条判断也适用于后续 reviewer：不能因为某个桥段是常见短剧套路就判复刻；只有当“动作组合 + 情绪功能 + 视觉外形”一起贴近源本高辨识段落时，才判同构风险。

## V25 vs V24

### V25 明显强于 V24 的地方

V24 的问题是“上游专业、正文偏渲染”：材料很多，但正文经常像把追问表和场景载体翻译成剧本。

V25 的改进是真实的：

- S5 先写单集短剧施工稿，正文更像从戏里长出来；
- EP2 电梯事故有烟、火、对讲断线、钢缆声，视觉冲击比 V24 更直接；
- EP4 拍卖厅有红酒染裙、债权砸脸、拍卖槌落下，比 V24 更有当众羞辱；
- EP7-EP8 不是纯听证会，有医生、转账单、大屏、棉签、孩子哭声、墨水掉落；
- EP9 旧物击穿有工作室、保险柜、婴儿毯、尾戒、雨中跪楼，情绪外化明显；
- 台词整体比 V24 更短，攻击、防守、揭露功能更清楚。

这说明 v14 的核心修正方向成立：**writer 不能直接拿上游规格写正文，必须先把单集排成短剧施工稿。**

### V25 不如 V24 的地方

V24 的 EP10 用 SEC 直播自白，虽然偏冷，但更贴豪门商战 / 出海商业壳。

V25 EP10 用枪抵眉心、枪响黑屏，短剧冲击更强，但外形风险更大，和源本枪口审判太近。

结论：V25 的短剧感 > V24；类型边界稳定度在 EP10 上 < V24。

## V25 vs V5

### V25 接近 V5 的地方

V25 已经找回 V5 的一部分强度逻辑：

- 证据不只是播报，开始绑定当众后果；
- 女主反击不只靠说，她能让对方失去脸面、资产、继承叙事和公众信任；
- EP7-EP8 有连续公开审判，不再只是静态解释；
- EP9 男主悔悟不是纯台词，有旧物、跪雨、自扇、玻璃门后的保安。

这比 V22 / V24 的“流程稳定但偏平”更接近 V5。

### V25 仍不如 V5 的地方

V5 的核心优势是动作后果链更硬：证据一出现，马上触发权力变化、身份剥夺、物件变化和可见羞辱。比如胸针摘除、基金冻结、继承墙红封、黑卡被剪、物证室旧物等，每个后果都很清楚。

V25 虽然更有画面，但有两处不够硬：

- EP5 商业打击还偏 CFO 解释，缺一个更强的现场后果；
- EP10 用枪制造强度，但没有像 V5 前面那样从豪门商战机制自然长出不可逆后果。

另外，V5 当时能 pass，是因为第 10 集枪响作为风险被显性留给导演拍板，前 1-9 集已经非常稳。V25 不同：它除了 EP10 枪外形风险，还有 Celeste 真相口径和信息重复消费问题，所以不能照 V5 那样 pass。

结论：V25 短剧感接近 V5，但还没有超过 V5；当前不应交导演。

## V25 vs V22

V22 的优势是完整链路已经能跑，故事相对清楚；问题是功能排序和源本贴得更近，整体像“强机制同构 + 有效外形迁移”。

V25 比 V22 好在：

- 场面更像短剧戏；
- 角色动作和身体反应更多；
- EP2 / EP4 / EP7-EP9 更有视觉记忆点；
- writer 施工稿确实减少了“把材料渲染成正文”的问题。

V25 不如 V22 的地方：

- V22 的终局卡点在类型边界上没有 V25 枪口这么危险；
- V25 因为追求强场面，EP10 更容易滑回旧外形。

结论：V25 的内容强度 > V22；边界干净度在 EP10 上 < V22。

## 最小返修建议

只修四处，不重跑全链。

### 1. EP10 卡点回 S4 / S5

目标：保留“男主交出一切求原谅，女主不收”的强度，删掉枪口外形。

可替换方向：

- Adrian 当众按下不可撤销控制权转让；
- 公开直播认罪，亲手毁掉 Lancaster 股价和家族信用；
- 董事会现场投票剥夺他 CEO / heir status；
- Ava 让他亲手销毁 Lancaster 家族信物或签署自毁条款；
- 卡在他即将按下 `GO LIVE` 或转让生效的一拍。

不要把 EP10 改成温和道歉戏。

### 2. Celeste 真相回 S3 / S8 / S7

锁死口径：她不是“从未怀孕”，而是伪造 Adrian 的继承人亲子叙事。

正文措辞要从 `fake pregnancy file` 改成类似 `forged prenatal / paternity file for the Lancaster heir`，避免和五年后孩子非亲生冲突。

### 3. EP6-EP10 信息层次回 S8 / S5 / S7

每集只保留一个新刀口：

- EP6：确认 Ava 当年怀孕；
- EP8：公开确认 Celeste 知情并设局，Adrian 的选择也被暴露；
- EP9：旧物击穿，他第一次触摸两个孩子存在过；
- EP10：不是重复“我听见你怀孕”，而是公开认罪 + 交权 + 求她看着他失去一切。

### 4. EP7 Adrian 状态回 S8 / S7

他可以阻止 Ava 继续公开，但动机应是保 Lancaster 家族脸面、股价、控制权，不是继续替 Celeste 洗白。
