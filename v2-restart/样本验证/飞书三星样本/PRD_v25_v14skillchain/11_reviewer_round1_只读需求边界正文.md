## verdict

不是 block。首批 1-10 集已经像短剧正文，不像分析报告；大部分场面有可见动作、空间、物件、声音和当众后果。但当前稿仍有几处必须返修的风险：核心真相口径不够稳、Adrian 的状态在 EP7 有漂移、孩子真相被多次提前消费，EP10 枪口卡点有回到旧外形的风险。

## 致命阻断

无明确致命阻断。

按第一轮可见材料判断，没有出现“整串黑帮外形复刻”“正文整体报告化”“完全洗飞背叛失子归来复仇主体验”的硬失败。现有问题属于必须修的结构 / 状态 / 边界风险。

## 应改风险

### 1. Celeste 假孕与五年后孩子的真相口径容易冲突

问题：

EP7 大屏揭出“假产检账单 / fake Celeste's pregnancy file”，EP8 又安排孩子到场做 family trust heir verification，结果是 `No biological relation`。如果“假孕”字面理解为她当年没有怀孕，五年后孩子就缺少清楚来源；如果她确实怀了 Nate 的孩子，只是伪造成 Adrian 的继承人，那么 EP7 的“假孕”表达会误导核心真相。

返修层级：

- 回 S3 / S8：先锁死真相口径，Celeste 到底是假怀孕、借别人的孩子、还是怀了 Nate 的孩子并伪造 Adrian 亲子关系。
- 回 S7：把正文措辞从“fake pregnancy file”改成更准确的“forged paternity / prenatal file for Adrian”一类表达，避免观众以为她从未怀孕。

### 2. EP7 Adrian 状态漂移：已知 Ava 怀孕受害后，仍像在保护 Celeste 和家族脸面

问题：

EP6 Adrian 已看到 Ava 烧伤、听到“Your child was inside me”，并在档案室确认 `Pregnancy positive. Estimated six weeks.`。但 EP7 看到 Celeste 假文件、工程师收款、服务电梯工单和语音后，他仍说 “Enough.”、“You're humiliating my family.”，还抬手让安保上前。这个反应会削弱他的悔悟曲线，也让他显得不是错信，而是继续主动压真相。

返修层级：

- 回 S8：明确 EP6 后 Adrian 的状态是“否认、恐惧、护家族股价”，还是“仍维护 Celeste”。两者不能混。
- 回 S5 / S7：EP7 的动作和台词要改成更准确的短剧状态，例如他试图阻止公开扩散、保护 Lancaster 股价，而不是像替 Celeste 挡刀。

### 3. 孩子真相从 EP6 到 EP10 被重复使用，EP10 终局卡点有被削弱风险

问题：

EP6 Ava 直接说“Your child was inside me”，档案室确认怀孕；EP8 Celeste 当众说 “You heard Ava in that elevator... You still walked away”；EP9 Adrian 看到超声单和婴儿毯并跪雨自责；EP10 又以 “I heard you in the elevator” 作为认罪核心。每一拍单独都成立，但连续使用同一真相，会让 EP10 的冲击不够新。

返修层级：

- 回 S3 / S8：重分配 Do Not Consume，明确每集只释放一层新信息。
- 回 S5：让 EP6 只确认“她怀孕”，EP8 公开确认“Celeste 知情并设局”，EP9 用旧物击穿“他从未看见她”，EP10 再兑现“他承认自己听见并选择离开”。现在这个层次已经有雏形，但台词重复度需要压低。
- 回 S7：删掉重复认罪句，保留每集唯一新刀口。

### 4. EP10 枪口黑屏卡点有旧外形回流风险

问题：

S2 已明确删除黑手党、军火、枪战式威胁作为默认外形。EP10 让安保把枪交给 Ava，她用枪抵住 Adrian 眉心并枪响黑屏。这个卡点很强，但视觉外形会把豪门商战复仇拉回“枪口生死威胁”，也容易和“地下车库枪口对峙”的源本外形产生联想。

返修层级：

- 回 S4 / S5：重新设计 EP10 的豪门商战等效高压动作，用控制权、公开认罪、继承权、资产处置、媒体直播或身体性羞辱承载“一命抵两命”的压迫，不默认用枪。
- 如果保留枪，回 S2 明确它为什么不是旧外形复刻，并在 S4 给出豪门空间内的合理来源和后果；否则当前不建议保留。

### 5. EP5 商业打击有轻微报告播报风险

问题：

EP5-1 的融资线、供应商、2021 rescue package 主要由 CFO 口头说明。信息短，不算严重报告化，但这是首批里最接近“会议 / 报告”的段落。它承担的是 Ava 真正伤到 Lancaster 帝国的证明，如果只靠口播，会弱于前后几场的可视动作。

返修层级：

- 回 S4：补一个可见后果原料，例如董事手机同时震动、银行代表离席、交易屏红线跳停、供应商代表当场撤牌。
- 回 S5 / S7：把 CFO 解释压成一两句，让画面和人的反应承担商业崩盘。

## 可选优化

### 1. EP1 “吞超声报告”很强，但可以更清楚区分她的选择与误会来源

现在 Adrian 没看清报告，Ava 主动吞下，观众能理解为她不再给他机会。若下游担心“为什么不直接告诉他”，可以补一个很短的动作：她要递出报告时被 Celeste 打断 / Adrian 转身护 Celeste。不要加长解释。

返修层级：S7 小改即可。

### 2. Julian 的存在感够用，但 EP10 可以避免让他只做嘲讽旁观者

Julian 在 EP3 建立了“救她、扶她成为 creditor”的功能，EP10 只有 “Pathetic.” 和 “Ava, don't.”，不影响主线，但略工具人。

返修层级：S5 / S7 可选优化，不是必须返修。

## 是否像短剧

像短剧。

理由：

- EP1 离婚 + 吞报告 + 事故电话，开局抓人。
- EP2 生死选择清楚，男主主动选择救 Celeste，罪成立。
- EP4 拍卖厅当众身份碾压，有红酒染裙、债权砸脸、拍卖槌落下。
- EP7 / EP8 晚宴连环证据公开，有当众羞辱和追问推进。
- EP9 旧物击穿、雨中自责，情绪外化明显。
- EP10 资产交出 + 冷拒 + 高压卡点，具备首批小闭环。

## 是否有明显报告播报

没有整体报告播报。

局部风险在 EP5-1，商业打击主要靠 CFO 解释，建议用可见后果替代部分口播。其余场次基本是人、物、空间和当众后果在推动。

## 是否有明显长剧冷对白

没有明显长剧冷对白。

台词总体短，攻击 / 防守 / 揭露功能清楚。少数商业解释句和 EP7 Adrian 的状态句需要重写，不是因为长，而是因为信息口径和人物状态不稳。

## round1 总结

方向可继续，但不应直接 pass。最少必要返修是：

1. 锁死 Celeste 假孕 / 孩子 / 亲子关系真相口径。
2. 调整 EP7 Adrian 的状态，避免和 EP6 已知事实冲突。
3. 重分配 EP6-EP10 孩子真相释放层次，减少重复消费。
4. 重新评估 EP10 枪口黑屏是否违反新壳边界。
5. 把 EP5 商业崩盘从口播再往可视场面推一步。
