# 01 最小需求确认

## Skill 工位切换

当前工位：S0 需求确认 skill。

本工位只确认是否足够开工，不分析源本、不写新本、不提前替 writer 排戏。

## 本轮任务确认

- 任务类型：基于源本的洗稿 / 换壳改写，不是从零原创，不是简单改名。
- 源本：`[修改版] 黑手党少主痛哭求原谅_前妻她杀疯了.txt`
- 新壳：海外豪门商战。
- 输出范围：首批 1-10 集，集数不动。
- 输出语言：对白默认英文；动作行 / 场景描述可用中文。
- 正文形态：可拍短剧剧本，必须 page-to-screen，观众能看到 / 听到动作、空间、身体反应、物件、声音和当众后果。
- 执行 workflow：`workflow_spec_v14_skill工位链候选版.md`
- 工位规格：`skill_chain_spec_v1.md`
- 验收协议：`workflow_execution_protocol_v1.md`
- 输出目录：`v2-restart/样本验证/飞书三星样本/PRD_v25_v14skillchain/`

## 改写力度

本轮按 v14 的洗稿逻辑执行：

- 保留源本有效体验：背叛、错信、失子、女主隐忍后归来、公开碾压、假白月光坍塌、男主追悔。
- 替换高辨识外形：黑手党、军火、黑帮拍卖、枪口威胁、帮派宴会等不作为新本默认外形。
- 新壳承载：用豪门集团、董事会、股权、信托、融资、慈善晚宴、企业塔楼、合规调查、舆论与控制权争夺承载同等压力。
- 不为反同构牺牲短剧强度：不能把强场面写成会议纪要、法务邮件、屏幕播报。

## 目标市场和语感

按海外短剧口径处理，主受众偏欧美 female-skewed billionaire revenge / hidden identity / payback。

本轮 S0 阶段只加载 0xsline `genre-guide.md`，用于确认题材映射和英文台词注意点：

- 豪门商战可映射为 Billionaire / CEO、Family Trust、Charity Gala、Hidden Identity、Revenge / Payback。
- 英文台词要短、直接、口语化，避免中式逐字翻译。
- 文化陷阱：减少“跪地道歉”作为唯一表达；如保留身体性屈辱，必须变成具体戏剧处境，而不是中式礼俗。

## 本轮禁区

- 不读取 V3 / V5 / V22 / V24 正文来模仿。
- 不沿用旧 workflow 当默认流程。
- 不把源本黑手党外形整串搬到新本。
- 不把新壳写成纯文件、会议、屏幕播报。
- 不让 S4 替 writer 写戏。
- S5 施工稿必须像戏的粗稿，不得写成字段表。
- S6 / S7 正文不出现内部 workflow 标签。
