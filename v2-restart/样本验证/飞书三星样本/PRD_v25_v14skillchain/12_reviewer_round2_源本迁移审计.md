## 最终 verdict

revise。

不是 block。源本首批有效性基本迁过来了，writer 施工稿也确实把追看机制转成了短剧戏，不是任务单。但当前不能 pass，原因是两个红线风险仍需最少返修：EP10 枪口卡点过度贴近源本外形；孩子真相和 Adrian 听见怀孕这条信息从 EP6 到 EP10 重复敲打，终局新鲜刀口不够清楚。

## 源本有效性是否洗飞

未洗飞。

源本首批的有效性是：

- EP1-2：隐忍妻子被旧爱假孕压垮，怀孕信息被男主错过 / 不信，事故里男主主动救旧爱、抛弃女主。
- EP3：女主醒来失子，新闻补刀，救命盟友给她复仇通道。
- EP4-5：五年后公开回归，用真实资源碾压男主帝国，并先惩女配。
- EP6-8：伤疤、医疗记录、公开审判、野种真相，把男主从高位打到社死和内疚。
- EP9-10：旧物击穿，男主献上一切求原谅，女主冷拒。

V25 新稿基本保住了这些追看功能：

- 车祸爆炸换成塔楼服务电梯火灾，仍是即时生死选择。
- 军火断供换成融资线、债权、酒店资产和信托控制权，仍能伤到男主帝国。
- 黑帮拍卖换成慈善拍卖 / 酒店债权竞拍，保住公开身份碾压。
- 黑道宴会审判换成 family trust 慈善晚宴，保住当众塌房。
- 废弃仓库保险箱换成旧工作室保险柜，保住旧物击穿。
- 黑帮地契 / 教父徽章换成 trust voting transfer、CEO resignation、asset authorization，保住献上一切的功能。

结论：主体验没有被洗飞。

## 是否改名复刻

整体不是改名复刻，但 EP10 有局部复刻风险。

不是整体复刻的原因：

- 关键事件载体已换：公路车祸 -> 塔楼电梯事故；军火线 -> 融资 / 债权 / 供应链；黑帮宴会 -> 家族信托慈善晚宴。
- 新本不是只把 Dante 改成 Adrian、Corleone 改成 Lancaster；多数压力来自豪门集团、资本、信托、媒体和董事会。
- EP4、EP5、EP7、EP8 的场面里，文件不是报告，而是当场打人的武器。

局部风险：

- EP10 的轮廓仍非常接近源本：男主湿透冲入公共餐厅、献上全部资产、跪求原谅、女主拿枪抵眉心、枪响黑屏。
- S2 明确要求删除黑手党、军火、枪战式威胁作为默认外形；但 S4 / S5 仍把“枪抵眉心”作为 EP10 关键卡点交给正文。这不是 S7 polish 才出现的问题，源头在 S4 / S5 的场面承载选择。

返修层级：

- 回 S4 / S5：重做 EP10 的等效高压动作。
- 如坚持保留枪，必须回 S2 写清这是例外，不是继承源本枪口外形；否则当前应默认改掉。

## Do Not Consume 是否提前消费

没有出现致命提前消费，但有重复消费削弱。

当前链路本来设计了分层：

- EP6：Adrian 只确认 Ava 当年怀孕。
- EP8：Celeste 当众反咬，公开 Adrian 听见却离开。
- EP9：双胞胎和旧物击穿。
- EP10：献上一切求原谅，Ava 不原谅。

问题在于最终正文里 EP6-EP10 反复围绕同一个句柄打：

- EP6 Ava 直接说 “Your child was inside me”，医疗记录又确认怀孕。
- EP8 Celeste 说 “She said she was pregnant. You still walked away.”
- EP9 旧物确认 twins, six weeks，并让 Adrian 雨中喊 “I killed them.”
- EP10 Adrian 又说 “I heard you in the elevator. I heard you say pregnant.”

这不算洗飞，也不是逻辑错误；但 EP10 使用的信息已经不是新刀口。源本 EP10 的有效性来自“他把最值钱的权力和命都交出来，女主仍不软”，不是再次解释他听见了怀孕。

返修层级：

- 回 S8：重新标注 EP10 可用信息，避免把已公开信息当新爆点。
- 回 S5 / S7：EP10 台词减少“我听见你怀孕”的复述，把重点压到“我公开认罪并实际交出控制权，但你仍不原谅”。

## writer 施工稿是否把源本追看机制转成短剧戏

是。

S5 不是字段表，也不是剧情任务单。它已经把源本追看机制转成每集可写的短剧粗稿：

- 每集有开场承接、中段升压、结尾钩子。
- EP2 的电梯生死选择写成了空间压力，不是事故说明。
- EP4 的资本碾压有拍卖槌、债权文件、银行代表、安保呼叫器。
- EP7 / EP8 的证据释放有当众围观、屏幕、医生、律师、孩子、闪光灯。
- EP9 用旧物和闪回打 Adrian，不靠解释。
- EP10 用文件散落、公众围观、跪地、枪口制造首批最高压。

但 S5 同时继承了两个上游风险：

1. S4 给了 EP10 枪抵眉心，导致正文局部贴回源本外形。
2. S3 / S8 对 EP6-EP10 的信息层次虽然有标注，但 S5 / S7 没把“每集唯一新刀口”压得足够干净。

所以 writer 施工稿方向是合格的，返修不应推倒重来。

## 最少必要返修

### 1. 重做 EP10 的豪门商战高压卡点

目标不是削弱，而是换掉枪口外形。

返修层级：S4 / S5，之后 S7 跟修正文。

最小要求：

- 保留 Adrian 当众交出 Lancaster 的控制权、CEO 身份、资产授权和公开认罪。
- 保留 Ava 冷拒，不原谅。
- 把“枪抵眉心 / 枪响黑屏”换成更贴新壳的不可逆动作，例如公开提交认罪声明、按下控制权转让生效、当众销毁 Lancaster heir signet、让董事会即刻投票剥夺他，或让他亲手签署自毁条款。
- 卡点仍要有“他愿意付命，她仍不收”的压迫感，但不要用源本同款枪口。

### 2. 精准化 Celeste 假孕 / 孩子非亲生口径

返修层级：S3 / S8 先定口径，S7 改措辞。

最小要求：

- 明确 Celeste 是“伪造 Adrian 的继承人孕检 / 亲子叙事”，不是从未怀孕。
- EP7 的证据措辞改成 forged prenatal / paternity records for the Lancaster heir，避免和 EP8 孩子到场 DNA 冲突。

### 3. 压低 EP10 对“我听见你怀孕”的重复复述

返修层级：S8 / S5 / S7。

最小要求：

- EP10 不再把“我听见你在电梯里说怀孕”当主要新爆点。
- Adrian 可以承认“我 chose her / walked away”，但重点应放在公开认罪、交权和求她亲眼看见他失去一切。
- EP9 已完成旧物击穿，EP10 不要再重复 EP9 的痛点台词。

### 4. 修 EP7 Adrian 的反应动机

返修层级：S8 / S5 / S7。

最小要求：

- 保留他在 EP7 前未彻底倒向 Ava、仍被家族颜面和 Celeste 叙事牵住。
- 但台词不要像无脑保护 Celeste。应改成“不要在董事和媒体前毁掉 Lancaster”，让他的阻止动机是家族控制权 / 舆论崩盘，而不是继续替 Celeste 洗白。

## 不建议扩张的返修

- 不需要重写 EP1-EP6。
- 不需要推倒重新做故事架构。
- 不需要把所有商业元素加厚。
- 不需要让 reviewer 逐句改英文台词。
- 不需要为了反复刻牺牲源本压放结构。

## round2 结论

V25 v14skillchain 首批是可修的 revise，不是 block。

它的问题不是“写得不像短剧”，也不是“源本完全洗飞”。真正要修的是：把 EP10 从源本枪口外形里拔出来，把孩子真相释放层次压干净，把 Celeste 真相口径写准。修完这些，再进入主控对比和用户 / 导演判断会更稳。
