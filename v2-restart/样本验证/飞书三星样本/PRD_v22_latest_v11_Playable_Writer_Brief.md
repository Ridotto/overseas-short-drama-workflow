# PRD_v22_latest_v11_Playable_Writer_Brief

## Run Contract

- 新壳：海外豪门商战，Blackwood Group + Vesper Systems。
- 目标市场：海外短剧。
- 输出语言：台词英文；动作和场景描述中文。
- 正文用途：人审稿 + AI 视频生成 page-to-screen。
- 改写力度：保追看机制，换事件载体。

## Current Story State

Ava Cross 是 Blackwood 家族被轻视的少夫人，真实身份是 Vesper Systems 隐名创始人。Adrian Blackwood 以为她无能、嫉妒 Serena。Serena 用“怀着 Blackwood 继承人”的身份夺位。观众知道 Ava 怀着 Adrian 的双胞胎，Adrian 不知道。

五年后，Ava 以 Vesper 掌舵人身份归来，手里掌握 Blackwood 最关键的 AI 芯片授权和供应链风控系统。Adrian 直到 Ep4 才知道她是 Vesper；直到 Ep6 才知道她当年怀孕；直到 Ep8 才知道 Serena 的孩子不是他的；直到 Ep9 才完整被双胎遗物击穿。

## Episode Briefs

### Episode 1: The Heir At My Table

- 本集观众等什么：Ava 会不会说出自己怀孕和真实身份。
- 本集兑现什么：她被夺妻位，被迫签离婚和信托退出。
- 本集最后卡什么：Serena 私下安排车祸。
- 选中高压场面：Blackwood 家族晚宴长桌，Ava 的名牌被撤，Serena 坐上她的位置。
- 谁压谁：Adrian 和 Serena 压 Ava。
- 谁被迫选择：Ava 在公开怀孕和保护孩子之间选择沉默。
- 谁失去什么：Ava 失去妻位和家族信托身份。
- 起手动作：Ava 锁起 VESPER 终端，藏双胎报告，端餐入厅。
- 压迫升级：Adrian 带 Serena 入场，律师递离婚和信托退出文件。
- 反扑 / 反转：Ava 吞下报告，签字，不让他们拿孩子做筹码。
- 信息边界：不公开 Vesper；不让 Adrian 知道双胎。
- Last Frame：楼梯阴影里，Serena 拨通司机电话。

## Screenplay Contract

- 正文只写观众能看见、听见、AI 能生成的东西。
- 台词短，必须推动关系、权力、信息、选择或卡点。
- 高压场必须有动作、身体、空间、声音或当众后果。
- 不能靠多人站着解释；证据出现后必须马上改变现场权力。
- 每集结尾停在答案前一拍。
