# PRD v16-v6workflow Beat Sheet / Outline

## 1. Story Engine

- 主角想要什么：Serena 要让 Adrian、Chloe 和 Blackwell 亲眼看见他们五年前抹掉的人和孩子，并把是否公开毁掉 Adrian 的权力握回自己手里。
- 阻碍是什么：Adrian 的傲慢和家族体面、Chloe 的继承人母亲叙事、Blackwell 董事会和媒体场。
- 利害关系是什么：如果 Serena 只是赢融资或赢公司，源本会洗飞；她必须让“谁有资格审判谁”发生倒置。

## 4. Outline 判断

本 outline 承接 Source Retention Anchor：每个 beat 都改变了局势、关系、信息或情绪，不只是剧情梗概。E5 特别补入财务后果，E8 保持公开围观场，E10 压成更简洁的直播处刑权。
