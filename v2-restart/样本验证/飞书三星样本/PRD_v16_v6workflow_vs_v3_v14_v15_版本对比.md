# PRD v16-v6workflow vs V3 / V14 / V15 版本对比

对比对象：

- V3：`PRD_v3_豪门商战版_首批1-10集.md`
- V14：`PRD_v14_v3workflow_localfix_豪门商战版_首批1-10集.md`
- V15：`PRD_v15_v4workflow_豪门商战版_首批1-10集.md`
- V16：`PRD_v16_v6workflow_豪门商战版_首批1-10集.md`

## 1. 总结论

V16 是 v6 草案的主创侧受控验证稿。

它的价值不在于“比所有旧版每句都更好”，而在于 v6 要验证的链路成立：

```text
Source Bible
-> Source Retention Anchor
-> Story Operating State
-> Episode function map
-> Writer brief with Scene Sequence
-> Screenplay
-> Beat survival check
```

文本上，V16 明显优于 V3；相对 V15 是有针对性的局部进步；相对 V14 则吸收了 E8/E10 的强点，但 V14 的个别段落仍更有原始戏剧张力。

## 2. 与 V3 对比

| 维度 | V3 | V16 |
|---|---|---|
| 第 2 集冲击 | 安全档案库 / CO2 系统，责任清楚但偏机制 | 玻璃服务通道 / 火门 / 手铐 / 血手印，身体危机更强 |
| 源本外形替换 | 豪门壳成立，但有“系统锁仓”冷感 | 豪门发布会危机更可视化，仍保留二选一痛点 |
| E7-E8 爆点 | 信托律师 / 文件揭露，偏听证会 | 孩子喊 Daddy Ben 先破谎，Chloe 公开反咬 |
| E9-E10 火葬场 | 录音笔、文件盒、刑责陈述，情绪成立但证据感重 | 旧玻璃、双 wristbands、GO LIVE，物件更少更直接 |

判断：V16 比 V3 更像短剧成片，也更少“资本/法律机制说明”。V3 的好处是完整易懂，但冲击和画面都弱于 V16。

## 3. 与 V15 对比

| 维度 | V15 | V16 |
|---|---|---|
| workflow 链路 | v4 完整跑通，Pressure-chain brief 有效 | v6 补上 Scene Sequence 和 Beat Survival Check，writer 输入更清楚 |
| E5 动利益 | 主控判定“可更显性” | 直接加入 bridge loan frozen / donor wall under North audit，再触发 Chloe 闯房 |
| E8 公开围观 | V15 主控指出 private hallway 公开感弱于 V14 | V16 保持在 ribbon ceremony 原场，记者、董事、父母都在 |
| E10 简洁度 | V15 更公开但人物较拥挤 | V16 压回空 press room + phone + ring + wristbands |
| 情绪饱满度 | V15 更长，局部更铺陈 | V16 更短更薄，节奏更利于 AI 短剧，但可能少一点台词余味 |

判断：V16 对 V15 的主要进步是：E5/E8/E10 三个已知问题被显式修正，且 v6 的 writer-facing brief 不再只是压力链摘要，而是含 Scene Sequence。退步风险是文本更瘦，独立 reviewer 需要判断是否因此少了 V15 的部分情绪厚度。

## 4. 与 V14 对比

| 维度 | V14 | V16 |
|---|---|---|
| E8 | 很强，全场公开，Chloe 反咬有身体狼狈感 | 同样公开，动作更短，保留 “signed / saw / closed” 共犯钉子 |
| E10 | 空 press room 很干净，signet ring 和双 phone 好 | 吸收空 press room / GO LIVE / signet ring，但删掉 Marcus 和较多外部解释 |
| E9 | hospital box / archive cage 仍有证据便利性 | Adrian 主动撕开旧 corridor，再由 Lydia 要他承认 my wife / my children |
| workflow 证据 | 局部修稿，接线证据不足 | v6 manifest 证据链更完整 |

判断：V16 吸收了 V14 的 E8/E10 方向，但不应宣称 V14 文本价值作废。V14 的 E8 细节更狠、更有狼狈感；V16 更克制，适合本轮 v6 的“薄 brief -> 可拍正文”验证。
