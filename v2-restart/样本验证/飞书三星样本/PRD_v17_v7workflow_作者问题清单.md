# PRD v17-v7workflow 作者问题清单

## 1. 最担心洗飞的地方

E7-E8 的骗局公开崩塌保留了源本“假继承人 / 出轨 / 反咬男主”的功能，但新壳里用儿童翼剪彩和 Bennett/Noah 触发，情绪强度是否足够接近源本宴会厅公开羞辱，需要 reviewer 重点看。

## 2. 最担心像改名复刻的地方

E9 仍然保留“旧物 + 旧画面 + 男主跪地崩溃”的功能。虽然外形从仓库保险箱换成旧中庭排水槽、brooch 和碎超声影像，但功能位置很接近源本。需要判断这是必要的体验继承，还是仍然过像。

## 3. 最担心短剧降级或长剧化的地方

E4-E6 里有拍卖、债权、基金会账本、监控硬盘这些商战材料。它们已经尽量让人物动作和公开后果承载，但仍可能被 reviewer 判断为商业证据偏多。

## 4. 旧版或源本局部更强的地方

源本 E2 车祸火海的直接视觉冲击仍然很强。V17 换成玻璃中庭坍塌和灭火白雾，冲击比档案室强，但未必超过源本车祸。

源本 E10 暴雨跪求的粗暴情绪很直接。V17 换成直播发布会当众交权，公开代价更商战，但原始疯感可能比源本弱。

## 5. 需要 reviewer 特别看的地方

- V17 是否真的比 V15 / V16 更少程序感；
- 正文是否像剧本，而不是说明书；
- 源本的钩子、卡点、爽点周期、关系拉扯是否保住；
- E2 是否足够强；
- E7-E8 是否足够 dramatic；
- E10 是否是有效钩子，而不是又变成“按直播按钮”的机制动作。
