# S1 源本有效性包

输入：S0 最小需求确认 + 源本 txt。  
本文件只拆源本为什么有效，不设计新剧情。

## 源本首批追看核心

源本前 10 集的核心追问不是“黑手党会不会赢”，而是：

```text
被男主误判、替代、抛弃的女主，
会不会带着真正身份和不可逆伤害回来，
让男主亲眼知道自己选错、害死了最不该失去的人。
```

观众追看的燃料来自四层连续升级：

1. 站队：女主被白月光替代，男主不信她；
2. 恨意：男主在危险中选择白月光，导致不可逆伤害；
3. 期待：女主真实身份强，五年后能反杀；
4. 火葬场：男主知道真相太晚，只能用失权、下跪、求死换赎罪资格。

## 必须继承的体验功能

- 被替代：白月光不是普通情敌，而是拿走“被男主承认的位置”。
- 被误判：男主不只是变心，而是把女主判断成恶人。
- 错误选择：男主在关键危机里做了可见选择。
- 不可逆伤害：伤害不能靠道歉、金钱、一次救命抹平。
- 隐藏能力：女主不是突然开挂，而是原本就有被男主忽视的能力。
- 权力反转：女主归来后不是解释委屈，而是直接让男主失去资源。
- 连环拆谎：白月光的谎言不能一口气讲完，要层层爆。
- 罪责回刺：白月光反咬男主，使男主知道自己也是凶手。
- 赎罪悬置：男主可以崩溃求赎，但不能立刻获得原谅。

## 强度来源

| 类型 | 源本有效点 | 新本必须保住的功能 |
|---|---|---|
| 情绪密度 | 吞下委屈、求救被弃、失子、重逢冷脸 | 让观众先替女主憋屈，再看她冷着脸收债 |
| 信息密度 | 真孕、假孕、夜枭、买凶、非亲生、男主害死孩子 | 每集至少改变一项关系/权力/真相状态 |
| 情节密度 | 离婚、车祸、流产、拍卖、断供、宴会揭露、旧物崩溃 | 事件必须发生在现场，不能只递报告 |
| 人物拉扯 | 男主护白月光，女主不解释；归来后男主想靠近，女主拒绝 | 不能把男女主写成只交换信息的人 |
| 反派压力 | 白月光持续作死，男主偏护提供保护伞 | 反派要有资源、误判和下一步反扑燃料 |

## Do Not Consume

首批中不能提前吃完：

- 女主是否仍爱男主；
- 男主是否获得原谅；
- 女主所有隐藏资产和全部反击计划；
- 支撑者是否完全可靠；
- 白月光背后是否还有更大的资本/帮手；
- 男主后续能否真正用行动赎罪。

## 水分 / 烂点

- 后段反复下跪、挡枪、受伤求原谅，容易重复；
- 证据揭露如果只靠屏幕和报告，会冷；
- 白月光如果只哭喊否认，会单薄；
- 男主崩溃如果只在雨里喊错了，会像追妻套路像素。

- 黑手党、军火线、地下世界拍卖会；
