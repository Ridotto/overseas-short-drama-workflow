# PRD v16-v6workflow Source Retention Anchor

> 从 Source Bible 压出的留存锚点。后续 Story Operating State、Episode function map、Writer brief 和正文必须承接它。

## 1. 源本核心期待

```text
她被丈夫和白月光一起抹掉
-> 她求救时，丈夫选择别人
-> 她失去孩子和旧身份
-> 她五年后以更高权力回归
-> 她让白月光在公众面前一步步崩
-> 男主从震怒到怀疑，再到被自己当年的选择击穿
-> 男主发现自己亲手造成孩子死亡
-> 男主公开跪求，女主握住是否毁掉他 / 是否原谅他的权力
```

## 2. 首批留存闭环

| 段落 | 留存动作 |
|---|---|
| E1-E2 | 先把女主压到最低：妻子位置被抢、孕情被踩、男主二选一放弃她 |
| E3-E5 | 女主回归，先爽一次：房间被买下、白月光被打、利益线被切 |
| E6 | 真相裂缝进入男主，但不一次给完整答案 |
| E7-E8 | 白月光继承人叙事当众崩，男主被拖进共犯位置 |
| E9-E10 | 男主主动追回旧物并公开跪求，女主卡住最终处刑权 |

## 3. 关键钩子 / 卡点气口

- E1 尾：Chloe 在 Serena 离场后把 red folder 交给合规调查员，说明女主仍被追杀/甩锅。
- E2 尾：Serena 血手印滑下玻璃，孩子和生死都未交代。
- E3 尾：五年后黑衣女人入场，先给脚步和房间反应，不解释。
- E4 尾：Adrian 问旧伤，Serena 留下“那扇门”的暗刺，不解释孩子。
- E5 尾：Chloe 说漏 Lydia / south gate，证人线打开。
- E6 尾：Adrian 看到 Baby A，但被告知“还有第二只”。
- E7 尾：Noah 当众喊 Bennett Daddy，所有镜头转向 Adrian。
- E8 尾：Chloe 当众反咬“你也看见她流血”，Adrian 不能再只当被骗者。
- E9 尾：Adrian 带两只腕带跪到 press room 外。
- E10 尾：Serena 手指停在 GO LIVE，下一秒决定是否公开毁掉他。

## 4. 爽点蓄力与兑现周期

```text
先压 Serena：brooch 被扯、ultrasound 被踩、玻璃门后失子
再让 Serena 反压：买下房间、摘下 brooch、让 Chloe 爬着退场
再压 Adrian：旧伤 / Baby A / Baby B 一步步逼他面对
再兑现火葬场：Adrian 跪下亲口播出自己的罪
```

爽点不能无蓄力。每次打脸都要先让对方占住一个公开位置：

- Chloe 先站在发布会、公益剪彩、继承人母亲位置；
- Adrian 先站在 CEO、丈夫、父亲、审判者位置；
- Serena 再让他们从这些位置上掉下来。

## 5. 情绪密度走势

| 集数 | 密度 |

| E1 | 高压：替代 + 孕情被踩 |
| E2 | 极高压：二选一 + 失子 |
| E3 | 高痛转冷：失子 + 重生入口 |
| E4-E5 | 高爽：回归 + 反打 + 证人线 |
| E6 | 中高：旧伤刺穿，真相开口 |
| E7-E8 | 极高爆：继承人骗局 + 公开反咬 |
| E9-E10 | 极高火葬场：主动追旧物 + 公开跪求 |

## 6. 信息密度走势

- E1-E2：观众知道 Serena 怀孕，Adrian 不信。
- E3：观众知道孩子没了，Adrian 不知道。
- E4-E5：Adrian 知道 Serena 变强，但不知道伤口深处。
- E6：Adrian 知道至少有 Baby A，但还不知道完整双胎和责任链。
- E7：公众知道 Noah 不是 Adrian 的孩子。
- E8：公众和 Adrian 知道 Chloe 欺骗；Adrian 被提醒自己也有罪。
- E9：Adrian 追回 Baby B，完整罪感成立。
- E10：Adrian 公开说出事实，但 Serena 是否播出仍悬置。

## 7. 情节 / 权力变化走势

```text
Adrian / Chloe 压 Serena
-> Serena 被系统和空间封死
-> Serena 失去孩子但活下来
-> Aria North 回来买下 Blackwell 的脸面
-> Chloe 从体面继承人母亲变成公开骗子
-> Adrian 从 CEO / 父亲 / 审判者变成罪人
-> Serena 掌握公开处刑权
```

## 8. 人物关系拉扯主线

拉扯不是温柔复合，而是：

```text
Adrian 想控制 / 查问 / 弥补
-> Serena 身体先退，语言少给
-> 他每追近一步，就碰到自己当年的选择
-> 他从“你变了”被逼到“我害了你们”
-> Serena 不允许他的崩溃自动兑换原谅
```

## 9. 反派压力阶梯

Chloe 每次作死都要升级：

1. E1 用怀孕和继承人位置替代 Serena；
2. E1 尾把 red folder 递出去，继续甩锅；
3. E5 闯 Serena 地盘，想用旧权力吓退她；
4. E7 公开维护继承人母亲人设；
5. E8 先撒谎，再求饶，再拖 Adrian 下水。

## 10. 绝不能洗掉的高价值节点

- E2：男主二选一，身体损失和失子必须发生在他选择之后。
- E4：女主回归必须直接改变男主熟悉的权力房间。
- E7-E8：白月光骗局必须当众裂开，不能只靠文件。
- E9：男主必须主动追回旧物。
- E10：首批结尾必须让女主握住公开处刑权，不给原谅答案。
