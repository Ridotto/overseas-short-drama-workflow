# PRD v15-v4workflow Episode Plan

## Story Engine

- 主角想要什么：Serena 要让 Blackwell 和 Adrian 亲眼看见他们五年前抹掉的人和孩子。
- 阻碍是什么：Adrian 的傲慢、Chloe 的继承人叙事、Blackwell 家族和媒体场。
- 利害关系是什么：Serena 如果只是赢公司，源本会洗飞；她必须赢回“谁有资格审判谁”的权力。
