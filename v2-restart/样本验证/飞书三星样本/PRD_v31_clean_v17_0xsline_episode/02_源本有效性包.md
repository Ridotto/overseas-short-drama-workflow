# V31 源本有效性包

工位：S1 源本拆文  
输入：`01_最小需求确认.md` + 源本文本  
输出给下游：源本追看功能、高价值节点、高辨识外形、DNC

## 1. 一句话源本有效性

一个隐忍妻子被丈夫为了假白月光抛弃，并因此失去孩子；五年后她以更高权力归来，把假白月光、男主和整个旧权力系统逐层打爆，逼男主亲手确认自己杀死了亲生孩子。

源本前 1-10 集真正有效的不是黑帮外形，而是：

```text
强误判 + 不可逆伤害
-> 女主带着隐藏身份归来
-> 反派谎言逐层公开崩塌
-> 男主从掌权者跌成求原谅的人
```

## 2. 首批 1-10 集留存骨架

| 源本集数 | 功能 | 观众追看点 |
|---|---|---|
| 1 | 白月光假孕登堂入室，男主逼离婚，女主隐藏真孕 | 她为什么不说？她会怎么离开？ |
| 2 | 男主在事故中救白月光，弃女主，造成不可逆失子 | 他会不会知道自己做了什么？ |

| 5 | 女主切断男主资源；白月光主动作死被女主压制 | 女主到底掌握多少力量？ |
| 6 | 男主开始查事故，知道女主当时怀孕 | 他还差哪一块真相？ |
| 7 | 女主公开打爆白月光假孕和买凶 | 男主会不会继续护错人？ |
| 8 | 孩子非亲生与男主罪责同时砸下 | 男主如何面对自己也有罪？ |

| 10 | 男主公开跪求原谅，交出权力；女主用死亡/惩罚卡点 | 她会不会原谅？他会不会真的付代价？ |

## 3. 核心期待

首批的主追问：

```text
当年亲手抛弃她的人，什么时候亲手确认自己毁掉的不是“前妻”，而是自己的孩子、婚姻和权力？
```

阶段期待：

- E1-E2：观众站到女主一边，恨男主和假白月光。
- E3-E4：期待女主归来后第一次反压。
- E5-E6：期待男主开始查，却还没资格靠近真相。
- E7-E8：期待假白月光公开崩塌。
- E9-E10：期待男主亲手坠落，并被女主拒绝廉价赎罪。

## 4. 源本高价值节点

| 节点 | 源本体验功能 | 新本必须保留的体验 |
|---|---|---|
| 逼离婚 + 隐藏孕检 | 强羞辱、女主忍住关键真相、观众替她憋屈 | 公开剥夺女主身份，女主藏住更大的个人/身体真相 |
| 事故弃救 | 不可逆伤害，男主主动选择错人 | 男主在现场做出选择，且这个选择直接造成女主失子/创伤 |
| 五年后归来 | 权力倒置、身份反转 | 女主成为男主必须仰赖的行业高位者 |
| 第一次打脸 | 女主不解释，用权力和现场动作压人 | 女主让白月光在公共空间失去脸面和安全感 |
| 资源切断 | 女主不是嘴狠，是真能伤到男主系统 | 她能冻结合同、投票权、供应链或信任资源 |
| 公开揭谎 | 反派谎言不是私下解释，而是当众崩 | 白月光必须在她最想坐稳的位置上被拆穿 |
| 男主亲手发现 | 火葬场不靠别人骂醒，而靠男主亲手触碰证据/物件 | 男主必须打开旧门、触碰孩子相关物件，身体先崩 |
| 公开求原谅 | 男主权力坍塌，女主不立刻松动 | 他必须当众交出身份/权力/位置，而女主仍冷处理 |

## 5. 源本高辨识外形

禁止直接带入新本：

- 黑手党少主、教父、地下女王、军火市场、暗网军火。
- 车祸、出租车爆炸、火光吞噬车身。
- 黑帮拍卖会、军火线断供、纽约黑帮大佬。
- 枪抵额头、下跪求婚、码头枪战、狗笼、鳄鱼池。
- Lily 的假孕 + 情夫 + 非亲生孩子可以保留体验功能，但不能按原桥段原样复刻。

## 6. DNC

首批不能提前消费：

- E1-E2 不能让 Adrian 知道 Serena 真孕。
- E3-E6 不能让 Adrian 完整知道流产/失子真相。
- E7 只能打爆 Bianca 的伪造和买凶，不能让 Adrian 亲手完成火葬场。
- E8 可以让 Bianca 把“你也有罪”砸出来，但不能替 E9 完成男主亲手发现。
- E9 才能让 Adrian 触碰双胞胎相关物件和旧系统记录。
- E10 不能让 Serena 原谅，只能让第一轮权力坍塌成立。

## 7. 源本弱点

后续批次需要避免，但首批只做提醒：

- 男主反复下跪、挡枪、重伤、求留身边，功能重复。
- 外敌不断接力会稀释“核心情感复仇”。
- 后段容易变成女主守盘 + 外敌战，追看期待转弱。
- 过多 OS 解释心理，应改成动作、现场后果和关系变化。
