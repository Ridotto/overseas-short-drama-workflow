# 03 洗稿边界包

## 新壳

海外豪门商战：家族集团、信托继承、债权、董事会、慈善拍卖、媒体曝光、私人医疗、塔楼事故。

## 强继承体验

1. 婚内隐忍妻子其实有隐藏价值。
2. 旧爱以“怀了继承人”的身份登堂入室。
3. 女主怀孕信息被男主错过 / 不信。
4. 男主在即时危险里主动选择旧爱，放弃女主。
5. 事故导致孩子死亡，女主从爱转恨。
6. 五年后女主以高位身份公开归来。
7. 白月光的继承人叙事、事故阴谋、出轨 / 非亲生要分阶段塌房。
8. 男主必须被旧物和孩子真相打穿。
9. 第 10 集必须形成“献上一切求原谅，但女主不接”的高压卡点。

## 必须替换的高辨识外形

- 黑帮、军火、教父、枪战、黑道宴会。
- 卡车车祸 + 出租车爆炸。
- 地下黑帮审判。
- 直接复用 V5 的胸针 / 黑卡 / 封条等整套动作簇。

## 新壳真相修正

为避免“假孕”和“非亲生”自相矛盾，本轮统一为：

```text
Celeste 当年确实怀孕，但孩子不是 Adrian 的。
她伪造的是“Lancaster 继承人身份”、孕周口径和 Adrian 父亲身份。
```

所以 EP7 公开的是伪造继承人叙事、事故阴谋和出轨证据；EP8 才公开 DNA 非亲生。

## 防洗飞红线

- Adrian 不能只是“不知道她在危险里”；他必须听见她求救。
- 塔楼事故不能变成安全流程播报。
- Ava 归来不能只是“更有钱”；必须让 Lancaster 当场失去某个权力位置。
- Celeste 不能只是职场小人；她要有继续作死的资源和误判。
- Ava 在 EP10 不能软化。
