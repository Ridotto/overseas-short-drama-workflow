# 13 主控横向对比与拍板

## 结论

V27 判定：

```text
workflow execution complete through S11
creative outcome revise
delivery outcome not pass yet
```

这版不是废稿。它比 V24/V25 更干净地跑通了 v16 的交接链，也修掉了 V25/V27 初稿最大的问题：EP10 枪口黑屏和“听见怀孕仍离开”信息重复消费。

但它仍不能 pass。主要问题不是洗飞或复刻，而是中后段短剧强度还没有稳定打过 V5：EP7-EP8 证据播报感仍重，Celeste 抵抗不够，EP6/EP9 孩子真相分配还有提前消费风险，EP10 商战卡点比枪口干净但冲击力需要再抬。

## V27 vs V25

### V27 更好的地方

- EP10 从枪口 / 枪响换成直播认罪 + CEO / 信托投票权罢免确认，类型边界更贴豪门商战。
- Celeste 真相锁成“假 Lancaster 继承人”，不再混成“完全假孕 + 非亲生”的矛盾。
- EP8 不再完整吃掉 EP10 的公开认罪牌，信息释放比 V25 干净。
- run log 不再当质量证明，S10/S11 没有被提前宣告 pass。

### V27 仍不够的地方

- EP7-EP8 还是大量依赖大屏、文件、律师、DNA 结果，Celeste 的反扑动作不够。
- EP10 的商业卡点干净，但“Press it and find out”有软化误读。
- EP6 已经让 Adrian 知道“your child was inside me”，EP9 旧物击穿的新鲜度被削弱。

判断：V27 比 V25 更干净、更可持续，但 raw 短剧冲击未明显超过 V25。

## V27 vs V24 / V22

V27 明显强于 V24/V22 的地方：

- EP2 不再像档案 / 安全流程，男主亲手选择更清楚。
- EP5 有记者镜头、玻璃柜、手机丢香槟桶，场面比单纯财务报告更可拍。
- EP8 有孩子哭、董事逼验、签字笔掉落，人物反应比纯 DNA 屏幕强。
- EP10 不再是冷的 SEC 文件戏，而是公共直播认罪和董事罢免。

V27 仍像 V24/V22 的地方：

- EP7-EP8 证据链还是太直，缺少反派用资源拖住真相的过程。
- 商战动作有时还是“文件出现 -> 屏幕亮 -> 大家震惊”，不是每次都变成权力位置当场被夺。

判断：V27 的短剧感和现场感 > V24/V22；但还没完全摆脱“证据推动”的惯性。

## V27 vs V5

V5 的优势仍然存在：

- 动作后果更硬：胸针、黑卡、封条、枪口等物件记忆点非常强。
- 高压节点更少解释，很多地方是一拍打下去立刻有后果。
- 观众更容易记住“她被夺走什么 / 他失去什么”。

V27 比 V5 更好的地方：

- 类型边界更干净，EP10 不再用枪口审判。
- 事实口径更稳，Celeste 继承人骗局更清楚。
- 新壳更完整：债权、信托、董事、直播、CEO 罢免都进入故事。

V27 还没超过 V5 的地方：

- V5 的强物件感和即时剥夺感更强。
- V27 的商业动作更合理，但冲击力有些被合理性稀释。
- V27 中段真相推进更顺，但少了“反派抵抗 -> 反被更狠打脸”的拉扯。

当前排序：

```text
类型干净度：V27 > V25 > V24/V22 > V5
短剧原始冲击：V5 >= V25 ~= V27 > V24 > V22
综合候选价值：V27 > V25 > V24/V22
```

## v16 是否被验证

部分验证，不能正式化。

### 已验证的有效点

1. S0-S11 交接链比 V25 更清楚，writer 没有再直接吃散材料。
2. S5 writer 施工稿仍然有效，正文能从行动链长出来。
3. 干净 reviewer 能抓到硬阻断，而且第二轮能区分 blocked 和 revise。
4. v16 的 run log 降权是对的：它只记录执行，不替剧本背书。
5. S0b 和真相账本有用，至少修掉了假孕 / 非亲生口径混乱。

### 未验证通过的点

1. v16 仍没有稳定保证“证据不播报，必须变成反派抵抗下的人物行动”。
2. v16 仍没有稳定保证“每一层新信息只打一刀，不提前吃掉后面爆点”。
3. v16 对商业动作的强度要求还偏“有权力动作”，但不一定“有观众记得住的权力位置失去”。
4. writer 施工稿能让正文不像报告，但还不能自动让正文打过 V5。

## 当前不该做什么

- 不把 V27 当 pass。
- 不继续把 reviewer 变成质量发动机。
- 不因为 EP7-EP8 证据播报就再堆一堆“禁止屏幕 / 禁止文件”的规则。
- 不把 V5 当目标复刻。

## 下一步建议

如果继续推进，应该是一次“按层返修”，不是继续改 spec：

1. 回 S3/S5：重排 EP6/EP9 孩子真相。EP6 只给“她当年可能怀孕 / 身体伤疤”，EP9 才给 twins 和具体遗物。
2. 回 S5/S6：重写 EP7-EP8 的反派抵抗。Celeste 不应立刻完整认罪，应先用董事、媒体、孩子、信托资格反咬，Ava 再一层层打穿。
3. 回 S6/S7：改 EP10 台词，删掉 “Press it and find out” 的软化误读，明确“这是你欠公众的，不是买我原谅的”。
4. 回 S4/S5：给 EP4/EP5 加一个更硬的“权力位置失去”，例如 board observer seat、veto right、trust committee freeze，而不只是债权文件。

这四项是样本返修，不是 v16 spec 继续膨胀。
