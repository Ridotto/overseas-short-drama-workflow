# PRD_v22_latest_v11_Stage_Skeleton

## 阶段一：Ep1-Ep2 低位被夺 + 火场原罪

- 阶段起点：Ava 是 Blackwood 家族里被轻视的妻子；Adrian 和 Serena 拿“继承人”压她；观众知道 Ava 怀双胎且有隐藏商业身份。
- 阶段任务：
  - 压低 Ava：妻位、尊严、母亲身份都被踩。
  - 钉住 Adrian 的原罪：他在火场选择 Serena，听见 Ava 说怀孕仍离开。
  - 留下 Serena 的杀意和事故责任。
- 阶段变化：
  - 关系：婚姻从可挽回变不可逆。
  - 权力：Ava 被赶出家族信托；Serena 入主。
  - 信息：观众知道双胎，Adrian 不知道。
  - 情绪：从委屈压抑到血债成立。
- 阶段边界：
  - 不能让 Ava 当场暴露 Vesper。
  - 不能让 Adrian 产生明确愧疚。
  - 不能让 Serena 事故责任被公开。
- 阶段末交给下一阶段的追问：Ava 活下来后会怎样回来。

## 阶段二：Ep3-Ep5 重生起势 + 第一次公开压制

- 阶段起点：Ava 失去双胎，Julian 把她从病房带回商业世界；五年后她成为 Vesper Systems 的掌控者。
- 阶段任务：
  - 建立 Ava 新高位：她不是靠男人，是掌握 Blackwood 命脉。
  - 第一次公开打脸 Serena。
  - 第一次让 Adrian 的豪门权力工具失效。
- 阶段变化：
  - 关系：Adrian 从俯视 Ava 到必须追查她。
  - 权力：Blackwood 的芯片供应和专利授权被 Ava 卡住。
  - 信息：Adrian 知道 Ava 是 Vesper，但不知道孩子真相。
  - 情绪：观众从虐转爽。
- 阶段边界：
  - 不揭 Serena 假继承人。
  - 不让 Adrian 完整认错。
  - 不让 Julian 替 Ava 主导复仇。
- 阶段末交给下一阶段的追问：Adrian 会不会查到火场那晚真正发生了什么。

## 阶段三：Ep6-Ep8 真相逐层公开 + 假继承人坍塌

- 阶段起点：Adrian 追查 Ava，Ava 用伤疤和医疗线索逼他开始面对过去。
- 阶段任务：
  - Ep6 只打开“她当年怀孕”的门。
  - Ep7 公开 Serena 伪造孕周 / 收买医生 / 事故链，但保留继承人血缘大爆点到 Ep8。
  - Ep8 用信托继承仪式和生物认证失败，把“孩子不是 Adrian 的”变成可见公共后果。
- 阶段变化：
  - 关系：Adrian 与 Serena 同盟断裂。
  - 权力：Serena 失去继承人母亲护身符。
  - 信息：Adrian 从怀疑变成被迫接受自己错信。
  - 情绪：Ava 的冷笑把男主钉进同罪。
- 阶段边界：
  - Ep7 不能提前给出完整亲子结论。
  - Ep8 不能直接让 Adrian 跪求 Ava。
  - 不能用大屏和报告作为唯一高潮；必须有徽章、信托门、董事席等现场后果。
- 阶段末交给下一阶段的追问：Adrian 怎么面对“我亲手害死自己的双胞胎”。

## 阶段四：Ep9-Ep10 私人物证击穿 + 公开赎罪启动

- 阶段起点：Serena 败露后反咬 Adrian；Adrian 开始寻找 Ava 当年留下的私人证据。
- 阶段任务：
  - Ep9 用双胎报告、未完成的婴儿毯、旧戒指击穿 Adrian。
  - Ep10 让 Adrian 当众交出 Blackwood 控制权和创始人徽章，进入追妻火葬场。
  - Ava 给出残酷条件，但不释放原谅。
- 阶段变化：
  - 关系：Adrian 从高位施害者变成跪地求赎者。
  - 权力：Blackwood 核心控制权转向 Ava。
  - 信息：Adrian 完整知道双胎和火场罪责。
  - 情绪：观众得到迟来崩溃，但继续追 Ava 是否会收下他。
- 阶段边界：
  - 不让 Ava 心软。
  - 不让 Adrian 的自毁被包装成已经偿还。
  - 不提前进入中后段挡枪 / 赎命。
- 阶段末交给下一阶段的追问：Ava 会不会让他活着赎罪，Serena 下一步会不会被清算。
