## 1. Rewrite Report

| Layer | 判断 | 说明 |

| concept | pass | V8 没走黑帮/车祸/亲子鉴定/保险箱旧物，改用资本犯罪、公开背锅、拘押拒医、继承信托、联邦证物箱和公开认罪。 |
| structure | pass | 1-10 集覆盖“被替代 -> 错误选择 -> 不可逆损失 -> 女主高位回归 -> 骗局裂开 -> 男主同罪 -> 火葬场 -> 公开求赎罪”的首批闭环。 |
| scene | pass with risk | 高压场面基本有动作承载：徽章被扯、签字、手铐、信托扫描、CEO key、托管箱、直播认罪。风险是第 7-8 集商业机制偏密，需要正文用动作和屏幕反馈压住。 |
| dialogue | pass with minor risk | 台词整体短，功能明确；少量商战信息需要避免解释腔，已用屏幕、签字、席位灯、拘押音频替代长解释。 |
| continuity | pass | 第 8 集只给托管箱编号，不提前消费第 9 集旧物；第 10 集只给“听见了”，不给原谅。 |
| adaptation | pass | 源本有效功能迁移，关键高辨识外形已扣除；不是 V3/V7 的改名复刻路线。 |

### Root Symptoms Checked

1. **第 2 集容易降成法务文件戏**  
   已处理为“发布会股价崩盘 -> 调查员进场 -> Adrian 签 referral -> Serena 戴手铐流血 -> 拘押车拒医 -> 监护仪长音”，文件出现后立刻触发人物命运变化。

2. **第 7-8 集容易变成证据听证会**  
   已处理为信托台和 CEO key 的动作链：Chloe 必须按扫描台，Adrian 必须亲手按担保键和插入 CEO key。证据不是播放给人看，而是逼人物承担责任。

3. **男主罪责不能只靠 Chloe 自白**  
   已处理为 Adrian 自己的签名、旧语音、CEO key 和托管箱证物。Chloe 的漏嘴只作为辅助，不是定罪核心。

4. **第 10 集不能变成诱导自杀或过度惩罚女主失当**  
   已处理为公开认罪、CEO 辞任、投票权移交和联邦调查，等效“社会性死亡”，不做自杀测试。

### Prioritized Actions

1. 保留第 2 集的身体后果：手铐、流血、拘押车、监护仪长音不能删。
2. 保留第 7 集的信托台动作：Chloe 不敢按、系统拒绝、Adrian 按担保键，是本版去同构的核心。
3. 保留第 8 集的 CEO key 和旧语音：这是男主同罪的硬证据。
4. 保留第 10 集的直播红灯和联邦探员：这是豪门商战版的“跪求火葬场”替代载体。

### Classification

| 项 | 结论 |
|---|---|
| must-restructure | 无 |
| suggested | 后续如导演嫌商战机制复杂，可在第 7 集减少一个术语，但不能删“信托台动作”。 |
| observable-risk | 第 3 集从病房到五年后跳转较快，但符合短剧节奏；后续可用 11 集以后回补 Marcus 与 Aria North 成形过程。 |

### Subtext Benchmarks

- Serena 的台词不解释“我很痛苦”，只用冷判、要求动作、撤销身份来表达痛苦。
- Adrian 前 1-8 集不能提前忏悔，必须先傲慢、怀疑、被证据击穿。
- Chloe 的台词保持柔弱转尖锐，不能太聪明到压过 Serena；她的力量来自伪装和 Adrian 的保护。
- Marcus 的台词少而锋利，主要制造“他不是纯好人”的后续空间。

### Voice Adjustments

| 角色 | 声音标准 | 已处理点 |
|---|---|---|
| Serena | 短、冷、审判式；不求解释 | “我不洗”“让他看”“听见了。但我没说原谅。” |
| Adrian | 前期 CEO 命令式；后期句子碎裂 | 前期“签 referral”；后期“是我签的” |
| Chloe | 表面柔弱，危机时尖叫泄底 | 第 8 集漏嘴只辅助，不承担全部罪责 |
| Marcus | 利益感、克制、占有欲潜伏 | “有些门，不是现在后悔就能打开的。” |

### Rhythm Notes

1. 每集结尾停在动作、屏幕、声音或身份坍塌上，不停在解释。
2. 商战信息用可见载体表达：股价、担保键、席位灯、CEO key、直播红灯、联邦探员。
3. 火葬场台词不铺长段忏悔，第 9 集用胎心音和监护仪长音承担主要情绪。
