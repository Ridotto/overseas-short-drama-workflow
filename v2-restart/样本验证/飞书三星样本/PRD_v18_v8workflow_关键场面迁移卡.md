# PRD V18 v8workflow 关键场面迁移卡

## KC1 E1 正妻位置被继承叙事取代

源本节点：Dante 带怀孕 Lily 回庄园，要求 Eve 签离婚。

源本体验功能：女主从期待告知怀孕，瞬间被男主和女配共同夺走妻子位置。

源本强度来源：孕检单在女主身上，另一女人以怀孕身份进门；男主一巴掌确认偏信。

源本禁止外形：黑帮庄园少主夫人、离婚协议拍台面、女配假摔后男主甩耳光原样复用。

新场景目的：让 Ava 在 Whitmore 庄园准备告诉 Adrian 自己怀孕，却被 Celeste 的“继承人怀孕发布”挤出婚姻和配偶信托。

新壳故事发动机如何支撑：Whitmore 董事会需要继承人和稳定股价；Celeste 的伪孕能让 Adrian 得到董事会支持，Ava 的平凡妻子身份反而成了障碍。

新冲突动作链：
1. 起手动作：Ava 锁起 Kestrel 风控模型，摸到孕检单，准备给 Adrian 惊喜。
2. 压迫升级：Adrian 带 Celeste 和家族律师进门，递出离婚与配偶信托放弃书。
3. 反击 / 反转 / 选择：Celeste 故意撞向水晶台阶并抓住 Ava 手腕，制造 Ava 推她的假象；Adrian 当众剥夺 Ava 的庄园通行卡。
4. 不可逆后果：Ava 吞下孕检单，签字离开；Celeste 拿到庄园主卧钥匙。

权力变化：Ava 从合法妻子被赶出 Whitmore 家庭与信托边界。

关系变化：Adrian 从丈夫变为审判者；Celeste 从旧爱变为准女主人。

信息差变化：观众知道 Ava 怀孕，Adrian 不知道；Celeste 不知道 Ava 是 Kestrel。

情绪变化：Ava 从期待到死心；Celeste 从柔弱到阴狠。

观众直接看到 / 听到什么：孕检单、信托放弃书、被注销的门禁卡、Celeste 手里的主卧钥匙、Ava 咽下纸团。

证据或文件触发行动：律师文件一出现，安保立即收回 Ava 门禁和车钥匙，不只是桌上摆件。

台词功能上限：逼签、羞辱、防守、冷拒。

卡点停在什么动作、反应、选择或后果前：Celeste 在楼上打电话给车队调度，说“make sure she never reaches the airport.”

禁止提前消费：不提前揭 Celeste 伪孕；不让 Adrian 知道 Ava 怀孕。

结论：可进入写作简报。

## KC2 E2 事故现场弃救选择

源本体验功能：男主做出不可逆选择，把女主和亲生孩子留给死亡。

新壳故事发动机如何支撑：Whitmore 的安保和救援资源听 CEO 指令；家族公关不能让“怀孕的 Celeste”出事。

新冲突动作链：
1. 起手动作：Ava 的离场车被调入施工封闭道，货车侧撞；Celeste 车在近处轻撞但尖叫。
2. 压迫升级：安保报告 Ava 车门卡死、电池包冒烟；Celeste 被镜头和家族医生围住。
3. 反击 / 反转 / 选择：Ava 拍窗喊自己怀孕；Adrian 以为她故意制造事故害 Celeste，命令救援队先转移 Celeste。
4. 不可逆后果：泡沫车和切割工具离开 Ava，电池起火，Ava 在火光里失去意识。

权力变化：Adrian 的 CEO 指令变成害死孩子的直接动作。

关系变化：Ava 对 Adrian 最后一丝求救信任断裂。

信息差变化：Adrian 仍不知道真孕；观众看见 Celeste 在担架上露出得意。

情绪变化：Ava 从求救到绝望；Adrian 从怒到冷。

观众直接看到 / 听到什么：车门变形、红色电池警报、切割工具被抬走、Ava 掌心血印拍在车窗上。

证据或文件触发行动：救援调度记录和车队 GPS 将成为 E7 证据。

台词功能上限：求救、命令、误判、拒绝。

卡点停在什么动作、反应、选择或后果前：Ava 车内安全气囊被火光照亮，她手里的婚戒落进血里。

禁止提前消费：不写 Ava 被 Julian 救出的过程。

结论：可进入写作简报。

源本禁止外形：黑医病房、黑帮首领合作复仇原样。

新冲突动作链：

2. 压迫升级：屏幕播放 Adrian 抱着 Celeste 接受媒体祝福，基金会以未出生继承人命名。
3. 反击 / 反转 / 选择：Julian 放下她旧代码和集团漏洞报告，指出她不该死在 Whitmore 后座。

情绪变化：空、痛、冷、决绝。

观众直接看到 / 听到什么：空荡病床、双胎死亡记录、电视里的继承人基金发布会、Ava 按下删除键。
