# PRD v17-v7workflow 主创读本笔记

## 1. 一句话故事核

源本是一个“被丈夫误判和替代的隐忍妻子，在不可逆失子后以隐藏权力身份归来，反向碾压旧爱骗局和前夫帝国，让前夫从暴君变成赎罪疯狗”的追妻火葬场复仇短剧。

## 2. 观众追看的核心期待

观众追的是四条东西：

- 女主什么时候不再忍，什么时候反杀；
- 男主什么时候知道自己亲手害死了自己的孩子；
- 白月光骗局什么时候当众崩；
- 男主知道真相后会崩到什么程度，女主会不会原谅。

这不是单纯复仇，也不是单纯商战。源本最强的是“误判造成不可逆伤害 -> 真相迟到 -> 权力反转 -> 赎罪被拒”的情绪链。

## 4. 高价值节点

### A. E1 替代与羞辱

客观事件：丈夫带着“怀孕白月光”回到女主所在的家，当众逼她签离婚。

强度来源：不是出轨本身，而是女主同时怀孕、同时曾为男主付出、同时被男主否定为“无用妻子”。白月光的柔弱表演让男主的误判更刺痛。

必须迁移：公开替代、身份剥夺、女主吞下/毁掉真相、临走前留下反杀气口。

禁止照搬外形：黑手党庄园、孕检单吞咽、原话式“庄园女主人”。

### B. E2 二选一与不可逆失子

客观事件：女主和白月光同时陷入危险，男主选择救白月光并抛下女主，女主流产。

强度来源：可见危险 + 男主冷酷选择 + 女主喊出真相没人信 + 男主离开的背影 + 爆炸/血/失子。

必须迁移：强视觉危机、男主错误选择、女主身体承受不可逆后果。

禁止照搬外形：出租车、卡车车祸、火海救人。

### C. E3 失子到复仇

客观事件：女主醒来确认孩子没了，同时看到男主为白月光庆祝。

强度来源：私人丧失和公众庆祝正面撞在一起。女主的复仇不是口号，是从身体失去开始。

必须迁移：无心跳 / 空小腹 / 外界庆祝白月光继承人 / 女主作出不可逆选择。

### D. E4 五年后回归

客观事件：女主以更高权力身份重返男主权力场，所有人改变态度。

强度来源：观众知道她是谁，场内人还不知道；白月光先踩她，马上被反压。

必须迁移：回归卡点、身份反转、公开打脸、男主从轻视到震动。

### E. E7-E8 骗局公开崩塌

客观事件：白月光假孕、出轨和买凶被公开拆穿，男主发现自己被绿。

强度来源：男主的面子、血脉、权力和判断力同时崩。白月光被逼到反咬男主：她不是唯一凶手，男主也是共犯。

必须迁移：公开围观、反派失控、男主被羞辱、男主罪责被点中。

### F. E9-E10 迟到真相和跪求

客观事件：男主找到女主当年怀孕与失子的证据，崩溃后当众跪求原谅。

强度来源：不是“拿到文件”，而是男主终于看见自己亲手做错的那一拍。下跪要有公开代价和权力坍塌。

必须迁移：旧物 / 旧画面 / 身体崩溃 / 公开交出权力 / 女主冷酷卡点。

## 5. 关系拉扯

首批里男女主不是普通对手，而是“她越强，他越晚知道自己错；他越靠近，她越把他推回罪人位置”。

拉扯不是甜，也不是反复聊天。它应表现为：

- Adrian 还想用旧丈夫身份命令她；
- 女主每次都让他看见自己失去的权力；
- 他越想解释，越发现自己没有解释资格；
- 女主的冷不是没情绪，而是她把情绪变成惩罚和控制。

## 6. 源本水分和不能继承

- 不能整串照搬：假孕入室、车祸、五年归来拍卖会、亲子鉴定、旧仓库孕检单、暴雨下跪。
- 不能继承低效多人对白。
- 不能把强危机换成文件档案。
- 不能把男主崩溃写成“他看报告很震惊”。
