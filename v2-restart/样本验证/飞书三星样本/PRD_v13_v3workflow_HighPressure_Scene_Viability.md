# PRD v13-v3workflow High-pressure Scene Viability

> 目标：在正文前先验证新壳是否能承载强戏。  
> 这不是多加规则，而是防止场面发动机选错后，正文层反复加料。

## 1. 节点 A：开局替代与离婚

源本体验：白月光怀孕登堂，男主当众逼离婚，女主隐藏自己的孕检单。

新壳场面发动机：

- 地点：Blackwell Global 发布会后台，家族董事、律师、安保都在。
- 直接冲突：Serena 想告诉 Adrian 自己怀孕且刚救下集团；Adrian 要她签离婚和责任承担。
- 人际压力：Chloe 站在 Adrian 身边，拿走 Serena 的家族位置和救市功劳。
- 可见后果：Adrian 扯下 Serena 的家族徽章，Chloe 踩脏双胎超声。
- 卡点：Serena 被推出后台时，看见 Chloe 准备把她名字送进罪案。

Viability：pass。强度来自丈夫和新欢当面抹掉她，不靠文件本身。

## 2. 节点 B：不可逆损失

源本体验：两车遇险，Dante 先救 Lily，Eve 隔窗求救，爆炸吞没她。

新壳场面发动机：

- 地点：发布会主厅 / 玻璃服务通道 / 混乱出口。
- 直接冲突：Serena 戴着手铐和出血身体求 Adrian 开门；Chloe 在另一侧装作受伤并抓住他。
- 人际选择：Adrian 当着记者、调查员、董事，命令安保先带 Chloe 离开，并亲手把 Serena 推回安保控制区。
- 可见后果：Serena 被人群撞倒，手铐勒进手腕，血落在白色地面；玻璃门在她面前合上。
- 不允许：不能写成“系统拒医键”。门、安保、调查员只是压力环境，真正伤人的是 Adrian 的选择。
- 卡点：Serena 隔着玻璃看 Adrian 抱 Chloe 离开，血从裙摆滴下。

Viability：pass。能保住车祸级“亲手放弃”的体验，但不复刻车祸外形。

## 3. 节点 C：五年后强回归

源本体验：高跟鞋入场，全场安静，观众想看她是谁。

新壳场面发动机：

- 地点：Blackwell 慈善拍卖 / 董事晚宴。
- 直接冲突：Chloe 以家族女主人姿态主持；Adrian 正在出售 / 认购关键资产。
- 人际压力：众人先认不出 Serena，只看见一位新资本控制人入场。
- 可见后果：拍卖槌停住，董事起立，Chloe 的笑僵住。
- 卡点：Episode 3 尾只让她入场，不摘墨镜、不解释身份。

Viability：pass。采用导演认可的“回归不急着说话”思路。

## 4. 节点 D：第 7-8 集撕开骗局

源本体验：白月光骗局被公开撕开，男主绿帽尊严坍塌，反派把男主罪责反咬出来。

新壳场面发动机：

- 地点：慈善医院剪彩 / 家族基金晚宴。
- 直接冲突：Chloe 带孩子和董事压 Serena；Serena 逼 Chloe 在众人面前维持“Blackwell 继承人”叙事。
- 触发真相：孩子意外受伤需要输血，Chloe 阻止 Adrian 上前；孩子本能喊真正父亲 Bennett。
- 人际压力：不是一份亲子报告，而是孩子、情人、母亲、名义父亲同时在场崩掉。
- 可见后果：Chloe 跪抱 Adrian 的腿求饶；Adrian 把她踹开；她崩溃喊出“你也杀了 Serena 的孩子”。

Viability：pass。真相由身体危机和当众失态触发，证据只可辅助。

## 5. 节点 E：男主追悔与首批终钩

源本体验：男主看见旧物崩溃，公开跪求原谅，女主用枪口 / 生死题卡住。

新壳场面发动机：

- 地点：五年前发布会后台 / 私人医院遗物室 / 直播发布会。
- 直接冲突：Adrian 被迫走回当年自己放弃 Serena 的地方；Serena 要他公开承认“不是误会，是我亲手选择”。
- 可见后果：Adrian 看见双胎手环和被踩脏超声；他在直播镜头前跪下，不再保留 CEO 体面。
- 卡点：Serena 的手停在直播按钮上，Adrian 已经跪下准备认罪。

Viability：pass。保留第十集强拉扯，不用枪或跳楼。
