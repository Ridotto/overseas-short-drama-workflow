# PRD_v24_refactor_v12_Source Bible

## 一句话故事核

一个被丈夫和白月光联手伤到失去孩子的隐忍前妻，五年后以更高权力身份归来，公开击碎白月光的骗局，逼曾经不可一世的丈夫在真相和悔恨里崩塌。

## 核心期待

- Eve 会不会从“被抛弃的妻子”变成“掌控一切的女王”。
- Dante 什么时候知道自己亲手害死了自己的孩子。
- Lily 的假孕、车祸和私生子骗局什么时候被公开打穿。
- Dante 求原谅时，Eve 会不会杀他、原谅他，还是继续折磨他。

## 首批分集功能

- 1：婚内替换羞辱 + 隐藏孕情 + Lily 杀机。
- 2：致命抛弃，Dante 选择 Lily，Eve 流产创伤成立。
- 3：Eve 失子后复仇动机锁死，五年后女王回归悬念。
- 4：身份碾压，Eve 回归压场并夺走 Dante 的生意命脉。
- 5：Dante 开始调查，Eve 对 Lily 第一次实体反击。
- 6：Eve 用伤疤和“孩子”刺穿 Dante 的认知，Dante 找到怀孕证据。
- 7：公众场合揭开 Lily 假孕和谋害链条，Dante 仍因颜面护短。
- 8：亲子真相打穿 Dante，Lily 反咬出 Dante 罪责。
- 9：Dante 找到孕检与婴儿物件，火葬场正式启动。
- 10：Dante 当众跪求，用帝国和性命换原谅，Eve 设最强卡点。

## 高价值节点

### HV1：白月光怀孕登堂入室

- 客观事件：Dante 带怀孕的 Lily 回庄园，要求 Eve 签离婚。
- 观众体验功能：替女主委屈，建立反派可恨和男主瞎眼。
- 强度来源：婚姻位置被当面替换；Eve 也怀孕但不能说。
- 钩子类型：情绪钩 + 信息钩。
- 爽点类型：后续打脸复仇的压抑蓄力。
- 压抑形成：Dante 认定 Eve 无用、善妒，还当场动手。
- 释放预期：Eve 将来以真实身份和孩子真相回击。
- 关系 / 权力变化：Eve 从妻子被降成弃妇；Lily 从外人坐上女主人位置。
- 信息差状态：观众知道 Eve 有孕和隐藏能力，Dante 不知道。
- 卡点停法：Lily 从柔弱切成恶毒，露出杀机。
- 源本高辨识外形：黑帮庄园、孕检单吞下、离婚协议、白月光大肚子。
- 下游写作抓手：新壳必须当众替换婚姻/家族继承位置，并让女主隐藏孕情可视化。

### HV2：事故中男主二选一

- 客观事件：Lily 设计车祸，两车被困，Dante 先救 Lily，留 Eve 在爆炸里。
- 观众体验功能：不可原谅的原罪成立。
- 强度来源：Eve 求救且明确说怀孕，Dante 仍选择不信。
- 钩子类型：危机钩。
- 爽点类型：后续火葬场所需的极端压抑。
- 压抑形成：生命、孩子和爱情三重被抛弃。
- 释放预期：Dante 将来知道真相后崩溃。
- 关系 / 权力变化：婚姻彻底死亡；Eve 的爱转成复仇。
- 信息差状态：Dante 误信 Lily，观众知道他犯下不可逆错误。
- 卡点停法：爆炸吞没 Eve。
- 源本高辨识外形：车祸、出租车、火海爆炸。
- 下游写作抓手：新壳必须用不同事故载体保留“他亲手选择救错人”的身体事实。

### HV3：五年后身份回归压场

- 客观事件：Eve 以 Night Owl 身份出现在黑帮拍卖会，夺走生意并压制 Dante。
- 观众体验功能：身份碾压、逆袭爽感、期待她如何清算。
- 强度来源：过去被轻视的人现在掌握对方命脉。
- 钩子类型：身份反转钩。
- 爽点类型：身份碾压 + 打脸复仇。
- 压抑形成：前 1-3 集的弃妇位置。
- 释放发生：全场震惊，Dante 第一次失控。
- 关系 / 权力变化：Eve 从被赶走的人变成资源上游。
- 信息差状态：Dante 不知道她为何恨到这种程度。
- 卡点停法：Eve 逼 Dante 认出前妻身份。
- 源本高辨识外形：黑帮拍卖会、军火线、枪口压下。
- 下游写作抓手：新壳用资本、债权、供应链或投票权实现同等权力翻转。

### HV4：公开揭穿 Lily 的骗局

- 客观事件：Eve 押来黑医，放证据，揭开 Lily 假孕、买凶、出轨和孩子非亲生。
- 观众体验功能：长压抑后的公开打脸。
- 强度来源：全场见证，Dante 的权威同时被羞辱。
- 钩子类型：信息钩 + 反转钩。
- 爽点类型：群众审判、证据反杀。
- 压抑形成：Lily 前期借假孕和偏爱作恶。
- 释放发生：证据逐层砸下，Lily 从女主人变成骗子。
- 关系 / 权力变化：Dante 与 Lily 的同盟裂开，但 Dante 的罪更重。
- 信息差状态：Dante 得到 Lily 有罪证据，但尚未完全承认自己害死孩子。
- 卡点停法：Lily 反咬 Dante 也有罪。
- 源本高辨识外形：黑医、宴会大屏、床照、亲子鉴定。
- 下游写作抓手：新壳保留公开审判和链式证据，但换成董事会 / 信托 / 安全事故证据。

### HV5：物件火葬场

- 客观事件：Dante 找到孕检单、婴儿毛衣和尾戒，确认自己害死亲骨肉。
- 观众体验功能：让男主崩溃从“知道”变成“看见”。
- 强度来源：抽象罪责落到孩子物件。
- 钩子类型：情绪钩。
- 爽点类型：迟来悔恨的反向爽。
- 压抑形成：Dante 长期瞎眼和强权。
- 释放发生：他跪地自扇、雨中痛哭。
- 关系 / 权力变化：男主从审判者坠落成求赎者。
- 信息差状态：Eve 不在场，观众先看到他的火葬场。
- 卡点停法：他决定当众求原谅。
- 源本高辨识外形：保险箱、孕检单、婴儿毛衣、暴雨跪地。
- 下游写作抓手：新壳需要保留“孩子物件 + 旧空间 + 自毁式悔恨”，但不能照搬保险箱和毛衣。

### HV6：当众跪求原谅

- 客观事件：Dante 闯入 Eve 与 Victor 的庆祝场合，交出家族资产，下跪求原谅，Eve 用枪逼他以命换命。
- 观众体验功能：不可一世男主彻底坠落，形成首个强付费卡点。
- 强度来源：公众见证 + 帝国转让 + 性命选择。
- 钩子类型：付费级生死/选择钩。
- 爽点类型：权力逆转、情感爆发中断。
- 压抑形成：Eve 被抛弃、失子、五年复仇。
- 释放发生：Dante 主动放弃一切。
- 关系 / 权力变化：Dante 从掌权者变成求饶者，Eve 掌握他的命。
- 信息差状态：观众不知道 Eve 会不会真让他死。
- 卡点停法：枪响黑屏。
- 源本高辨识外形：黑帮餐厅、枪、家族地契、跪地。
- 下游写作抓手：新壳必须把“杀死自己”改成商战语境的自毁选择，而不只是签几份文件。

## 源本事实冲突 / 不下传项

- 源本前期多说“孩子”，第九集变为“双胞胎”。新本可直接锁定为“双胞胎”，避免后续漂移。
- Lily 的“假孕”与五年后有孩子可解释为：她假称怀的是 Dante 的孩子，实际孩子来自他人；新本必须说清。
- Eve 的能力从武器设计、黑客到军火巨头跨度大，新本需提前给她的商业能力一个稳定来源。
- 不复用黑帮、军火、枪口、车祸、保险箱、黑医拖进宴会这些高辨识外形。
