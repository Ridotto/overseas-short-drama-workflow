# PRD_v21_v11workflow_Source_Bible

## 1. 一句话故事核

源本是“被丈夫为白月光抛弃并失去孩子的隐忍妻子，五年后以更高权力身份回归，夺走前夫的权势、撕开白月光骗局，并把前夫逼到崩溃追妻火葬场”的短剧。

## 2. 核心期待

观众追的不是单一复仇，而是三条期待并行：

- 女主从低位到高位，什么时候当众碾压所有轻视她的人。
- 男主什么时候知道自己为了假孩子害死了真孩子，并从高位暴君坠落到跪地求原谅。
- 女配骗局什么时候被撕开，撕开之后会不会反咬出更重的男主罪责。

## 4. 高价值节点

### HV1：登堂替换

- 客观事件：男主带怀孕白月光回家，要求女主签离婚。
- 观众体验功能：迅速制造委屈、愤怒和代入，让观众等女主反杀。
- 强度来源：婚姻三年被一句“你什么都不会”抹掉；白月光装柔弱反咬。
- 关系 / 权力变化：女主从合法妻子被踢到无名低位；女配获得男主公开保护。
- 信息差状态：观众知道女主也怀孕、且有隐藏能力；男主不知道。
- 卡点停法：女配在女主离开后露出真面目，要继续害她。
- 源本高辨识外形：厨房、离婚协议、孕检单吞下、白月光摔倒、耳光。
- 下游写作抓手：换成豪门家族晚宴/董事继承餐，离婚从私人协议升级为家族和媒体面前的公开替换。

### HV2：生死选择与不可逆伤害

- 客观事件：女配制造事故，男主救女配不救女主，女主流产。
- 观众体验功能：把“渣”从感情错误升级成无法被轻易原谅的生命债。
- 强度来源：女主亲口求救和怀孕信息被男主当成骗局。
- 关系 / 权力变化：男主的选择彻底杀死旧关系；女主黑化有了正当性。
- 信息差状态：男主仍以为自己保护的是亲生孩子。
- 卡点停法：爆炸/火光吞没女主。
- 源本高辨识外形：车祸、两辆车、救 Lily、弃 Eve。
- 下游写作抓手：换成豪门庄园 helipad/救援通道火灾，避免车祸外形复刻；保留“只能救一边、他选错”的体验。

### HV3：五年后身份反转回归

- 客观事件：女主以“夜枭”身份回归，夺走交易和权力。
- 观众体验功能：从憋屈释放为爽感；让男主第一次低头。
- 强度来源：不是单靠外貌，而是她掌握了男主赖以生存的供应/权力系统。
- 关系 / 权力变化：女主从被赶走者变成能让男主家族断供的人。
- 信息差状态：男主先认不出她，再被迫承认她是高位者。
- 卡点停法：女主把枪口/威胁压在男主面前，问他敢不敢动她。
- 源本高辨识外形：黑帮拍卖、粉钻、军火线、枪口。
- 下游写作抓手：换成奢侈集团债权拍卖/董事会融资席，女主以 Nightingale Capital 掌控债务、码头和供应链。

### HV4：女配第一次被清算

- 客观事件：女配带人砸女主场子，被女主反制羞辱。
- 观众体验功能：兑现对女配的第一波爽点，但不能一次结案。
- 强度来源：女配仍仗着男主和家族名号作死，女主让她看到旧账才刚开始。
- 关系 / 权力变化：女配从狐假虎威者变成恐惧者。
- 信息差状态：女配知道自己制造事故，男主还不知道完整真相。
- 卡点停法：女主宣布“五年前的账才刚开始”。
- 源本高辨识外形：酒店套房、保镖、踩手指。
- 下游写作抓手：换成顶层套房/品牌发布会后台，女配带 PR 和安保闯入，女主用酒店安防、股权和身体压制让她当众失控。

### HV5：伤疤和怀孕线索

- 客观事件：男主看到女主身上的事故伤疤，开始查五年前医疗记录。
- 观众体验功能：火葬场前置，让男主进入不安和否认。
- 强度来源：身体痕迹比口头控诉更有效。
- 关系 / 权力变化：男主第一次被女主的痛压住。
- 信息差状态：男主知道“怀孕”，但还没知道真假孩子、双胞胎/旧物等完整打击。
- 卡点停法：他拿到残缺医疗记录，意识到她当时怀过他的孩子。
- 源本高辨识外形：地下车库、手腕伤疤、医疗报告。
- 下游写作抓手：换成董事停车层/私人医疗档案馆；伤疤不靠大段解释，要在灯光、衣袖和男主手停住里显出来。

### HV6：公共撕假孕与假继承人

- 客观事件：女主当众拆穿女配假孕、出轨、孩子非男主亲生。
- 观众体验功能：强打脸、权力羞辱和男主第一层崩塌。
- 强度来源：全圈层见证男主被戴绿帽、养错继承人。
- 关系 / 权力变化：女配保护伞裂开；男主威严崩。
- 信息差状态：不能把女主流产真相完全一次说透；Ep9 才靠旧物击穿。
- 卡点停法：女配反咬“你也有罪”，男主意识到自己不是旁观受害者。
- 源本高辨识外形：黑医、屏幕录音、亲子鉴定、当场验血。
- 下游写作抓手：换成豪门继承信托/慈善基金命名礼，利用真人证人、孩子本能反应、信托席位和公众目光，而不是只靠屏幕报告。

### HV7：旧物火葬场

- 客观事件：男主找到孕检单、婴儿衣物、旧戒指，崩溃自责。
- 观众体验功能：让男主真正“从可恨到卑微”，但不替他洗白。
- 强度来源：小物件承载被他错过的爱和孩子。
- 关系 / 权力变化：男主从追问者变成跪地求赎罪者。
- 信息差状态：女主不知道他已彻底看到旧物，或不在乎。
- 卡点停法：暴雨中跪地痛哭。
- 源本高辨识外形：仓库保险箱、孕检单、婴儿毛衣、雨中自扇。
- 下游写作抓手：换成 Vale 旧湖屋/婚后储藏箱，保留物件功能，替换黑帮仓库外形。

### HV8：当众下跪交出一切

- 客观事件：男主在高端场合当众下跪，交出家族权力求原谅。
- 观众体验功能：最大高低位反转；女主拒绝廉价原谅。
- 强度来源：男主曾经最在乎的身份、股份、尊严全部放在女主脚边。
- 关系 / 权力变化：女主完全高位；男主低到尘埃。
- 信息差状态：女主仍不允许“下跪=还债”。
- 卡点停法：女主给极端选择，画面停在结果前。
- 源本高辨识外形：旋转餐厅、教父徽章、枪口、枪响。
- 下游写作抓手：换成顶层股东晚宴/空中餐厅，男主交出投票权、家族信托和 CEO 印章；生死选择用屋顶边缘/应急门，不用枪。

## 5. 钩子与卡点

- Ep1 卡：女配露出要继续害女主的真面目。
- Ep2 卡：女主被火吞没，孩子命运悬置。
- Ep3 卡：五年后高跟鞋入场，身份未揭。
- Ep4 卡：女主身份揭开并逼男主动不了她。
- Ep5 卡：男主资产被切，女配被女主踩到恐惧。
- Ep6 卡：男主拿到怀孕线索。
- Ep7 卡：公共撕假孕后，男主仍本能护面子，引出更大反咬。
- Ep8 卡：女配反咬男主亲手害死女主孩子。
- Ep9 卡：男主雨夜崩溃，准备找女主求赎。
- Ep10 卡：女主提出“一命还两命”，男主真的走向边缘。

## 6. 爽点 / 压抑-释放

- 压抑：女主被替换、被误解、求救无门、失去孩子。
- 第一释放：五年后高位亮相，男主家族第一次被迫让路。
- 第二释放：女配套房/后台被反压，旧账开始结算。
- 第三释放：公共仪式上假孕和假继承人崩盘。
- 第四释放：男主崩溃下跪，但女主不轻易原谅，爽点不滑成和解。

## 7. 关系 / 权力变化

- 女主与男主：妻子低位 -> 被抛弃者 -> 高位债权/控制人 -> 审判者。
- 男主与女配：保护者 -> 被欺骗的保护者 -> 想杀但被罪责压住的人。
- 女主与女配：被压迫者 -> 清算者；女配必须持续作死才保住短剧摩擦。
- 女主与男配：救命盟友/资本盟友；用来刺激男主嫉妒，但首批不让感情线吞主线。

## 8. 信息差与真相路径

首批不能一次把所有真相摊完。

- Ep1：观众知道女主怀孕，男主不知道。
- Ep2：男主听到女主喊怀孕但不信；女主流产成为观众确认痛点。
- Ep3：观众知道女主已重生；男主还不知道她是 Nightingale。
- Ep4：男主知道她是 Ava / Nightingale，但不知道孩子死因全貌。
- Ep6：男主知道她当时怀孕，但不知道 Serena 假孕、孩子非亲生的完整事实。
- Ep7-Ep8：男主知道自己被 Serena 骗，且自己的选择害了 Ava；但真正的旧物和孩子细节留到 Ep9。
- Ep9：男主看到双胞胎、旧物、自己曾经无视的生活细节，彻底崩。
- Ep10：女主知道他跪，但不把下跪当偿还。

## 9. 写作手法 / 表达手法

- 高压场面要靠空间和身体：门被撞开、光压下来、有人跪、有人被拽、物件被摔、股份章被递到脚边。
- 台词短，主要完成逼迫、拒绝、反击、卡点，不解释长因果。
- 反转不能全靠屏幕和报告，要让证人、孩子反应、公众目光、身体选择和权力动作一起完成。
- 每集结尾停在下一步之前，不把答案说完。

## 10. 水分 / 烂点 / 禁止复用外形

- 禁止把黑手党、枪战、军火线原样换名为“商业合同”。
- 禁止用会议室长对白代替高压场面。
- 禁止 Ep7-Ep8 只靠报告、屏幕、亲子鉴定、反派否认推进。
- 禁止女主回归只靠“变美、变有钱”，必须带着能改写男主公司命运的权力。
- 禁止男主在 Ep10 因下跪就获得原谅。
- 禁止提前把后续追妻火葬场、男配裂痕、女主心软全部消费掉。
