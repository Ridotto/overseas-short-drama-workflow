- 动作和场景描述中文：已确认。

## 2. 本轮创作判断

源本最强体验不是“黑手党”和“军火”，而是：

1. 女主被丈夫低估、被情敌踩脸，仍藏着更强身份。
2. 男主在关键生死选择里选错人，导致不可逆失去孩子。
3. 女主五年后高位归来，不靠解释，靠当众后果碾压。
4. 女配的谎言不是一次揭穿，而是连续反噬男主的尊严、权力和自我认知。
5. 第 9-10 集从“知道错了”升级到“被物件打穿”，最后男主公开下跪。

新壳必须承载这些体验，但不能复用：

- 黑手党家族 / 军火线 / 黑帮宴会 / 地下拍卖；
- 车祸、火海、两辆车二选一；
- 大屏证据连播 + 亲子鉴定报告 + 当场验血的串联外形；
- 仓库保险箱 + 婴儿毛衣 + 雨中下跪的连续外形。
