# PRD v15-v4workflow Source Retention Anchor

> 这是最薄的源本留存锚点。后续 Story Operating State、Episode Plan、Pressure-chain Writer Brief 和正文都必须承接它。

## 核心期待链

```text
她被丈夫和白月光一起抹掉
-> 她求救时，丈夫选择别人
-> 她失去孩子和旧身份
-> 她五年后以更高权力回归
-> 她让白月光在公众面前一步步崩
-> 男主先震惊、再怀疑、再被真相击穿
-> 男主发现自己亲手造成孩子死亡
-> 男主公开跪求，女主握住是否毁掉他/原谅他的权力
```

## 必须等效迁移的钩子 / 卡点

| 集数 | 必须保留的追问惯性 |
|---|---|
| E1 | 女主离开后，反派仍继续害她；她的孕情/价值没有被男主承认 |
| E2 | 男主二选一放弃女主，身体损失发生，生死未卜 |
| E3 | 女主失子后重生，五年后只给回归姿态，不给完整解释 |
| E4 | 女主压过男主熟悉的权力场，男主第一次意识到她不可控 |
| E5 | 女主证明自己能动利益和动人，反派怕但继续作死 |
| E6 | 男主进入真相，但不能一次全知道 |
| E7 | 白月光继承人叙事被公开撕开，男主颜面被挂起来 |
| E8 | 白月光反咬男主，让“他也是凶手”成为下一步无法逃避的问题 |
| E9 | 男主主动追回旧物，被自己当年的选择击穿 |
| E10 | 男主公开把尊严交给女主，女主卡在公开毁灭动作前 |

## 情绪密度和峰谷

首批不是均匀刺激，节奏是：

```text
E1-E2 极压：替代 + 放弃 + 失子
E3 缓一拍但不降：悲痛断裂 + 重生
E4-E5 爽点兑现：回归 + 反打
E6 再拉紧：旧伤刺穿男主
E7-E8 极爆：继承人骗局 + 绿帽 + 反咬罪责
E9-E10 火葬场：追悔 + 跪求 + 女主不赦免
```

## 爽点压抑-释放方式

爽之前必须先压：

- E4 打脸之前，先让 Chloe 当众羞辱女主；
- E7-E8 崩盘之前，先让 Chloe 占住“合法继承人母亲”的位置；
- E10 跪求之前，先让 Adrian 一次次想用旧权力靠近但失败。

## 关系拉扯

关系拉扯不是温柔对话，而是：

```text
Adrian 想靠近 / 查问 /补偿
-> Serena 身体和态度都抗拒
-> Adrian 被更具体的旧伤和旧物拖回罪责
-> Serena 不允许他把迟来的深情变成赦免
```

## 信息差 / 真相揭露节奏

- E1-E2：女主说真话，但男主不信；
- E6：男主只看到裂缝；
- E7-E8：白月光骗局公开裂开；
- E9：男主自己找回完整罪证；
- E10：男主公开说出罪，女主还不放过。

## 付费前后追问惯性

每个强卡点都要停在观众知道“下一秒更狠”的位置：

- 门要关了，她还没被救；
- 女主进场了，她还没摘墨镜；
- 孩子喊错人了，大人还没反应完；
- 男主跪下了，女主还没按直播。

## 不能被新壳洗掉的体验

1. 男主必须是做错选择的人，不是被系统/家族/流程误导的旁观者；
2. 女主损失必须是身体和孩子级，不是职场损失；
3. 白月光必须当众失态；
4. 女主必须亲手制造局面，而不是提交审计材料；
5. 男主追悔必须来自主动寻找旧伤，不是别人把证据端给他；
6. 首批结尾必须保留“女主是否处刑/是否公开毁掉他”的追问。
