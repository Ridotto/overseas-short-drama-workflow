# PRD v16-v6workflow Story Operating State

> 写作前故事状态。它不是人物设定表，也不是剧情摘要；它给 writer 当前场面的创作坐标。

## 1. 本批开始时

### Serena Hale / Aria North

- 想要：让 Blackwell、Adrian 和 Chloe 亲眼看见他们五年前抹掉的人和孩子；不是只赢公司，而是夺回审判权。
- 怕什么：怕自己仍会被旧伤和孩子记忆击穿；怕 Adrian 把迟来的崩溃包装成爱和赎罪。
- 误会 / 防御：她不相信任何补偿，默认所有纸面转让都是逃避真实罪责。
- 身体 / 情绪状态：旧玻璃门、手铐、孕期失血造成手腕、小腹和锁骨创伤；外表冷，身体会先暴露抗拒。
- 主动权：五年后掌握 North Capital、Blackwell 融资和媒体议程。

### Adrian Blackwell

- 想要：起初保住 Blackwell 和 Chloe 的继承人叙事；中段想查 Serena 变成什么；后段想从罪责里求一条赦免。
- 怕什么：怕自己不是被骗者，而是当年真正做选择的人。
- 误会：E1-E5 仍把 Serena 的回归理解成报复和抢权；E6 后开始害怕她说的是真的。
- 身体 / 情绪状态：傲慢到失控，越靠近 Serena 越被旧伤挡回去。
- 主动权变化：E1-E2 有绝对权力；E4 起房间权力被 Serena 买走；E8 后道德位置崩。

### Chloe Whitmore

- 想要：保住 Blackwell 未婚妻 / 继承人母亲 / Foundation 主席位置。
- 怕什么：Noah 和 Bennett 的关系曝光；五年前 red folder 和 service corridor 责任链曝光。
- 策略：先装柔弱，后用孩子和家族体面绑住 Adrian；暴露后拖 Adrian 下水。
- 主动权：靠 Adrian 和镜头维持，离开公开人设就会慌。

### Marcus Arden

- 想要：帮助 Serena 站稳，同时利用 North Capital 打穿 Blackwell。
- 功能：提供外部资源和保护，但不替 Serena 完成复仇高潮。

## 3. 本批观众最想看

- Serena 如何从被拖走的人，变成让整个房间让路的人；
- Adrian 什么时候意识到他当年不是“被骗”，而是亲手选择；
- Chloe 什么时候在公开镜头下露出真脸；
- Serena 会不会让 Adrian 公开身败名裂；
- Serena 是否仍有一点旧痛，但不让旧痛变成原谅。

## 4. 本批必须推进

1. 建立 Serena 被替代和被放弃的身体级损失。
2. 完成五年后回归和第一轮反压。
3. 打开 Chloe 假继承人叙事。
4. 让 Adrian 逐步从掌权者变成追旧物的人。
5. 以公开处刑权作为首批终钩。

## 5. 本批绝不能提前消费

- 不能在 E2 让 Adrian 知道孩子已死。
- 不能在 E4-E5 让 Serena 解释完整五年经历。
- 不能在 E6 一次给出双胎完整证据。
- 不能在 E7 让 Chloe 完整认罪。
- 不能在 E10 给 Serena 原谅或明确不原谅的最终答案。
