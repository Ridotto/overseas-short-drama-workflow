# PRD V18 v8workflow 洗稿方案

## 0. 本步用途

在写正文前确认“这次怎么洗”。本文件给主控和 reviewer 看，也给关键场面迁移卡提供选择依据。

## 1. 一句话方案

把“黑手党少主误信伪怀孕白月光、弃救怀孕前妻、五年后被前妻以黑道女王身份反杀”的追妻火葬场，洗成“家族集团继承人误信旧爱伪孕叙事、在事故现场抛弃怀孕妻子、五年后被前妻以危机投资人与供应链女王身份夺回控制权”的海外豪门商战短剧。

## 2. 保留什么

- 核心期待：女主何时反杀、男主何时知道、女主是否原谅。
- 钩子惯性：每集停在身份、证据、选择或权力后果落下前一拍。
- 爽点周期：先压三集，再以女主回归、权力碾压、公开揭谎、男主下跪递进兑现。
- 情绪走势：期待破灭 -> 绝望失子 -> 冷硬复仇 -> 男主崩溃。
- 关系拉扯：旧婚姻不是消失，而是变成女主审判男主的绳索。
- 信息差：观众先知道女主怀孕和女配作恶，男主后知后觉。
- 权力变化：男主从家族 CEO 和继承人变成求女主裁决的人。

## 3. 新壳故事发动机

Whitmore Group 是一个用婚姻、慈善基金会、继承信托和董事投票维持家族统治的跨国集团。Adrian 需要一个“怀孕旧爱”来完成家族继承叙事，Celeste 需要借这个孩子进入 Whitmore 核心，董事会需要稳定继承人和股价。

Ava 过去被当作没用的花瓶妻子，实际三年里替集团补过供应链风控漏洞。事故后她被 Julian Frost 救走，用五年建立 Kestrel 风控系统和危机投资基金。她回来不是抢男人，而是掌握 Whitmore 的债务、供应链、董事票和基金会审计命门。

所以剧情自然发生：家族企业为了继承和面子牺牲她；她回来后用同一套企业权力系统反过来审判他们。

## 4. 换掉什么

- 黑帮庄园 -> Whitmore 海滨庄园和董事会。
- 黑帮车祸救人 -> 慈善晚宴后车队事故与企业安保救援调度。
- 黑帮拍卖会 -> 家族基金会慈善拍卖 / 股东见面晚宴。
- 军火断供 -> 全球酒店供应链、物流保险和债务授信冻结。
- 黑医证词 -> 私人诊所主任、基金会医疗档案和事故调度录音。
- 黑道宴会公开羞辱 -> 年度股东会特别听证。
- 跪地交家族 -> 男主交出投票信托、CEO 辞呈、公开证词。

## 5. 删除什么

- 不进入黑帮火并、枪战、地下私刑。
- 删除纯炫酷身份标签，女主强要通过可见动作证明。
- 删除“众人震惊”作为独立爽点；现场后果必须具体。
- 删除男主用大段悔恨台词替代行动。

## 7. 高压节点处理

进入关键场面迁移卡：

- KC1：E1 正妻位置被旧爱和继承叙事取代。
- KC2：E2 事故现场的弃救选择。
- KC3：E3 失子觉醒 + 五年后回归入口。
- KC4：E4 慈善拍卖/债务控制权身份反转。
- KC5：E6 伤疤和双胎线索逼查。
- KC6：E7-E8 股东会公开揭谎与亲子复核。
- KC7：E9 男主找到遗物崩溃。
- KC8：E10 男主交权求原谅。

## 8. 主要风险

- 洗飞：若只剩资本操作，会丢掉失子、误判、追妻火葬场核心。
- 改名复刻：若 E4/E7/E10 只是把黑帮宴会换成董事会，会同构。
- 短剧降级：若证据只靠屏幕和长对白，会冷。
- 长剧化：若人物坐着解释商业规则，会拖。
- 状态漂移：E6 前男主不能提前相信 Ava；E8 后男主才彻底崩。

## 9. 写作原则

- 商战信息必须外化成动作：投票权失效、安保换边、董事离席、信托冻结、股价警报、供应链停摆。
- 台词只做攻击、防守、逼迫、拒绝、试探、揭露、反击和选择。
- 每集至少有一个关系/信息/权力状态变化。
