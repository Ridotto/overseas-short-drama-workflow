# PRD v13-v3workflow Adaptation Boundary

> 本文件是审计线产物，给主控和 reviewer 查洗飞 / 同构 / 降级。  
> Writer 不读取完整版本，只读取 `Writer Brief`。

## 1. 本轮需求边界

- 新壳：海外豪门商战 / 家族企业权力斗争。
- 目标市场：海外短剧。
- 集数：不变。
- 范围：首批 1-10 集。
- 改写力度：中度偏重。
- 目标：不能只是换名；不能丢掉源本 1-10 的追剧体验。

## 2. 强继承：体验，不继承标志性外形

| 源本有效性 | 本轮强继承方式 |
|---|---|
| 女主被替代 | 女主被丈夫、家族和公众一起从继承人叙事里抹掉 |
| 男主错误选择 | 男主在众目睽睽下选择旧爱和集团颜面，放弃女主求救 |
| 不可逆损失 | 女主失去双胎，并被迫从原身份消失 |
| 五年后回归 | 女主以资本控制人身份回到男主最熟悉的权力场 |
| 反派骗局 | 旧爱伪造救命功劳 / 继承人叙事 / 亲密关系 |
| 男主崩溃 | 男主看到自己亲手造成损失，不只是听别人告诉他 |
| 女主不原谅 | 男主交出权力和尊严，女主仍保留惩罚权 |

## 4. 禁止写法

- 禁止把高压戏写成“屏幕显示 -> 人震惊 -> 又一个文件 -> 再震惊”。
- 禁止让关键选择由系统 / 授权 / key / 合规程序自动完成。
- 禁止 writer 为了证明源本迁移而在正文里显式复述源本功能。
- 禁止把女主写成冷静审计员，她必须有行动压迫和情绪伤口。
- 禁止用多轮低信息对白替代动作。

## 5. 允许的辅助工具

商业壳可以出现合同、股权、董事会、媒体、直播、安保、医院记录。

但它们只能做辅助，不能做场面发动机。高压场的发动机必须是：

```text
谁在场；
谁逼谁选择；
谁当众失去；
谁身体 / 空间上被压住；
谁亲手做了不可撤回的事。
```
