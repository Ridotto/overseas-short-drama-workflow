# PRD v15-v4workflow Source Bible

> 本文件是源本分析底稿，不直接给 writer 当 prompt。  
> 源本：`[修改版] 黑手党少主痛哭求原谅_前妻她杀疯了.txt`  
> 范围：1-10 集。  
> 目标：回答源本为什么有效，以及首批 1-10 必须迁移的观众体验。

## 1. 一句话故事核

一个为爱隐忍的女人，被丈夫和白月光一起误判、替代、抛弃，在生死二选一中失去孩子；五年后她以更高权力回归，把白月光骗局和男主罪责一层层公开撕开，逼男主从不可一世变成跪地求原谅。

## 2. 核心期待

```text
她什么时候不再忍？
他什么时候发现自己救错人、爱错人、害死了自己的孩子？
白月光什么时候当众翻车？
女主什么时候把曾经踩她的人全部踩回去？
男主跪到什么程度，女主会不会原谅？
```

## 4. 高价值节点

### HV1：E1 被替代和被抹掉

- 发生了什么：白月光怀孕进入家门，男主当面逼女主离婚，把“庄园女主人”位置交给白月光。
- 有效原因：女主还怀着孩子、还想给男主惊喜，却被男主当众否定三年价值。
- 强度来源：一边是她隐藏的孕检单，一边是对方高调宣称怀了男主孩子。
- 人戏核心：男主不只是变心，而是亲手确认“你不配坐这个位置”。
- 禁止复用外形：黑手党庄园、吞孕检单、厨房意面、白月光大肚子整串照搬。

### HV2：E2 生死二选一

- 发生了什么：两车遇险，女主被困，白月光也呼救；男主先救白月光，把女主留在爆炸里。
- 有效原因：女主不是单纯遇难，而是在向男主求救时被男主亲手放弃。
- 强度来源：车门卡死、汽油、火、拍窗、怀孕求救、男主抱走别人。
- 人戏核心：男主被迫选择，他的选择导致不可逆损失。
- 禁止复用外形：车祸、出租车爆炸、两车并排复制。

### HV3：E3 失子和人格断裂

- 发生了什么：女主醒来摸小腹，得知孩子没了，同时看到男主在电视里保护白月光和“孩子”。
- 有效原因：失去孩子后立刻看见仇人被公开保护，痛和恨同时完成。
- 强度来源：空的小腹、电视公开护短、Victor 给出复仇通道。
- 人戏核心：她不是“变强”，而是从“想被爱”断裂成“要他们陪葬”。

### HV4：E3 尾 / E4 回归压场

- 发生了什么：五年后，女主以更高权力身份回到男主权力场。
- 有效原因：观众等的不是身份解释，而是“她回来后第一脚踩谁”。
- 强度来源：先不说话，众人反应；再用价格、权力、耳光改变房间秩序。
- 禁止复用外形：拍卖粉钻、夜枭军火女王照搬。

### HV5：E5 女主能力兑现

- 发生了什么：女主切断男主生意，白月光上门挑衅被女主当场制服。
- 有效原因：回归不是空架子，她能动男主利益，也能让白月光付身体代价。
- 强度来源：反派主动闯入、带人却被反压、手指被踩碎。

### HV6：E6 真相入口

- 发生了什么：男主看到女主伤疤和医疗记录，第一次知道她当年怀孕。
- 有效原因：他开始怀疑“我当年是不是害错了人”。
- 风险：如果只读报告，会降级成信息戏。必须让他先被身体旧伤刺到。

### HV7：E7-E8 白月光骗局崩塌

- 发生了什么：买凶、假孕、出轨、非亲生、绿帽、流产真相连环爆。
- 有效原因：白月光不是被证据击败，而是在众人面前失控，男主颜面被踩碎。
- 强度来源：当众围观、男主掐脖、白月光跪抱腿、反咬男主也有罪。
- 风险：如果写成证据播放，会变听证会。

### HV8：E9 男主追悔火葬场

- 发生了什么：男主看到旧物和旧空间，确认孩子存在，跪地自扇。
- 有效原因：抽象罪责变成摸得到的旧物、血迹、被他当年忽视的小幸福。
- 风险：旧物不能像礼包一样送到他手上，他必须主动追查并撞墙。

### HV9：E10 权力倒置与首批终钩

- 发生了什么：男主公开跪求，把自己从权力顶端降到女主脚下，女主握住原谅/处刑权。
- 有效原因：观众要看不可一世者失去尊严，而女主不立刻赦免。
- 卡点：停在女主是否公开按下毁灭按钮 / 是否让他彻底社会性死亡前一拍。

## 6. 爽点 / 压抑-释放

源本不是一直爽，而是先压到底再释放：

```text
压抑：被替代 -> 被误判 -> 被放弃 -> 失子
释放：五年后回归 -> 打白月光 -> 断男主利益 -> 公开撕谎
二次压抑：男主还想用旧权力靠近她
二次释放：男主尊严坍塌 -> 跪求 -> 女主不原谅
```

迁移时必须保留这种“先被踩，再让踩她的人当众失态”的落差。

## 11. 源本高辨识外形

这些外形不能整串照搬：

- 黑手党庄园厨房、意面、吞孕检单；
- 两车车祸、出租车爆炸；
- 夜枭 / 军火女王 / 黑帮拍卖；
- 亲子鉴定报告当众甩；
- 仓库、小毛衣、雨中跪地、枪口抵额整套形状。

## 12. 源本水分 / 烂点

- 证据连续播放容易变听证会；
- 第 6-8 集如果只靠报告，会冷；
- 白月光直接认罪太多，会像作者交代；
- 男主追悔如果靠旧物礼包，会显得方便；
- 台词里重复“你害死孩子 / 我错了”会稀释情绪。
