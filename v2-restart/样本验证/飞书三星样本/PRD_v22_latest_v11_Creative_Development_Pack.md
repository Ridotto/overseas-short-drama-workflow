# PRD_v22_latest_v11_Creative_Development_Pack

## Run Contract

- 新壳：海外豪门商战。
- 目标市场：海外短剧市场，偏强情绪、强反转、强身份落差。
- 目标受众 / 口味：喜欢 betrayed wife comeback、billionaire family power、public humiliation、late regret、no-easy-forgiveness。
- 题材类型：现代豪门 + 跨国科技财团 + 资本复仇。
- 风格元素：高定晚宴、董事会、家族信托、芯片供应链、车祸火场、雨夜赎罪。
- 输出语言：台词英文；动作和场景描述中文。
- 正文用途：人审稿 + AI 视频生成 page-to-screen。
- 改写力度：保体验功能，换事件载体和行业外形。

## New Story Premise

- 新本一句话故事：被豪门继承人抛在火里失去双胞胎的隐名商业天才，五年后以全球芯片供应链女王身份归来，夺走他的集团命脉，撕穿假继承人骗局，让迟来的前夫跪交整个 Blackwood 帝国。
- 核心设定：Blackwood Group 的 AI 芯片和自动驾驶供应链，暗中依赖一套名为 VESPER 的风控授权系统。所有人以为 Ava 只是没背景的豪门弃妇，其实她是 Vesper Systems 的隐名创始人。
- 核心期待：Ava 什么时候让 Adrian 知道他亲手毁掉了自己的双胞胎；Adrian 会跪到什么程度；Ava 会不会原谅。
- 核心亮点：低位妻子吞下双胎报告；事故火场抛弃；五年后用供应链和投票权回归；继承人信托仪式公开失败；雨夜跪交 Blackwood。
- 核心梗概：Ava 为爱隐名嫁入 Blackwood，替 Adrian 稳住集团供应链，却被初恋 Serena 以假继承人逼退。Serena 设计车祸，Adrian 抱走 Serena，Ava 在火里流产。五年后，Ava 以 Vesper Systems 掌舵人身份回来，在 Blackwood 最需要芯片授权时卡死命脉，公开揭穿 Serena 的假继承人身份。Adrian 找到双胎遗物后崩溃，下跪交出集团控制权，但 Ava 只让他继续活在赎罪里。

## Story Engine

- 主角想要什么：Ava 要夺回尊严、让 Serena 和 Adrian 付出代价，并把 Blackwood 曾经压在她身上的权力反过来踩碎。
- 阻碍是什么：Adrian 前期不信她，Serena 占着继承人母亲位置，Blackwood 家族董事会只认血脉和股权，Julian 的帮助也带利益交换。
- 赌注是什么：如果 Ava 输，她会再次被定义成骗子、弃妇、商业投机者；如果 Adrian 继续错信 Serena，Blackwood 会把真正的继承权、供应链和公众信用交给假继承人。
- 观众追什么：Ava 每次回击能拿走什么；Serena 的谎什么时候爆；Adrian 何时知道双胞胎；Ava 怎么拒绝迟来的深情。
- 核心误会 / 信息差：Adrian 误以为 Ava 曾经嫉妒 Serena、伪造事故、没有商业能力；真相是 Ava 曾怀双胎并被 Serena 陷害，Serena 的孩子也不是 Adrian 的。
- 核心关系张力：Ava 恨 Adrian，但每次看到他迟来痛苦都会被旧伤刺到；Adrian 爱得太晚，所有深情都像给自己赎罪；Julian 是救命盟友，也提醒 Ava 不要再被 Blackwood 困住。

## Core Truth + Pursuit / Payoff Model

- 全剧主追问：Ava 会不会亲手摧毁 Blackwood，并让 Adrian 为双胞胎付出不可逆代价？
- 当前阶段追问：Ava 如何从“被赶走的前妻”变成 Blackwood 命脉掌控者；Adrian 什么时候知道 Serena 的孩子不是自己的。
- 单集追问生成方式：每集围绕一个公开身份 / 权力标识 / 物件被夺或被揭开推进，例如妻位、信托、供应链、病历、继承人徽章、创始人徽章。
- 真实事实是什么：
  - Ava 是 Vesper Systems 隐名创始人。
  - Ava 当年怀了 Adrian 的双胞胎。
  - Serena 的儿子不是 Adrian 的，是她和外部情人所生。
  - Serena 伪造孕周、产检记录，并安排车祸制造“被 Ava 伤害”的假象。
  - Adrian 在火场选择 Serena，客观上害死 Ava 的双胞胎。
- 角色当前误以为什么：
  - Adrian 以为 Serena 给了他 Blackwood 继承人，Ava 当年只是嫉妒离开。
  - Serena 以为 Ava 只是在靠 Julian 的资本回归，不知道 Ava 掌握完整 VESPER 授权。
  - Blackwood 董事会以为 Ava 是外部收购者，不知道她曾是集团关键系统真正设计者。
- 观众当前知道什么、还不知道什么：
  - 观众从 Ep1 知道 Ava 怀双胎和隐藏身份。
  - 观众从 Ep2 知道 Serena 设计事故。
  - 观众到 Ep8 才看到假继承人当众失效。
  - 观众到 Ep9 才看到 Adrian 私人物证崩溃。
- 误导方伪造 / 遮蔽 / 扭曲了哪一层：
  - Serena 伪造产检和孕周；
  - 删除车祸路口监控；
  - 以“继承人母亲”身份占据 Blackwood 家族信托；
  - 将 Ava 描述成嫉妒、危险、商业失败者。
- 哪一层事实在哪个阶段兑现：
  - Ep1-Ep2：观众知道 Ava 双胎和 Serena 杀意。
  - Ep4：Adrian 知道 Ava 是 Vesper。
  - Ep6：Adrian 知道 Ava 当年怀孕。
  - Ep7：公众知道 Serena 伪造孕检和事故线。
  - Ep8：公众知道 Serena 的孩子不是 Adrian 的。
  - Ep9：Adrian 私人知道双胞胎和自己未曾珍惜的物件。
  - Ep10：Adrian 用 Blackwood 身份公开赎罪，但 Ava 不原谅。
- 哪一层事实、关系变化、情绪击穿不能提前兑现：
  - Ep6 前不能让 Adrian 承认怀孕事实。
  - Ep8 前不能让 Adrian 确认假继承人。
  - Ep9 前不能让 Adrian 完整崩溃。
  - Ep10 前不能公开下跪。
  - 首批不能让 Ava 原谅或接受 Adrian。
- 兑现后变化：
  - Ep4 后：Ava 拿到外部高位，Adrian 失去供应链主动权。
  - Ep6 后：Adrian 的怀疑转成恐惧。
  - Ep8 后：Serena 失去继承人母亲保护伞，Adrian 公开丢脸。
  - Ep10 后：Adrian 失去 Blackwood 高位，进入赎罪线。
- 已播事实自洽检查：本版锁定 Ava 双胎；Serena 有孩子但非 Adrian；Serena 伪造的是父亲身份和孕周，不再写“完全没怀孕”。

## Character Operating State

### Ava Cross / Vesper

- 当前欲望：让 Blackwood 承认曾经被他们踩掉的人，才是他们现在最需要的人；让 Adrian 亲眼看见自己错信谁。
- 当前误判：她以为只要夺走权力就不会再疼，但孩子线一碰仍会失控。
- 底线：不让 Serena 继续以继承人母亲身份站在高位；不接受 Adrian 用一句爱或一份文件抵债。
- 行动方式：公开场合少说，直接拿走身份标识、投票权、供应链权限；每次反击都让对方当众失去一样东西。

### Adrian Blackwood

- 当前欲望：稳住 Blackwood 继承权、保住 Serena 和“儿子”的名分，并弄清 Ava 为什么变成 Vesper。
- 当前误判：把 Ava 的复仇理解成嫉妒和商业挑衅；以为财产和妻位可以召回她。
- 底线：前期不能承认自己害了 Ava；Ep9 后底线崩溃，只剩求赎。
- 行动方式：前期用家族命令、安保、董事会和资源压人；中后期用自毁、下跪和交权求原谅。

### Serena Vale

- 当前欲望：保住准夫人位置和儿子的 Blackwood 继承权。
- 当前误判：以为 Adrian 的旧情和家族血脉焦虑能永远保护她。
- 底线：一旦继承人身份受威胁，她会不顾体面地撒谎、反咬、装可怜。
- 行动方式：公开场合装弱，私下雇人、销证、带媒体闹场。

### Julian Stone

- 当前欲望：借 Ava 重新切入 Blackwood 供应链，同时真心欣赏她的狠和能力。
- 当前误判：以为 Ava 能完全把旧情切掉。
- 底线：不做白给的救命恩人，所有帮助都带战略利益。
- 行动方式：提供资本、安保和商业入口，不替 Ava 完成最关键的复仇动作。

## Relationship / Power State

- Ep1 起点：Adrian 与 Serena 高位，Ava 低位；观众知道 Ava 有隐藏强身份。
- Ep4 变化：Ava 外部权力压过 Adrian，Serena 首次公开恐慌。
- Ep6 变化：Adrian 看到伤疤和病历，权力仍在但心理开始塌。
- Ep8 变化：Serena 继承人保护伞崩；Adrian 公开低位。
- Ep10 变化：Adrian 跪交 Blackwood；Ava 拥有绝对处置权。

## Character Operating Notes

- Ava 的台词少而冷，真正表达靠动作：吞报告、撤授权、摘徽章、打开露台门。
- Adrian 前 1-6 集不能软；他越笃定，Ep9 崩溃越有偿还感。
- Serena 不能只恶毒，她要懂得借家族继承权和媒体同情保护自己。
- Julian 不能像救世主。他是把 Ava 从废墟带回资本桌的人，但 Ava 的复仇必须由她自己完成。
