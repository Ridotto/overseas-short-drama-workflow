# PRD_v21_v11workflow_Creative_Development_Pack

## Run Contract

- 新壳：海外豪门商战。
- 目标市场：AI 出海短剧，英文台词，中文动作描述。
- 目标受众 / 口味：偏强情绪、快反转、强身份逆转、女主复仇、男主火葬场；接受夸张豪门权力动作，但不能冷成职场会议。
- 题材类型：豪门婚恋复仇 + 家族集团争夺 + 资本控局。
- 风格元素：顶层餐厅、私人庄园、董事席、信托仪式、债权拍卖、港口供应链、私人 helipad、媒体闪光灯。
- 输出语言：台词英文；动作和场景描述中文。
- 正文用途：人审稿 + 后续 AI 视频生成。
- 改写力度：强换事件载体和权力系统；强继承源本追看功能、情绪链和首批节奏。

## New Story Premise

### 新本一句话故事

隐忍三年的豪门妻子 Ava 被丈夫 Ethan 为怀孕初恋公开抛弃，并在一场庄园 helipad 火灾中失去孩子；五年后，她以神秘资本操盘手 Nightingale 的身份回归，夺走 Vale 家族的债权、供应链和继承权，把欺骗她的初恋和错信初恋的丈夫一步步逼到公开崩塌。

### 核心设定

- Vale Group 是横跨奢侈品、港口物流和冷链供应链的百年家族集团。
- Ava 婚后三年被当成无能阔太，实际一直匿名替 Vale 修补资金链和供应链漏洞。
- Serena 以“怀了 Vale 继承人”为筹码登堂入室。
- 五年后 Ava 控制 Nightingale Capital，专做 distressed debt、供应链收购和董事席位夺取。

### 核心期待

- Ava 何时让 Ethan 亲眼看到自己当年错救的“继承人”是假的。
- Ethan 何时发现自己真正害死的是自己的孩子。
- Ava 会不会被 Ethan 的下跪打动，还是继续让他活着偿还。

### 核心亮点

- 豪门身份替换不是离婚室内吵架，而是在家族继承餐上公开宣布。
- 生死抛弃不是车祸，而是私人 helipad 火灾，Ethan 必须在两道火线之间选择。
- 女主回归不是黑帮女王，而是能让男主集团当场断贷、断港口、断董事席的资本控制人。
- 假孕假继承人不是一张亲子报告，而是在继承信托命名礼上被真人、孩子反应和信托席位共同撕开。

### 核心梗概

Ava Hart 隐瞒 Nightingale 身份嫁给 Ethan Vale，三年里替他填平供应链危机，却被他视作只能摆盘煮汤的无用妻子。Ethan 带着怀孕初恋 Serena 回到 Vale 庄园，要求 Ava 签离婚并交出“Vale 女主人”身份。Serena 随后制造 helipad 火灾，Ethan 在火海中救走 Serena，把呼喊怀孕的 Ava 留在燃烧的隔离门后。Ava 流产后被 Julian Cross 救下。

五年后，Ava 以 Nightingale Capital 掌控 Vale 的债权和港口融资，回到债权拍卖夜，公开压下 Ethan。她先切断 Vale 的现金和港口线，再当众撕开 Serena 假孕、假继承人的骗局。Ethan 找到当年的旧物和双胞胎超声，明白自己为了别人的孩子害死了自己的孩子，彻底崩溃。首批结尾，他在 Skybridge 股东晚宴上交出 Vale 一切，跪求 Ava 原谅；Ava 让他用命换两条命。

## Story Engine

- 主角想要什么：Ava 要夺回尊严、权力和孩子的债；不是要证明自己还值得 Ethan 爱。
- 阻碍是什么：Ethan 的旧权力、Serena 的假继承人牌、Vale 家族董事会对血脉和面子的执念。
- 赌注是什么：Ava 如果只靠恨，会变成另一个操控者；如果心软过早，孩子的债会被廉价抹平。
- 观众追什么：Ethan 什么时候知道，Ava 什么时候当众打脸，Serena 什么时候被撕，Ava 什么时候拒绝原谅。
- 核心误会 / 信息差：Ethan 以为 Serena 怀的是他的孩子；以为 Ava 制造事故；以为 Ava 三年毫无价值。
- 核心关系张力：Ava 曾真爱 Ethan，但孩子的死让爱变成不可直接回头的伤；Ethan 后期越卑微，越不能让观众忘记他犯过的错。

## Character Operating State

### Ava Hart / Nightingale

- 当前欲望：从 Vale 家族手里夺走权力，并让 Ethan 和 Serena 看到他们当年毁掉的不是一个“无用妻子”。
- 当前误判：她以为只要控制所有债权和局面，就能控制自己的心。
- 底线：孩子的债不能被道歉、下跪、股份交换抵消。
- 行动方式：少解释，多用资产、场面和当众选择逼别人暴露。

### Ethan Vale

- 当前欲望：前期要保住 Vale 面子和 Serena 这个“继承人”；后期要知道真相、求 Ava 留下。
- 当前误判：前期把 Serena 的柔弱当真，把 Ava 的沉默当无能；中段把 Ava 的报复理解成“她还在意”。
- 底线：Vale 家族的名字和继承人；但真相击穿后，他会把这些全部交给 Ava。
- 行动方式：高位时冷硬命令，崩溃后用身体和尊严偿还。

### Serena Blake

- 当前欲望：嫁进 Vale，坐稳继承人母亲位置。
- 当前误判：她以为 Ethan 的愧疚和家族面子会永远护住她。
- 底线：不能让假孕、真实孩子父亲和 helipad 火灾暴露。
- 行动方式：装弱、摔倒、哭、反咬；被逼急后用“你也有罪”拖 Ethan 下水。

### Julian Cross

- 当前欲望：帮助 Ava 摧毁 Vale，也借 Ava 的能力吃下 Vale 资产。
- 当前误判：他以为 Ava 对 Ethan 只剩恨。
- 底线：不能让 Ava 再被 Ethan 伤害；也不能失去与 Ava 的资本联盟。
- 行动方式：挑衅 Ethan，给 Ava 资源，但首批不越过主线。

## Relationship / Power State

- Ep1 起点：Ethan/Vale 高位，Ava 低位，Serena 借孕肚获得特权。
- Ep4 变化：Ava 以 Nightingale 身份拿回高位，Ethan 第一次被迫后退。
- Ep6 变化：Ethan 从控制者变成追问者；Ava 不再解释，只展示伤疤。
- Ep8 变化：Serena 的保护伞裂开；Ethan 从被欺骗者转为有罪者。
- Ep10 变化：Ethan 交出股份、投票权和尊严；Ava 拿住最终审判权。

## Character Operating Notes

- Ava 的每场戏都要有一个明确动作：签、摔、压、夺、停、转身，不要只冷笑。
- Ethan 前期不能软，他越冷，后期跪得越有火葬场强度。
- Serena 不能只是蠢坏，她要懂得利用豪门对继承人的焦虑。
- Julian 不替 Ava 复仇，他只提供入口和镜面刺激。
