# PRD V18 v8workflow 作者问题清单

## 1. 我最担心洗飞的地方

V18 把黑帮外形换成家族企业、基金会、债权、信托、供应链和股东会，整体能承载“权力系统反噬”的体验，但 E5-E8 的商业规则密度比源本高。风险是观众爽感从直观暴力变成程序后果后，若拍摄和剪辑不够锋利，部分海外短剧用户可能觉得“爽点来得没源本那么粗暴直接”。

## 2. 我最担心像改名复刻的地方

E1-E3 的功能和源本贴得较近：旧爱怀孕进门、事故弃救、女主失子后五年回归。这些是源本发动机，不能删，但 V18 已尽量把外形换成配偶信托、车队调度、企业救援指令、继承人基金发布会。Reviewer 仍需重点看 E2 是否还像源本车祸弃救的近距离换皮。

## 3. 我最担心短剧降级或长剧化的地方

E7-E8 股东会有大量证据和程序动作，是最容易长剧化的位置。正文里加了直播无法关闭、投票牌变色、基金会记录、信托血样、董事会秘书等现场后果，但仍需 reviewer 判断它是否足够可视化，还是偏“文件审判”。

## 4. 我认为旧版或源本局部更强的地方

源本 E10 枪响卡点更原始、更狠；V18 改成直播认罪卡点，更贴新壳，但视觉冲击不如枪响。它的优势是同构风险低、能直接打击家族企业名誉，弱点是需要演员和节奏把“直播红灯”拍出压迫感。

## 5. 关键场面迁移卡可能没执行到位的地方

- KC3 的五年成长过程正文压得很薄，只用 Julian 和 Ashline 项目交代，没有展开 Ava 如何建立 Kestrel Capital。
- KC5 伤疤逼查有执行，但“two lives”一句可能偏直接，可能需要更强的动作反应承载。
- KC6 公开揭谎有程序后果，但 E7 的 Dr. Ellis 和 dispatcher 证词仍有“上台说证据”的风险。
- KC8 男主交权有执行，但他交出的文件和 Whitmore 股东直播的后果还可以更尖。

## 6. 我需要 reviewer 特别看的地方

1. 不看源本时，V18 1-10 是否自己站得住、能追。
2. E7-E8 是否短剧化，还是变成企业合规会议。
3. E2 是否同构过高。
4. 英文对白是否有功能，是否有金句化或解释腔。
5. E10 的卡点是否足够强，是否比 V17 更有场面抓手。
