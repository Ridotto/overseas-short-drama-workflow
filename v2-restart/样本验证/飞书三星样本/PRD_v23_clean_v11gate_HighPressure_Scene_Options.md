# PRD_v23_clean_v11gate_HighPressure Scene Options

## 1. Ep2 生死选择

### 候选 A：King Tower 智能封锁

- 场面一句话：发布会夜智能总部误报火警，Chloe 在贵宾电梯，Serena 在服务器层氧气抽排区，Adrian 只能先开一扇门。
- 谁压谁：Chloe 用假孕和家族继承压 Adrian；Adrian 用命令压安保；系统压 Serena。
- 谁旁观：安保、发布会后台人员、Chloe 的护士。
- 谁被迫选择：Adrian。
- 身体 / 关系 / 社会性代价：Serena 缺氧、撞玻璃、失去孩子；Adrian 的选择成为后续罪根。
- 爽点前怎么压低：Serena 已被签走配偶席位，还想最后保住孩子。
- 爆点是什么：Adrian 明知 Serena 在玻璃后求救，仍命令打开 Chloe 通道。
- 可见后果 / 失去 / 反扑 / 卡点：白色消防泡沫吞没 Serena 的手。
- 卡点停在哪个画面 / 动作 / 声音：警报声中，Serena 的手从玻璃上滑下。
- 信息边界：不显示双胎；不显示完整流产结果。
- 同构风险：保留二选一救援功能，但载体不是车祸火海。
- 短剧降级风险：系统事故可能偏冷，必须写玻璃门、缺氧、手掌、警报。
- AI 成片可视化风险：可拍，红灯、玻璃、白色气体、手掌都清楚。
- 场面存活问：删掉屏幕 / 文件 / 报告后，仍有封锁门、警报、身体缺氧、Adrian 选择和 Serena 倒下。通过。

### 候选 B：私人飞机舱门二选一

- 淘汰原因：豪门感强，但与“车祸/爆炸”太接近，且生产难度高。

### 候选 C：医院手术室资源二选一

- 淘汰原因：逻辑容易变医疗伦理，不像豪门商战，也弱化 Adrian 主动罪责。

### 优选

选中候选 A。

原因：它把源本“救一个弃一个”的强体验换成新壳下的门禁权力和总部控制，现场戏足，后续 Ep10 可以回环。
