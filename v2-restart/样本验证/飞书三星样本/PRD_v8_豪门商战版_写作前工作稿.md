## 2. 二次需求确认记录

1. 新壳能承载源本最强节点：可以。豪门商战的等效强压力不是枪和车祸，而是公开签字、刑事背锅、投票权、继承信托、银行授信、CEO 辞任和联邦调查。
2. 改写力度：中度偏重。保留“被误判 -> 错误选择 -> 不可逆损失 -> 强势回归 -> 骗局拆穿 -> 男主火葬场”的功能链，但重建事件载体。
3. 必须迁移体验：情绪压迫、女主反击、男主误判后追悔、女配骗局崩塌、权力反转。
4. 必须避开外形：车祸、火海、黑帮枪战、亲子鉴定、保险箱旧物、雨夜跪地、枪口测试。
5. 必须出现的新壳元素：家族企业董事会、跨境资本、信托、合规、公开发布会、银行授信、联邦调查、继承权/投票权。
6. 集尾钩子允许等效重排：允许，但首批必须覆盖一次完整小闭环，且第 9-10 集不能提前消费后续感情回暖。
7. 明确禁区：不能让 Serena 只是审计者；她必须用资本动作改写局势。不能让 Adrian 的罪责只由 Chloe 喊出来；必须由 Adrian 自己签过的文件、指令和托管箱证物击穿。

## 3. 洗稿边界包

必须保留的体验功能：

- 女主被丈夫公开误判和替代。
- 男主做出无法洗白的错误选择。
- 女主承受不可逆损失，并从“被抹掉的人”变成能审判所有人的人。
- 白月光不是单纯争宠，而是构建了完整骗局。
- 男主知道真相后，不是普通后悔，而是权力身份整体坍塌。

必须换掉的源本外形：

- 黑帮身份、枪战、车祸、火海、假孕、亲子鉴定、保险箱旧物、雨夜跪地、枪抵额头。

可以自由换的外壳变量：

- 女主隐藏能力：从武器设计/黑客改为跨境资本重组、风控系统、家族企业危机补丁。
- 男主权力：从黑手党少主改为家族企业 CEO / 控股继承人。
- 女配骗局：从假怀孕改为伪造救命合同、伪造合规署名、操控刑事背锅。
- 男配：从敌对黑帮改为救下女主并扶她上位的投资人 Marcus Arden。

必须删除 / 修复的烂点：

- 不能让所有反转靠大屏播放资料。
- 不能靠长台词解释商战术语。
- 不能让第 2 集变成“事故替换事故”。
- 不能让第 10 集测试变成诱导自杀；V8 改成公开认罪和社会性死亡。

需要用户拍板的问题：

- V8 目前默认保留“失去双胎”的情绪核，但不用车祸承载；如用户想降低悲惨程度，后续可改成“失去生育能力 / 父亲死亡 / 入狱背锅”。本轮不等待额外拍板，按当前样本默认继续。

## 5. 新壳场面迁移卡

### 迁移卡 A：第 1 集逼离婚与背锅

源本节点：白月光登堂、男主逼签离婚。  
源本体验功能：女主被替代，男主轻视她的价值，女配虚伪上位。  
源本禁止外形：厨房、孕妇白月光、吞孕检单。  
新场景目的：让 Serena 从“救集团的人”被压成“背锅的人”。  
新冲突动作链：
1. 起手动作：Serena 刚完成 Atlas 风控补丁，准备拿超声去告诉 Adrian。
2. 压迫升级：Adrian 带 Chloe、律师和董事进入后台，要求 Serena 签离婚和 criminal referral。
3. 第一次反击/反转：Serena 让 Chloe 当场启动补丁，Chloe 不敢碰 root-key。
4. 不可逆后果：Adrian 撕下 Serena 配偶徽章，把她挡出发布会和婚姻。
权力变化：Adrian 和董事会压 Serena；Chloe 获得公开身份。  
关系变化：妻子变成风险源，白月光变成准合伙人。  
信息差变化：观众知道 Serena 是真救命人，Adrian 不知道。  
观众直接看到什么：徽章被扯掉、超声被踩、背锅文件压到她面前。  
证据如何触发人物行动：root-key 不解释背景，只让 Chloe 不敢启动，形成第一道裂缝。  
台词上限：每轮不超过 2 句解释，不讲长合规。
