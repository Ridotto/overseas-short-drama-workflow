# 洗稿方案摘要

## S2 gate

- gate_status: pass_for_s3

- 过线证据 2：已经明确 forbidden shape，避免后续滑回狼人 / 地下赌场 / 宠物链
- 当前风险：如果后续只保住事件功能，不保住人身压迫和公开代价，新壳会立刻掉平

## 强继承体验

这一版必须强继承的不是狼人，而是以下体验：

1. **强绑定无法轻易切断**  
   男女主因为一场危险夜晚被迫绑在一起，不是普通偶遇。

2. **男主先误判，再转成极强保护**  
   他必须先犯错，后面每次出手保护才有反差和补偿感。

3. **孕与胎儿安危是核心债务**  
   不是“她怀孕了”这个信息，而是这个孩子真的会在羞辱、暴力、转运中出事。

4. **女主持续遭受家内与外部势力压迫**  
   她不是被单点攻击，而是一直有人想把她拖下去。

5. **对手压力是持续燃烧的**  
   不是一集一个反派，而是有人不断往她身上加码。

6. **保护要公开化、可见化**  
   男主不能只私下好，要在当众羞辱里把立场亮出来。

## 等效迁移外形

### 原壳：狼人命定伴侣

迁成：

- `old-money` 继承人 + 家族信托逼婚 / 继承考核
- 一场“错误闯入的夜晚”让女主被卷进继承链
- 怀孕让这条继承链从绯闻变成事实风险

### 原壳：宠物陪玩 + 狗链 + 发情夜

迁成：

- 女主不是“被送来的候选名媛”，而是误闯进家族候选夜的外部普通女孩
- 男主处于药物 / 遗传病 / 继承高压下的失控边缘
- 那一夜形成的不是兽化契约，而是：
  - 丑闻级亲密事实
  - 怀孕事实
  - 信托继承风险

### 原壳：地下赌场 / 兔女郎 / 鞭抽

迁成：

- 私人债务俱乐部 / 会员制黑房 / 老钱圈地下招待套房
- 女主被亲爹当抵债货物塞进去
- 她被围住、被羞辱、被打到见血，孩子出事
- 男主闯入时，必须还是“最脏的现场”，不能洗成找人谈判

### 原壳：医院卧底医生 / 转院劫持

迁成：

- 私人医疗体系里的内鬼医生
- 以“更好的救治”名义把女主和孩子转走
- 实际是把她送进敌对家族手里

### 原壳：学校羞辱“怀了野种”

迁成：

- 精英学院 / 奖学金体系里的当众羞辱
- 女主被指成“用身体攀附权贵、带着不明孩子上位”
- 男主必须在公开场合认下她和孩子

## 自由改写变量

- 家族名称、职业、行业、地理壳
- 男主失控原因的现实化包装
- 母辈权力结构
- 对手家族的外部身份包装
- 女主的学校 / 奖学金 / 社会阶层外壳

## 必须删除的烂点

- 狼人专有名词
- 纯奇幻设定解释
- 宠物陪玩、狗链、兔女郎这类高辨识奇观外形
- “因为设定所以这样”的偷懒解释

## forbidden shape

后续任何层不允许直接回用：

- 狼人 / fated mate / wolf king / 月圆 / 诅咒 / 狼嚎
- 宠物陪玩 / 狗链 / 狼犬
- 地下赌场 / 兔女郎 / 赌桌强奸威胁
- 校医直接播报 + 论坛围观的原句式

## 新壳可承载范围

这版新壳最能承载的强戏空间有：

- 老钱庄园 / 候选夜 / 家族餐厅 / 私密翼楼
- 精英学院 / 奖学金场域 / 校门口 / 教室公开场
- 私人债务俱乐部 / 地下会员套房 / 黑车转运
- 家族医院 / 私人医疗转运线 / 公路截停
- 顶级住宅 / 母辈家族资源场 / 回校护短场

## 例外风险

如果后续把“强绑定”写成：

- 纯继承协议
- 纯法律文件
- 纯检测报告
- 纯家族会议决定

那这条新壳就会只剩“合理”，不剩“短剧”。

## 一句话总结

这版洗稿方案的要点不是把狼人故事翻译成人类故事，而是：

> 把“命定伴侣的宿命强制感”，翻成“你已经被卷进我的家族、我的孩子、我的敌人和我的名声里，谁也别想把你干净放走”的现实强绑定。
